# 教师资格考试备考（初中数学）

> 初中数学教师资格考试共三科，笔试全部通过后方可参加面试。

---

## 考试科目

| 科目 | 代码 | 考试时间 | 满分 | 合格线 |
|------|------|---------|------|--------|
| 综合素质（中学） | 301 | 120 分钟 | 150 | 70（折算后） |
| 教育知识与能力（中学） | 302 | 120 分钟 | 150 | 70 |
| 数学学科知识与教学能力（初中） | 304 | 120 分钟 | 150 | 70 |

> 合格线说明：卷面满分 150 分，折算为 120 分制后 70 分及格，实际卷面约需 90 分以上

---

## 三个科目备考指南

### 科目一：综合素质（中学）🔴🔴

- **难度**：★★☆☆☆（三科中最易）
- **核心**：三观一德（教育观、学生观、教师观、职业道德）+ 写作
- **攻略**：背口诀 + 练材料分析 + 积累写作素材

👉 [综合素质完整知识体系与重点](./comprehensive-quality/)

### 科目二：教育知识与能力（中学）🔴🔴🔴

- **难度**：★★★★☆（三科中最难背）
- **核心**：教学原则/方法 + 德育原则/方法 + 学习心理
- **攻略**：理解为主、口诀记忆 + 大量练辨析题和简答题

👉 [教育知识与能力完整知识体系与重点](./education-knowledge/)

### 科目三：数学学科知识与教学能力（初中）🔴🔴🔴

- **难度**：★★★★☆（专业性强）
- **核心**：教学设计 + 案例分析 + 数学思想方法
- **攻略**：练写教案 + 分析教学案例 + 巩固大学数学

👉 [数学学科知识完整知识体系与重点](./math-subject/)

---

## 备考时间线建议

```
第1-2周   科目一：三观一德 + 写作素材积累
第3-5周   科目二：教育基础+教学+德育+学习心理
第6-8周   科目三：大学数学+初中数学+教学设计
第9-10周  三科综合刷题 + 查漏补缺
第11周    模拟考试 + 冲刺
```

## 三科关联关系

```
科目一（综合素质）         科目二（教育知识与能力）
  ├ 职业理念 ───────────→ 教育基础知识和基本原理
  ├ 教师职业道德 ───────→ 德育
  └ 写作能力              └ 学习心理
                              │
                              ↓
                    科目三（数学学科知识与教学能力）
                    ├ 课程知识（依据科目二的教育理论）
                    ├ 教学知识（教学方法+思想方法）
                    └ 教学技能（教学设计+案例分析）
```
