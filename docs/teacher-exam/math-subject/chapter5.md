# 第五章 教学设计

> 本章考情：教学设计题 30 分，必考
> 最重要的一章，需要大量练习写教案

---

## 5.1 教学设计基本环节 🔴🔴🔴

| 环节 | 内容 | 注意事项 |
|------|------|---------|
| 教材分析 🔴🔴 | 地位与作用、前后联系、重点难点 | 要说清"为什么这是重点" |
| 学情分析 🔴 | 已有基础、认知困难、学习兴趣 | 要具体，不能泛泛而谈 |
| 教学目标 🔴🔴🔴 | 四基目标+核心素养 | 行为动词要准确 |
| 教学重难点 🔴🔴 | 重点+难点+突破策略 | 重点≠难点 |
| 教学过程 🔴🔴🔴 | 情境→探究→巩固→小结→作业 | 每个环节写设计意图 |
| 板书设计 | 主板书+副板书 | 清晰、结构化 |
| 教学反思 | 课后对教学过程的反思 | 体现教学智慧 |

## 5.2 教学目标设定 🔴🔴🔴

### 三维目标（传统写法）

```
1. 知识与技能：理解/掌握……
2. 过程与方法：经历……过程，发展……能力
3. 情感态度与价值观：体会……，培养……
```

### 新课标四基目标写法 🔴🔴🔴

```
1. 理解/掌握……（基础知识）
2. 经历……过程，能够……（基本技能+基本活动经验）
3. 体会……思想方法，发展……素养（基本思想+核心素养）
```

### 行为动词层次 🔴🔴

| 层次 | 动词 | 含义 |
|------|------|------|
| 了解 | 知道、认识、辨认 | 能回忆和识别 |
| 理解 🔴 | 解释、说明、判断 | 能解释含义和关系 |
| 掌握 🔴 | 运用、证明、推导 | 能在新情境中运用 |
| 运用 🔴🔴 | 灵活运用、综合运用 | 能综合解决复杂问题 |

---

## 5.3 教学重难点 🔴🔴

**重点**：本节课最核心的知识或技能，由教材地位决定
**难点**：学生不容易理解或容易出错的内容，由学生认知水平决定

**确定重点的方法**：
1. 看本节内容在章节中的核心地位
2. 看是否为后续学习的基础

**突破难点的方法** 🔴🔴：
1. 直观演示 — 化抽象为具体
2. 分步讲解 — 降低认知坡度
3. 类比迁移 — 从已知到未知
4. 变式练习 — 从不同角度理解

## 5.4 教学过程设计模板 🔴🔴🔴

### 通用模板

```
一、创设情境，引入新课（3-5分钟）
   情境内容：……
   设计意图：激发兴趣，引出课题

二、探究新知（15-20分钟）
   活动1：……
   设计意图：……
   活动2：……
   设计意图：……
   活动3：……
   设计意图：……

三、巩固练习（10-15分钟）
   基础题：……（巩固基础）
   提高题：……（深化理解）
   设计意图：分层练习，检验目标达成

四、课堂小结（3-5分钟）
   引导学生从知识和方法两方面总结
   设计意图：梳理知识，提炼方法

五、布置作业
   必做题：……
   选做题：……
   设计意图：分层作业，巩固提升
```

### 不同课型的教学设计 🔴🔴

| 课型 | 重点环节 | 设计要点 |
|------|---------|---------|
| 概念课 🔴🔴🔴 | 概念的形成过程 | 丰富实例→概括本质→辨析理解→应用巩固 |
| 定理/公式课 🔴🔴 | 定理的发现与证明 | 创设情境→提出猜想→分析证明→应用拓展 |
| 习题课 🔴 | 解题方法总结 | 典型例题→方法提炼→变式训练→反思提升 |
| 复习课 🔴 | 知识结构构建 | 知识梳理→体系构建→综合应用→查漏补缺 |

## 5.5 常考课题教学设计 🔴🔴🔴

### 课题1：二次函数的图像与性质 🔴🔴🔴

**教材分析**：二次函数是初中数学的核心内容，是函数知识的深化，也是高中函数学习的基础。

**教学目标**：
1. 理解二次函数 y=ax&sup2; 的图像是抛物线，掌握其开口方向、对称轴、顶点等性质
2. 经历描点法作图和性质探究的过程，发展直观想象和数学抽象素养
3. 体会数形结合思想，感受数学的对称美

**教学重点**：二次函数 y=ax&sup2; 的图像与性质
**教学难点**：a 的正负对开口方向的影响，|a| 对开口大小的影响

**教学过程**：

```
一、创设情境
   展示喷泉水流、抛物运动的图片
   提问：这些运动轨迹有什么共同特征？

二、探究新知
   活动1：描点法画出 y=x&sup2; 的图像
   - 列表取值（x=-3,-2,-1,0,1,2,3）
   - 描点连线，观察图像形状
   - 归纳：抛物线，开口向上，关于y轴对称，顶点在原点

   活动2：在同一坐标系中画 y=x&sup2; 和 y=-x&sup2;
   - 比较开口方向
   - 归纳：a>0开口向上，a<0开口向下

   活动3：画 y=2x&sup2; 和 y=&frac12;x&sup2; 的图像
   - 比较开口大小
   - 归纳：|a|越大，开口越小（抛物线越"窄"）

三、巩固练习
   基础题：说出 y=3x&sup2;、y=-2x&sup2; 的开口方向和对称轴
   提高题：已知抛物线 y=ax&sup2; 经过点(2,8)，求a的值

四、课堂小结
   知识：y=ax&sup2; 的图像是抛物线
   方法：描点法、数形结合

五、布置作业
   必做题：课本习题
   选做题：探究 y=a(x-h)&sup2;+k 的图像
```

### 课题2：全等三角形的判定（SAS）🔴🔴

**教学重点**：SAS 判定方法的理解和应用
**教学难点**：证明过程中"对应"关系的确定

**教学过程**：

```
一、情境引入
   展示两块完全相同的三角板
   提问：如何判定两个三角形全等？

二、探究新知
   活动1：画&triangle;ABC，使AB=3cm，&angle;A=60°，AC=4cm
   - 剪下三角形，与同桌比较
   - 发现：两边和夹角确定→三角形唯一确定

   活动2：归纳 SAS 判定方法
   - 文字语言：两边和它们的夹角分别相等的两个三角形全等
   - 符号语言：在&triangle;ABC和&triangle;DEF中，
     AB=DE，&angle;B=&angle;E，BC=EF
     &rArr; &triangle;ABC&cong;&triangle;DEF(SAS)

   活动3：强调"夹角"
   - 反例：两边和其中一边的对角（SSA）不能判定全等

三、巩固练习
   基础题：直接用SAS判定全等
   提高题：需先证出边或角相等，再用SAS

四、小结
   知识：SAS判定方法
   方法：找对应边角→书写规范
   注意：必须是对应的"夹角"

五、作业
```

## 近三年真题精选

**1.**（2024上）教学设计题：请为"勾股定理"设计一个完整的教学方案（含教学目标、重难点、教学过程）。

> **参考答案要点**：
> 教学目标：①理解勾股定理的内容；②经历勾股定理的发现过程；③体会数形结合思想。
> 重点：勾股定理的理解和应用。难点：勾股定理的发现和证明。
> 教学过程：①情境引入（展示直角三角形三边关系）；②探究发现（画方格纸上的直角三角形，数格子发现 a&sup2;+b&sup2;=c&sup2;）；③证明（赵爽弦图/面积法）；④应用（求直角三角形边长）；⑤小结（知识+方法）。

**2.**（2023下）教学设计题：请为"一元一次方程的解法（移项）"设计教学过程。

> **参考答案要点**：
> ①复习引入：回顾等式性质；②探究新知：通过具体方程 2x+5=3x-2 引导发现移项规律；③归纳：移项要变号；④巩固练习：由简到繁，先不含分母→含分母；⑤小结：移项法则+解方程步骤。

**3.**（2024上）下列教学目标中，表述最规范的是（ ）

A. 让学生理解二次函数 🔴 &emsp; B. 培养学生数学能力 &emsp; C. 掌握二次函数 y=ax&sup2;+bx+c 的图像与性质 🔴 &emsp; D. 提高学生分析问题能力

> **答案**：C &emsp; 行为动词准确，内容具体可检测

**4.**（2023下）简述确定教学重点和难点的方法。

> **答**：确定重点的方法：①看本节内容在单元/章节中的核心地位；②看是否为后续学习的基础；③看课标的要求程度。确定难点的方法：①分析学生已有认知水平与本节内容的差距；②预判学生容易出错或混淆的地方；③考虑内容的抽象程度和思维跨度。

**5.**（2023上）案例分析：某教师在"平行四边形性质"教学中，直接给出"平行四边形对边相等、对角相等、对角线互相平分"三个性质，然后让学生做练习。请评价并给出改进建议。

> **答**：该教师的教学存在不足：①忽视了性质的探究过程，学生缺乏发现体验；②没有体现"学生主体"理念；③不利于培养学生的几何推理能力。改进：①设计操作活动，让学生测量、比较、猜想；②引导学生用全等三角形证明猜想；③从"发现→猜想→证明→应用"的完整过程中学习。

**6.**（2024上）教学设计中的"设计意图"应该写什么？

> **答**：设计意图应说明：①该环节对应哪个教学目标；②该环节体现了什么教学理念或策略；③该环节对学生学习的促进作用。例如："通过画图操作让学生直观感知，体现数形结合思想，发展直观想象素养。"

**7.**（2023下）请为"圆的切线判定定理"设计教学过程。

> **参考答案要点**：①复习引入：圆的切线定义和性质；②探究：用直尺在圆上尝试画切线→发现切线与半径垂直；③归纳：经过半径外端且垂直于这条半径的直线是圆的切线；④辨析："经过半径外端"和"垂直于半径"缺一不可；⑤应用：过圆上一点画切线。

**8.**（2024上）简述"讲练结合法"在数学教学中的应用策略。

> **答**：①精讲精练：讲在关键处，练在点子上；②讲练交替：讲一个知识点练一道题，避免集中讲完再练；③分层练习：基础题→提高题→拓展题；④反馈及时：学生练习后及时点评纠错；⑤重视过程：不只讲答案，更要讲思路和方法。

**9.**（2023上）教学设计题：请为"中位数和众数"设计完整的教学方案。

> **参考答案要点**：①情境引入：某公司招聘，声称平均工资8000元，但多数员工不到5000元→引出平均数的局限性；②探究：引入中位数和众数的概念；③比较：平均数、中位数、众数各自的特点和适用场景；④应用：选择合适的统计量分析实际数据；⑤小结：三个统计量的联系与区别。

**10.**（2024上）评析以下教学目标设定的优缺点："使学生理解一次函数的图像是一条直线。"

> **答**：优点：内容指向明确（一次函数的图像）。缺点：①"使学生"体现了教师本位，应以学生为主体；②"理解"层次较低，应上升到"掌握"或"运用"；③缺少过程性目标和素养目标。修改建议："理解一次函数 y=kx+b 的图像是直线，经历描点作图和探究性质的过程，体会数形结合思想，发展直观想象素养。"
