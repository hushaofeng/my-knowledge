### 第一章 大学数学基础 🔴🔴

#### 1.1 数学分析

| 知识点 | 核心内容 | 考频 |
|--------|---------|------|
| 极限 🔴 | 数列极限、函数极限、极限运算法则、两个重要极限 | ⭐⭐⭐ |
| 连续 🔴 | 连续定义、间断点分类、闭区间连续函数性质 | ⭐⭐ |
| 导数 🔴🔴 | 导数定义、求导法则、隐函数求导、高阶导数 | ⭐⭐⭐⭐ |
| 微分中值定理 🔴 | 罗尔定理、拉格朗日中值定理 | ⭐⭐⭐ |
| 导数应用 🔴🔴 | 单调性、极值、最值、凹凸性、拐点 | ⭐⭐⭐⭐ |
| 不定积分 | 换元积分法、分部积分法 | ⭐⭐ |
| 定积分 🔴 | 牛顿-莱布尼茨公式、定积分应用 | ⭐⭐⭐ |

**重要极限**：
- lim(x→0) sinx/x = 1
- lim(x→∞) (1+1/x)^x = e

**求导公式**（必须熟记）：
- (x^n)' = nx^(n-1)
- (sinx)' = cosx, (cosx)' = -sinx
- (e^x)' = e^x, (lnx)' = 1/x

## 第二部分 课程知识 🔴🔴

### 第三章 初中数学课程 🟡🔴

#### 3.1 课程性质

- **基础性**：面向全体学生，提供未来发展的基础
- **普及性**：义务教育的组成部分
- **发展性**：促进人的全面发展

#### 3.2 课程基本理念 🔴🔴

1. 课程要面向全体学生，适应学生个性发展
2. 课程内容要反映社会的需要，体现数学的本质
3. 教学活动是师生积极参与、交往互动、共同发展的过程
4. 学习评价的主要目的是全面了解学生数学学习的过程和结果
5. 信息技术与课程内容整合

#### 3.3 课程目标 🔴🔴

**四基**：基础知识、基本技能、基本思想、基本活动经验

**四能**：发现问题、提出问题、分析问题、解决问题

**核心素养**（2022新课标）🔴🔴：

| 素养 | 含义 |
|------|------|
| 数学抽象 | 舍弃事物物理属性，抽出数学本质 |
| 逻辑推理 | 从已有命题推出新命题 |
| 数学建模 | 用数学方法解决实际问题 |
| 直观想象 | 借助几何直观和空间想象 |
| 数学运算 | 在规则指引下变形和求解 |
| 数据分析 | 收集整理描述分析数据 |

#### 3.4 课程内容结构

| 领域 | 七年级 | 八年级 | 九年级 |
|------|--------|--------|--------|
| 数与代数 | 有理数、整式、一元一次方程 | 实数、一次函数、整式乘法与因式分解 | 二次函数、一元二次方程 |
| 图形与几何 | 相交线平行线、三角形 | 全等三角形、勾股定理、四边形 | 相似、锐角三角函数、圆 |
| 统计与概率 | 数据收集整理 | 数据分析 | 概率初步 |
| 综合与实践 | 数学活动 | 数学活动 | 数学活动 |

## 第三部分 教学知识 🔴🔴🔴

> 🔴🔴🔴 高频大题区域，案例分析+论述题必考

### 第四章 数学教学方法 🔴🔴🔴

#### 4.1 常用教学方法

| 方法 | 含义 | 适用场景 |
|------|------|---------|
| 讲授法 | 教师系统讲解 | 概念引入、定理证明 |
| 讨论法 | 学生分组讨论 | 开放性问题 |
| 发现法 🔴 | 引导学生自主发现 | 规律探究 |
| 讲练结合法 | 讲解+练习 | 技能训练 |
| 问题驱动法 🔴 | 以问题链引导 | 概念课、复习课 |

#### 4.2 数学概念教学 🔴🔴🔴

**概念教学的环节**：

1. **概念的引入**：从实际情境或数学内部引入
2. **概念的形成**：通过观察、比较、抽象、概括
3. **概念的理解**：明确内涵与外延
4. **概念的应用**：在具体问题中使用概念
5. **概念的深化**：与其他概念建立联系

**概念教学原则**：
- 用具体实例引入，避免直接给定义
- 强调概念的本质属性
- 注意概念间的联系与区别
- 重视概念的应用

#### 4.3 数学命题教学 🔴🔴

**命题教学（定理/公式）的环节**：

1. **命题的发现**：引导学生猜想
2. **命题的证明**：分析证明思路
3. **命题的应用**：在问题解决中运用
4. **命题的拓展**：推广与变式

#### 4.4 数学思想方法 🔴🔴🔴

| 思想方法 | 含义 | 典型应用 |
|---------|------|---------|
| 数形结合 🔴🔴 | 以数解形、以形助数 | 函数图像、几何计算 |
| 分类讨论 🔴🔴 | 按标准分类逐一讨论 | 绝对值、参数问题 |
| 化归 🔴🔴 | 未知转化为已知 | 分式→整式、高次→低次 |
| 方程与函数 🔴 | 用方程或函数建模 | 应用题、最值问题 |
| 整体思想 | 将部分视为整体 | 代数化简 |
| 逆向思维 | 从结论反推 | 证明题 |

## 第四部分 教学技能 🔴🔴🔴

> 🔴🔴🔴 教学设计 30 分，案例分析 40 分，共 70 分占 47%

### 第五章 教学设计 🔴🔴🔴

#### 5.1 教学设计的基本环节

1. **教材分析** 🔴 — 地位与作用、重点与难点
2. **学情分析** 🔴 — 已有基础、认知困难
3. **教学目标设定** 🔴🔴🔴
4. **教学重难点确定** 🔴
5. **教学过程设计** 🔴🔴🔴
6. **板书设计**
7. **教学反思**

#### 5.2 教学目标设定 🔴🔴🔴

**三维目标**（传统）→ **四基目标**（新课标）

**新课标教学目标写法** 🔴🔴：

```
1. 理解/掌握……（基础知识）
2. 经历……过程，能够……（基本技能+基本活动经验）
3. 体会……思想方法，发展……素养（基本思想+核心素养）
```

**目标设定要求**：
- 行为动词准确（了解→理解→掌握→运用）
- 可观察、可检测
- 体现核心素养发展

#### 5.3 教学重难点 🔴

- **重点**：本节课最核心的知识或技能
- **难点**：学生不容易理解或容易出错的内容
- **突破难点的方法**：直观演示、分步讲解、类比、变式练习

#### 5.4 教学过程设计模板 🔴🔴🔴

```
一、创设情境，引入新课（3-5分钟）
   设计意图：……

二、探究新知（15-20分钟）
   活动1：……
   设计意图：……
   活动2：……
   设计意图：……

三、巩固练习（10-15分钟）
   基础题：……
   提高题：……

四、课堂小结（3-5分钟）
   引导学生从知识和方法两方面总结

五、布置作业
   必做题+选做题（分层设计）
```

### 第六章 教学评价与案例分析 🔴🔴🔴

#### 6.1 教学评价 🔴

| 评价类型 | 含义 | 作用 |
|---------|------|------|
| 诊断性评价 | 教学前摸底 | 了解起点 |
| 形成性评价 🔴 | 教学过程中 | 及时反馈调整 |
| 终结性评价 | 教学结束后 | 总结性判断 |

**评价原则**：过程与结果并重、定性与定量结合、多元评价

#### 6.2 案例分析答题框架 🔴🔴🔴

**常见题型**：

1. **评价教师的教学行为** — 从教学理念、教学方法、教学过程等角度分析
2. **分析学生的错误** — 找出错误→分析原因→给出对策
3. **分析教学设计的优劣** — 目标是否合理、过程是否有效、是否体现新课标理念

**答题模板**：

```
一、总体评价：该教师的教学行为是合理的/存在不足的。

二、具体分析：
1. 教学目标方面：……（是否体现核心素养、是否合理）
2. 教学方法方面：……（是否体现学生主体、是否恰当）
3. 教学过程方面：……（情境创设、探究活动、练习设计）
4. 数学思想方法方面：……（是否渗透数学思想）

三、改进建议：
1. ……
2. ……
```

#### 6.3 学生典型错误分析 🔴🔴

| 错误类型 | 典型表现 | 原因分析 | 教学对策 |
|---------|---------|---------|---------|
| 概念不清 | (-3)&#178; = -9 | 混淆运算顺序 | 强化概念辨析 |
| 忽视条件 | SSA 判定全等 | 条件意识薄弱 | 反例教学 |
| 分类遗漏 | |a| 去绝对值只取正 | 分类讨论训练 |
| 负迁移 | (a+b)&#178; = a&#178;+b&#178; | 不完全归纳 | 变式对比 |
| 计算错误 | 移项不变号 | 运算法则不熟 | 规范训练 |
