## 1.1 数学分析 🔴🔴🔴

### 极限 🔴🔴

**数列极限定义**：对任意 &epsilon;>0，存在 N，当 n>N 时，|a_n - A|<&epsilon;，则 lim a_n = A

**函数极限**：
- lim(x→x₀) f(x) = A
- 左极限 lim(x→x₀⁻) f(x)、右极限 lim(x→x₀⁺) f(x)
- 极限存在 &hArr; 左右极限存在且相等 🔴

**两个重要极限** 🔴🔴：
1. lim(x→0) sinx/x = 1
2. lim(x→∞) (1+1/x)^x = e，即 lim(x→0) (1+x)^(1/x) = e

**极限运算法则**：
- lim[f(x)&plusmn;g(x)] = limf(x) &plusmn; limg(x)
- lim[f(x)&middot;g(x)] = limf(x) &middot; limg(x)
- lim[f(x)/g(x)] = limf(x)/limg(x)（分母极限不为0）

**常见未定式**：0/0 型、&infin;/&infin; 型 → 洛必达法则 🔴

### 连续 🔴

**连续定义**：lim(x→x₀) f(x) = f(x₀)，即函数值 = 极限值

**间断点分类** 🔴🔴：

| 类型 | 条件 | 举例 |
|------|------|------|
| 可去间断点 🔴 | 左右极限存在且相等 &ne; 函数值 | f(x)=sinx/x 在 x=0 |
| 跳跃间断点 | 左右极限存在但不相等 | 符号函数 sgn(x) |
| 无穷间断点 | 至少一侧极限为 &infin; | f(x)=1/x 在 x=0 |
| 振荡间断点 | 极限不存在且非无穷 | f(x)=sin(1/x) 在 x=0 |

第一类间断点 = 可去+跳跃（左右极限都存在）
第二类间断点 = 无穷+振荡（至少一侧极限不存在）

**闭区间连续函数性质** 🔴：
1. 最值定理 — 有最大值和最小值
2. 有界性定理 — 函数有界
3. 零点定理 🔴 — f(a)&middot;f(b)<0 &rArr; &exist; c&isin;(a,b)，f(c)=0
4. 介值定理 — f(x) 取到 f(a) 与 f(b) 之间一切值

### 导数 🔴🔴🔴

**导数定义** 🔴🔴：

f'(x₀) = lim(&Delta;x→0) [f(x₀+&Delta;x)-f(x₀)]/&Delta;x

**基本求导公式** 🔴🔴🔴：

| 函数 | 导数 |
|------|------|
| C（常数） | 0 |
| x^n | nx^(n-1) |
| sinx | cosx |
| cosx | -sinx |
| tanx | sec&sup2;x |
| e^x | e^x |
| a^x | a^x&middot;lna |
| lnx | 1/x |
| log_a(x) | 1/(x&middot;lna) |

**求导法则** 🔴🔴：
- (u&plusmn;v)' = u'&plusmn;v'
- (uv)' = u'v+uv'
- (u/v)' = (u'v-uv')/v&sup2;
- 链式法则：[f(g(x))]' = f'(g(x))&middot;g'(x) 🔴🔴

**隐函数求导** 🔴：F(x,y)=0，对 x 求导时将 y 视为 x 的函数

**高阶导数**：f''(x) = (f'(x))'

### 微分中值定理 🔴🔴

| 定理 | 条件 | 结论 | 几何意义 |
|------|------|------|---------|
| 罗尔定理 🔴 | 闭区间连续+开区间可导+f(a)=f(b) | &exist;c&isin;(a,b)，f'(c)=0 | 水平切线 |
| 拉格朗日中值定理 🔴🔴 | 闭区间连续+开区间可导 | &exist;c&isin;(a,b)，f'(c)=[f(b)-f(a)]/(b-a) | 平行于弦的切线 |

### 导数应用 🔴🔴🔴

**单调性** 🔴🔴：
- f'(x)>0 &rArr; f(x) 单调递增
- f'(x)<0 &rArr; f(x) 单调递减

**极值** 🔴🔴🔴：
- 必要条件：f'(x₀)=0（驻点）或 f'(x₀) 不存在
- 第一充分条件：f'(x) 在 x₀ 两侧变号
  - 左正右负 &rArr; 极大值
  - 左负右正 &rArr; 极小值
- 第二充分条件：f'(x₀)=0，f''(x₀)>0 极小，f''(x₀)<0 极大 🔴

**凹凸性与拐点** 🔴：
- f''(x)>0 &rArr; 凹（开口向上）
- f''(x)<0 &rArr; 凸（开口向下）
- f''(x) 变号的点 = 拐点

### 积分 🔴

**不定积分**（求原函数）：
- &int;x^n dx = x^(n+1)/(n+1) + C（n&ne;-1）
- &int;1/x dx = ln|x| + C
- &int;e^x dx = e^x + C
- &int;sinx dx = -cosx + C

**换元法** 🔴：&int;f(g(x))g'(x)dx = &int;f(u)du（令 u=g(x)）

**分部积分法** 🔴：&int;uv'dx = uv - &int;u'vdx

**定积分** 🔴🔴：
- 牛顿-莱布尼茨公式：&int;a^b f(x)dx = F(b)-F(a)
- 几何意义：曲线 y=f(x) 与 x 轴之间的面积

## 1.2 高等代数 🔴🔴

### 行列式 🔴🔴

**二阶行列式**：
|a b; c d| = ad - bc

**三阶行列式**（沙路法）🔴

**行列式性质** 🔴：
1. 行列式转置，值不变
2. 互换两行（列），值变号
3. 某行（列）有公因子可提出
4. 两行（列）相同或成比例，值为 0
5. 某行（列）拆成两数之和，可拆成两个行列式之和

**克莱姆法则** 🔴：n 元 n 个方程的线性方程组，若系数行列式 D&ne;0，则 x_i = D_i/D

### 矩阵 🔴🔴🔴

**矩阵运算**：
- 加法：对应元素相加
- 数乘：每个元素乘以该数
- 乘法：A_m&times;n &middot; B_n&times;p = C_m&times;p 🔴🔴（不满足交换律）

**逆矩阵** 🔴🔴：
- 定义：AB=BA=I，则 B=A⁻¹
- 存在条件：|A|&ne;0（非奇异/可逆）
- 求法：初等变换法、伴随矩阵法

**矩阵的秩** 🔴：非零子式的最高阶数，初等行变换化为阶梯形，非零行数 = 秩

### 线性方程组 🔴🔴

**解的判定** 🔴🔴🔴：

| 条件 | 结论 |
|------|------|
| r(A)=r(A&vert;b)=n | 唯一解 |
| r(A)=r(A&vert;b)<n | 无穷多解 |
| r(A)&ne;r(A&vert;b) | 无解 |

**齐次线性方程组 Ax=0**：
- 一定有零解
- r(A)<n &rArr; 有非零解 🔴

## 1.4 概率与统计 🔴🔴

### 随机事件 🔴🔴

| 公式 | 含义 |
|------|------|
| P(A&cup;B) = P(A)+P(B)-P(A&cap;B) | 加法公式 |
| P(A&cap;B) = P(A)&middot;P(B&vert;A) | 乘法公式 |
| P(B&vert;A) = P(A&cap;B)/P(A) | 条件概率 |
| P(Ā) = 1-P(A) | 对立事件 |

### 古典概型 🔴🔴

条件：有限个等可能基本事件
公式：P(A) = A 包含的基本事件数 / 基本事件总数

**常见模型**：
- 抽球问题
- 排列组合计算 🔴：C(n,m) = n!/[m!(n-m)!]

### 贝叶斯公式 🔴

P(A_i&vert;B) = P(B&vert;A_i)&middot;P(A_i) / &sum;P(B&vert;A_j)&middot;P(A_j)

### 随机变量 🔴

| 类型 | 描述 | 数字特征 |
|------|------|---------|
| 离散型 | 分布列 P(X=x_i)=p_i | 期望 E(X)=&sum;x_i&middot;p_i |
| 连续型 | 密度函数 f(x) | 期望 E(X)=&int;x&middot;f(x)dx |

**方差** 🔴：D(X) = E(X&sup2;) - [E(X)]&sup2;

### 正态分布 🔴🔴

X ~ N(&mu;, &sigma;&sup2;)
- &mu; 决定对称轴位置
- &sigma; 决定曲线胖瘦
- 标准化：Z = (X-&mu;)/&sigma; ~ N(0,1)
- 3&sigma; 原则 🔴：
  - P(&mu;-&sigma; < X < &mu;+&sigma;) &approx; 68.3%
  - P(&mu;-2&sigma; < X < &mu;+2&sigma;) &approx; 95.4%
  - P(&mu;-3&sigma; < X < &mu;+3&sigma;) &approx; 99.7%

## 近三年真题精选

### 选择题

**1.**（2024上）函数 f(x) = x&sup3;-3x 在 x=1 处的导数值为（ ）

A. 0 &emsp; B. 3 &emsp; C. -3 &emsp; D. 6

> **答案**：A
> **解析**：f'(x) = 3x&sup2;-3，f'(1) = 3-3 = 0

**2.**（2024上）设 A 为 3 阶方阵，|A|=2，则 |2A| =（ ）

A. 4 &emsp; B. 8 &emsp; C. 16 &emsp; D. 6

> **答案**：C
> **解析**：|kA| = k&sup3;|A|（n 阶方阵），|2A| = 2&sup3;&times;2 = 16

**3.**（2023下）lim(x→0) sin3x/x 的值为（ ）

A. 0 &emsp; B. 1 &emsp; C. 3 &emsp; D. 1/3

> **答案**：C
> **解析**：sin3x/x = 3&middot;sin3x/(3x)，由重要极限 sinu/u→1，得 3&times;1=3

**4.**（2023上）函数 f(x) = x/(x&sup2;-1) 的间断点个数为（ ）

A. 0 &emsp; B. 1 &emsp; C. 2 &emsp; D. 3

> **答案**：C
> **解析**：x&sup2;-1=0 &rArr; x=&plusmn;1，2个间断点

**5.**（2023上）从1,2,3,4中任取两个数，两个数之和为偶数的概率为（ ）

A. 1/3 &emsp; B. 2/3 &emsp; C. 1/2 &emsp; D. 1/4

> **答案**：A
> **解析**：C(4,2)=6种，和为偶数：(1,3),(2,4)共2种，P=2/6=1/3

### 解答题

**6.**（2024上）求函数 f(x) = x&sup3;-3x+1 的单调区间和极值。

> **解**：
> f'(x) = 3x&sup2;-3 = 3(x+1)(x-1)
> 令 f'(x)=0，得 x=-1 或 x=1
>
> | 区间 | (-&infin;,-1) | (-1,1) | (1,+&infin;) |
> |------|-------------|---------|-------------|
> | f'(x) | + | - | + |
> | f(x) | &nearr; | &searr; | &nearr; |
>
> 单调递增区间：(-&infin;,-1) 和 (1,+&infin;)
> 单调递减区间：(-1,1)
> 极大值 f(-1)=3，极小值 f(1)=-1

**7.**（2023下）已知矩阵 A = [1 2; 3 4]，求 A⁻¹。

> **解**：
> |A| = 1&times;4-2&times;3 = -2 &ne; 0，可逆
> A⁻¹ = (1/|A|)&middot;A* = (1/-2)&middot;[4 -2; -3 1] = [-2 1; 3/2 -1/2]

**8.**（2023上）设随机变量 X~N(2, 4)，求 P(X>4) 的近似值。（已知 &Phi;(1)=0.8413）

> **解**：
> 标准化：Z = (X-2)/2 ~ N(0,1)
> P(X>4) = P(Z>(4-2)/2) = P(Z>1) = 1-&Phi;(1) = 1-0.8413 = 0.1587

### 简答题

**9.**（2024上）简述罗尔定理的内容及其几何意义。

> **答**：若函数 f(x) 满足：①在闭区间[a,b]上连续；②在开区间(a,b)上可导；③f(a)=f(b)，则至少存在一点 c&isin;(a,b)，使得 f'(c)=0。
> 几何意义：在满足条件的曲线上，至少存在一点，该点的切线平行于 x 轴。

**10.**（2023下）设线性方程组 &lbrace;x₁+2x₂=1; 2x₁+4x₂=&lambda;&rbrace;，问 &lambda; 取何值时方程组有解？

> **解**：
> 增广矩阵 [1 2 &vert; 1; 2 4 &vert; &lambda;] → [1 2 &vert; 1; 0 0 &vert; &lambda;-2]
> r(A)=1，要使有解需 r(A)=r(A&vert;b)，即 &lambda;-2=0，&lambda;=2
> 当 &lambda;=2 时，r(A)=r(A&vert;b)=1<2，有无穷多解
