# 第二章 中学课程

> 本章考情：选择+简答为主，约 8-12 分
> 重点：课程类型、新课改、课程编制

---

## 2.1 课程概述 🔴🔴

### 课程的词源

- **中国**：宋代朱熹"宽着期限，紧着课程" — "课程"一词最早出现 🔴
- **西方**：斯宾塞《什么知识最有价值》首次将"课程"用作教育学的专门术语 🔴

### 课程类型 🔴🔴🔴

| 分类标准 | 类型 | 特点 | 代表/举例 |
|---------|------|------|----------|
| 组织方式 | 学科课程 🔴 | 按学科逻辑组织，强调知识体系 | 赫尔巴特、六艺 |
| | 活动课程 | 以儿童活动为中心 | 杜威 🔴 |
| | 综合课程 | 打破学科界限，融合多学科 | STS课程、科学课 |
| 修习方式 | 必修课程 | 全体学生必须修 | 国家课程 |
| | 选修课程 | 学生自主选择 | 校本选修 |
| 管理主体 | 国家课程 🔴 | 国家统一制定 | 统编教材 |
| | 地方课程 | 省级教育部门制定 | 地方文化课 |
| | 校本课程 🔴 | 学校自主开发 | 学校特色课 |
| 呈现方式 | 显性课程 | 有计划有组织 | 课表上的课 |
| | 隐性课程 🔴🔴 | 以间接、内隐方式影响 | 校风、师生关系、校园文化 |

**辨析题高频** 🔴🔴：
- "隐性课程不是课程" → ❌ 隐性课程也是课程，只是以间接方式影响学生
- "活动课程优于学科课程" → ❌ 各有优缺点，应互相补充

## 2.2 课程理论流派 🔴

| 流派 | 代表人物 | 核心主张 | 优缺点 |
|------|---------|---------|--------|
| 学科中心课程论 | 赫尔巴特、斯宾塞、布鲁纳 | 以学科知识为中心，按逻辑体系组织 | 系统但不贴近生活 |
| 学生中心课程论 | 杜威 🔴 | 以儿童兴趣和活动为中心 | 贴近学生但不够系统 |
| 社会中心课程论 | 布拉梅尔德 | 以社会问题为中心 | 关注社会但窄化教育 |

## 2.3 课程编制 🔴🔴

### 课程目标

**布鲁姆教育目标分类** 🔴🔴：

| 领域 | 层次（由低到高） |
|------|----------------|
| 认知领域 | 知识→理解→应用→分析→综合→评价 |
| 情感领域 | 接受→反应→形成价值观念→组织价值体系→价值体系个性化 |
| 动作技能领域 | 知觉→模仿→操作→准确→连贯→习惯化 |

### 课程内容 🔴🔴🔴

**三级层次**：

| 层次 | 含义 | 特点 |
|------|------|------|
| 课程计划 🔴 | 国家教育行政部门制定的教学工作的指导性文件 | 宏观，规定学科设置和顺序 |
| 课程标准 🔴🔴 | 每门学科的教学指导文件（原称"教学大纲"） | 中观，是编写教材和教学的依据 |
| 教材 🔴 | 教师和学生使用的教学材料 | 微观，课程标准的具体化 |

**辨析题** 🔴：
- "课程标准就是教材" → ❌ 课程标准是教学指导文件，教材是具体教学材料
- "教材就是教科书" → ❌ 教材包含教科书，也包含其他教学资源

### 课程评价 🔴

| 模式 | 提出者 | 核心观点 |
|------|--------|---------|
| 目标评价模式 | 泰勒 🔴 | 以目标为中心评价 |
| 目的游离评价模式 | 斯克里文 | 评价应关注实际效果而非预定目标 |
| CIPP 模式 | 斯塔弗尔比姆 | 背景→输入→过程→成果 |

## 2.4 新课改 🔴🔴🔴

### 核心理念

**"以人为本"** 🔴🔴 — 一切为了每一位学生的发展

### 课程结构改革 🔴🔴

| 改革内容 | 具体表现 |
|---------|---------|
| 整体设置九年一贯制 | 小学→初中一体化 |
| 小学阶段 | 综合课程为主 |
| 初中阶段 | 分科课程与综合课程相结合 🔴 |
| 高中阶段 | 分科课程为主 |
| 新增课程 | 综合实践活动课 🔴🔴（必修） |

**综合实践活动课** 🔴🔴：
- 内容：研究性学习、社区服务与社会实践、劳动与技术教育、信息技术教育
- 性质：必修课程
- 特点：自主性、开放性、实践性

### 学习方式改革 🔴🔴

| 传统方式 | 新方式 | 特点 |
|---------|--------|------|
| 被动接受 | 自主学习 | 主动、独立 |
| 独立完成 | 合作学习 | 互动、团队 |
| 机械记忆 | 探究学习 🔴 | 问题、过程、开放 |

### 课程管理改革 🔴

**三级课程管理**：国家课程 → 地方课程 → 校本课程

### 简答题必背 🔴🔴

**新课改的具体目标**（口诀：方平的节目有结）：

1. 改变课程功能 — 知识传授→引导学生学会学习、学会做人
2. 改变课程结构 — 学科本位→均衡性、综合性、选择性
3. 改变课程内容 — 难繁偏旧→联系生活和社会
4. 改变课程实施 — 死记硬背→主动参与、乐于探究
5. 改变课程评价 — 甄别选拔→促进发展
6. 改变课程管理 — 集中统一→三级管理
