# 第六章 中学心理辅导

> 本章考情：选择题为主，约 4-6 分
> 重点：心理健康标准、常见心理问题、辅导方法

---

## 6.1 心理健康标准 🔴

1. 智力正常 — 最基本条件
2. 情绪稳定乐观
3. 意志健全
4. 人格完整和谐
5. 自我意识正确
6. 人际关系良好
7. 社会适应良好
8. 心理行为符合年龄特征

**辨析** 🔴：
- "心理健康就是没有心理疾病" → ❌ 心理健康不仅是没有疾病，还包括积极的心理状态
- "心理健康是一成不变的" → ❌ 心理健康是动态的、相对的

## 6.2 常见心理问题 🔴🔴

### 焦虑症 🔴🔴

| 维度 | 内容 |
|------|------|
| 表现 | 过度担忧、紧张不安、心悸、出汗、失眠 |
| 类型 | 考试焦虑🔴、社交焦虑、广泛性焦虑 |
| 辅导方法 | 放松训练🔴、系统脱敏法、认知疗法 |

### 抑郁症 🔴

| 维度 | 内容 |
|------|------|
| 核心症状 | 情绪低落、兴趣丧失、精力减退 |
| 伴随症状 | 失眠/嗜睡、食欲变化、自我评价低 |
| 辅导方法 | 认知行为疗法、支持性心理治疗、及时转介 |

### 强迫症 🔴

| 维度 | 内容 |
|------|------|
| 表现 | 强迫观念（反复出现的不合理想法）+强迫行为（反复做某事） |
| 举例 | 反复洗手、反复检查门锁 |
| 辅导方法 | 暴露与反应预防、认知疗法 |

### 恐惧症

| 维度 | 内容 |
|------|------|
| 表现 | 对特定对象或情境产生强烈恐惧 |
| 类型 | 社交恐惧症、广场恐惧症、特定恐惧症 |
| 辅导方法 | 系统脱敏法 🔴 |

### 网络成瘾 🔴

| 维度 | 内容 |
|------|------|
| 判断标准 | 上网时间过长、无法控制、影响正常生活 |
| 辅导方法 | 行为契约法、时间管理、替代活动 |

## 6.3 心理辅导方法 🔴🔴

### 主要辅导方法

| 方法 | 核心技术 | 适用问题 | 关键人物 |
|------|---------|---------|---------|
| 系统脱敏法 🔴🔴 | 逐步暴露+放松训练 | 恐惧症、焦虑症 | 沃尔普 |
| 合理情绪疗法 🔴🔴 | ABC理论，改变不合理信念 | 各种情绪问题 | 埃利斯 |
| 来访者中心疗法 | 无条件积极关注+真诚+共情 | 自我成长问题 | 罗杰斯 |
| 行为契约法 | 约定行为目标和奖惩 | 行为矫正 | — |
| 代币法 | 好行为得代币，兑换奖励 | 行为塑造 | — |

### ABC 理论 🔴🔴🔴

**A（Activating events）** — 诱发事件
**B（Beliefs）** — 信念（认知）
**C（Consequences）** — 情绪和行为结果

**核心观点**：不是事件 A 直接导致结果 C，而是信念 B 导致了 C。改变不合理信念 B，就能改变 C。

**常见不合理信念** 🔴：
1. 绝对化要求 — "我必须考满分"
2. 过分概括化 — "一次失败=我是个失败者"
3. 糟糕至极 — "考砸了=一切都完了"

**举例**：
- 事件A：考试没考好
- 不合理信念B："我太笨了，什么都做不好"
- 结果C：抑郁、放弃
- 合理信念B'：这次没考好，我可以找到原因下次改进
- 结果C'：积极面对、继续努力

### 心理辅导原则 🔴

1. 面向全体学生 — 预防为主
2. 预防与发展相结合
3. 尊重与理解学生
4. 学生主体性
5. 个别化对待
6. 整体性发展
