## 4.1 认知过程 🔴🔴

### 感觉与知觉

**感觉适应** 🔴：

| 类型 | 含义 | 举例 |
|------|------|------|
| 暗适应 | 由亮到暗，感受性提高 | 进电影院适应黑暗 |
| 明适应 | 由暗到亮，感受性降低 | 从电影院出来被亮光晃眼 |

**知觉特性** 🔴🔴：

| 特性 | 含义 | 举例 |
|------|------|------|
| 选择性 | 把对象从背景中区分出来 | 万绿丛中一点红 |
| 整体性 | 把事物看成一个整体 | 只看部分也能认出整体 |
| 理解性 | 用已有经验理解知觉对象 | 内行看门道，外行看热闹 |
| 恒常性 | 知觉条件变化，映像不变 | 远处人虽小但知其身高不变 |

### 注意 🔴🔴

| 类型 | 有无目的 | 是否需要意志 | 例 |
|------|---------|------------|------|
| 无意注意 | 无 | 否 | 突然巨响、鲜艳颜色 |
| 有意注意 | 有 | 是 | 认真听课、专心解题 |
| 有意后注意 🔴 | 有 | 否（自动化） | 熟练骑车、打字 |

**注意的品质** 🔴：

| 品质 | 含义 | 影响因素 |
|------|------|---------|
| 注意的广度 | 同一时间能清楚把握的对象数量 | 对象排列、活动任务、个体经验 |
| 注意的稳定性 | 集中在某一对象上的持续时间 | 对象特点、个体状态 |
| 注意的分配 | 同时进行两种以上活动 | 活动熟练度、活动关系 |
| 注意的转移 | 主动从一个对象转到另一个 | 原注意的紧张度、新对象的吸引力 |

### 记忆 🔴🔴🔴

**记忆过程**：识记→保持→再认/回忆

**艾宾浩斯遗忘曲线** 🔴🔴🔴：
- 遗忘规律：先快后慢，呈负加速型
- 启示：及时复习 🔴

**遗忘原因** 🔴：

| 学说 | 核心观点 |
|------|---------|
| 衰退说 | 不用的记忆自动消退（感觉记忆、短时记忆） |
| 干扰说 🔴 | 前后学习材料互相干扰 |
| 压抑说 | 情绪动机压抑（弗洛伊德） |
| 提取失败说 | 存储但提取不出来（舌尖现象） |

**前摄抑制与倒摄抑制** 🔴🔴：
- 前摄抑制：前面学习的材料对后面学习的干扰
- 倒摄抑制：后面学习的材料对前面学习的干扰
- 早上学习只受倒摄抑制，晚上学习只受前摄抑制 → 早晚学习效果好

**知识保持方法** 🔴🔴：
1. 及时复习
2. 分散复习优于集中复习
3. 反复阅读与试图回忆相结合
4. 多种感官参与
5. 超额学习 150% 🔴

### 思维 🔴

**思维特征**：间接性、概括性

**思维种类**：

| 分类标准 | 类型 |
|---------|------|
| 发展水平 | 直觉动作思维→具体形象思维→抽象逻辑思维 |
| 指向性 | 聚合思维（求同）vs 发散思维（求异） |
| 创新程度 | 常规思维 vs 创造性思维 |

**创造性思维特征** 🔴🔴：流畅性、变通性、独创性

## 4.2 学习理论 🔴🔴🔴

> 占分最高的知识点

### 行为主义 🔴🔴🔴

| 理论 | 人物 | 核心实验 | 关键概念 |
|------|------|---------|---------|
| 经典条件作用 | 巴甫洛夫 🔴 | 狗唾液分泌 | 获得、消退、泛化、分化 |
| 试误说 | 桑代克 | 饿猫迷箱 | 三大定律：准备律、练习律、效果律 🔴 |
| 操作性条件作用 | 斯金纳 🔴🔴 | 白鼠按压杠杆 | 强化、惩罚、消退 |
| 社会学习理论 | 班杜拉 🔴 | 波波玩偶实验 | 观察学习、替代强化 |

**斯金纳强化程式** 🔴🔴🔴（必考）：

| 概念 | 操作 | 行为变化 | 举例 |
|------|------|---------|------|
| 正强化 | 给予愉快刺激 | 行为增加 🔺 | 表现好→表扬 |
| 负强化 🔴 | 撤销厌恶刺激 | 行为增加 🔺 | 表现好→免作业 |
| 正惩罚 | 给予厌恶刺激 | 行为减少 🔻 | 犯错→罚站 |
| 负惩罚 | 撤销愉快刺激 | 行为减少 🔻 | 犯错→取消周末活动 |
| 消退 | 不给予任何强化 | 行为减少 🔻 | 哭闹→不理睬 |

**辨析题高频** 🔴🔴🔴：
- "负强化就是惩罚" → ❌ 负强化是撤销厌恶刺激使行为增加，惩罚是使行为减少
- "惩罚比强化更有效" → ❌ 强化比惩罚更有效，惩罚只能抑制不能消除

| 理论 | 人物 | 核心观点 | 关键概念 |
|------|------|---------|---------|
| 认知-发现说 | 布鲁纳 🔴 | 学习是主动形成认知结构 | 发现学习、认知结构、学科基本结构 |

**先行组织者** 🔴🔴：先于学习任务本身呈现的引导性材料，比学习任务更抽象、概括

## 4.3 学习动机 🔴🔴🔴

### 学习动机分类 🔴

| 分类标准 | 类型 | 举例 |
|---------|------|------|
| 动力来源 | 内部动机 🔴 | 对数学本身感兴趣 |
| | 外部动机 | 为考试分数、家长奖励 |
| 社会意义 | 高尚动机 | 为中华崛起而读书 |
| | 低级动机 | 为个人名利 |
| 作用时间 | 近景直接动机 | 为期末考试及格 |
| | 远景间接动机 | 为将来当老师 |

### 学习动机理论 🔴🔴🔴

#### 成就动机理论（阿特金森）

| 类型 | 特点 | 倾向 |
|------|------|------|
| 追求成功者 | 倾向选择中等难度任务 🔴 | 力争成功 |
| 避免失败者 | 倾向选择极难或极易任务 🔴 | 回避失败 |

#### 韦纳归因理论 🔴🔴🔴（必考）

| 因素 | 控制点 | 稳定性 | 可控性 |
|------|--------|--------|--------|
| 能力 | 内部 | 稳定 | 不可控 |
| 努力 🔴🔴 | 内部 | 不稳定 | **可控** |
| 任务难度 | 外部 | 稳定 | 不可控 |
| 运气 | 外部 | 不稳定 | 不可控 |
| 身心状态 | 内部 | 不稳定 | 不可控 |
| 外界环境 | 外部 | 不稳定 | 不可控 |

**教育启示** 🔴🔴：引导学生归因于"努力"（唯一可控因素）

**辨析题** 🔴：
- "考试失败归因于能力不足会更有动力" → ❌ 归因于能力（不可控）会降低动机
- "应引导学生归因于努力" → ✅ 努力是唯一可控因素

#### 自我效能感理论（班杜拉）🔴🔴

**自我效能感**：个体对自己能否成功完成某任务的主观判断

**影响因素** 🔴：
1. 自身成败经验（最重要）🔴
2. 替代经验
3. 言语劝说
4. 情绪唤醒

#### 马斯洛需要层次理论 🔴🔴

七个层次（由低到高）：生理→安全→归属与爱→尊重→认知→审美→自我实现 🔴

**教育启示**：只有低层次需要满足后，高层次需要才能成为动力

### 学习动机与学习效果 🔴

**耶克斯-多德森定律** 🔴🔴：
- 中等强度的动机最有利于任务完成
- 任务难度越高，最佳动机水平越低
- 任务难度越低，最佳动机水平越高

## 4.4 学习迁移 🔴🔴

### 迁移分类 🔴

| 分类标准 | 类型 | 含义 |
|---------|------|------|
| 效果 | 正迁移 🔴 | 一种学习对另一种产生积极影响 |
| | 负迁移 | 一种学习对另一种产生消极影响 |
| 方向 | 顺向迁移 | 先前学习影响后续学习 |
| | 逆向迁移 | 后续学习影响先前学习 |
| 水平 | 水平迁移（横向） | 并列关系间的迁移 |
| | 垂直迁移（纵向） | 上下位关系间的迁移 |

### 迁移理论 🔴

| 理论 | 人物 | 核心观点 |
|------|------|---------|
| 形式训练说 | — | 迁移是心理官能的训练 |
| 相同要素说 | 桑代克 | 共同元素才能迁移 |
| 概括化理论 | 贾德 🔴 | 经验类化是迁移的关键 |
| 关系转换理论 | 苛勒 | 对关系的顿悟是迁移的关键 |

### 促进学习迁移的教学 🔴🔴

1. 精选教材

3. 合理安排教学程序
4. 教授学习策略
