## 第一章 教育基础知识和基本原理 🔴🔴🔴

> 🔴🔴🔴 高频考点，选择题+辨析题+简答题全覆盖

### 1.1 教育的概念

- **广义教育**：一切增进人的知识和技能、发展人的智力和体力、影响人的思想品德的活动
- **狭义教育**：学校教育，有目的、有计划、有组织地施加影响

### 1.2 教育的起源

| 学说 | 代表人物 | 核心观点 | 评价 |
|------|---------|---------|------|
| 生物起源说 | 利托尔诺、沛西·能 | 教育是动物本能 | ❌ 否认了教育的社会性 |
| 心理起源说 | 孟禄 | 教育是儿童对成人无意识模仿 | ❌ 否认了教育的意识性 |
| 劳动起源说 | 米丁斯基 | 教育起源于劳动 | ✅ 马克思主义观点 |

### 1.3 教育发展历程

| 时期 | 特征 |
|------|------|
| 原始社会教育 | 无阶级性、与生产劳动相结合、方法简单 |
| 古代社会教育 | 有阶级性、与生产劳动相脱离、具有道统性 |
| 近代社会教育 | 公立教育崛起、义务教育普及、世俗化 |
| 20 世纪后教育 | 全民化、民主化、多元化、终身化、信息化 |

### 1.4 教育与社会发展 🔴🔴

**教育与生产力**：
- 生产力制约教育：规模、速度、人才规格、内容、手段
- 教育对生产力的促进作用（经济功能）：劳动力再生产、科学知识再生产、技术创新

**教育与政治经济制度**：
- 政治经济制度制约教育：领导权、受教育的权利和程度、教育目的、内容
- 教育对政治经济制度的促进作用（政治功能）：培养人才、传播思想、促进民主

**教育与文化**：
- 文化制约教育：内容、环境、模式
- 教育对文化的促进作用（文化功能）：传承、筛选、传播、创造

### 1.5 教育与人的发展 🔴🔴

**人身心发展的规律**（口诀：顺阶不补差）：

| 规律 | 含义 | 教育启示 |
|------|------|---------|
| 顺序性 | 由低到高、由简到繁 | 循序渐进，不可拔苗助长 |
| 阶段性 | 不同年龄阶段特征不同 | 分阶段教育，不能一刀切 |
| 不平衡性 | 同一方面不同速、不同方面不同步 | 抓住关键期 |
| 互补性 | 机能之间可以互补 | 扬长避短 |
| 个别差异性 | 人与人之间有差异 | 因材施教 |

**影响人身心发展的因素**：

| 因素 | 作用 | 地位 |
|------|------|------|
| 遗传 | 提供物质前提和可能性 | 物质前提 |
| 环境 | 提供发展的现实条件 | 现实基础 |
| 学校教育 | 主导作用 | 主导 |
| 个体主观能动性 | 决定作用 | 决定因素 |

### 1.6 教育目的 🔴🔴

**教育目的理论**：

| 理论 | 代表人物 | 核心观点 |
|------|---------|---------|
| 社会本位论 | 涂尔干、孔德 | 教育为社会培养公民 |
| 个人本位论 | 卢梭、裴斯泰洛齐 | 教育为人的发展 |
| 生活本位论 | 斯宾塞 | 教育为完满生活做准备 |

**我国教育目的**：培养德智体美劳全面发展的社会主义建设者和接班人

### 1.7 教育制度

- **学制类型**：双轨制（英国）、单轨制（美国）、分支型学制（中国/苏联）
- **我国现行学制**：分支型，四段（初等→中等→高等→继续）

## 第三章 中学教学 🔴🔴🔴

> 🔴🔴🔴 高频大题区域

### 3.1 教学概述

- **教学的任务**：双基（基础知识和基本技能）+ 发展智力体力 + 培养品德 + 促进个性发展
- **教学工作的基本环节**：备课→上课→作业→辅导→评价

### 3.2 教学过程 🔴🔴

**教学过程本质**：一种特殊的认识过程

**教学过程规律**（口诀：舰长直教）：

1. **间接性规律**：直接经验与间接经验相统一
2. **发展性规律**：掌握知识与发展能力相统一
3. **教育性规律**：传授知识与思想教育相统一
4. **双边性规律**：教师主导与学生主体相统一

### 3.3 教学原则 🔴🔴🔴

| 原则 | 含义 | 关键词 |
|------|------|--------|
| 直观性原则 | 通过实物或直观教具 | 看得见 |
| 启发性原则 🔴 | 引导学生独立思考 | 不直接给答案 |
| 巩固性原则 | 复习已学知识 | 温故知新 |
| 循序渐进原则 | 按学科逻辑顺序 | 由易到难 |
| 因材施教原则 | 根据学生差异 | 不同对待 |
| 理论联系实际原则 | 学以致用 | 知行合一 |
| 量力性原则 | 适应学生发展水平 | 可接受 |
| 科学性与思想性统一原则 | 传授知识+思想教育 | 既教书又育人 |

### 3.4 教学方法 🔴🔴

| 类型 | 方法 | 适用场景 |
|------|------|---------|
| 语言传递 | 讲授法、谈话法、讨论法、读书指导法 | 知识传授 |
| 直观感知 | 演示法、参观法 | 形象认知 |
| 实际训练 | 练习法、实验法、实习作业法 | 技能培养 |
| 引导探究 | 发现法 | 思维训练 |

### 3.5 教学组织形式

- **班级授课制**：最基本的形式（夸美纽斯创立）
- **个别教学**：最早的教学组织形式
- **分组教学**：外部分组+内部分组
- **设计教学法**：克伯屈，学生自主决定学习内容
- **道尔顿制**：柏克赫斯特，自学+答疑

## 第四章 中学生学习心理 🔴🔴🔴

> 🔴🔴🔴 占分最高的章节

### 4.1 认知过程

**感觉与知觉**：
- 感觉适应：暗适应（由亮到暗）、明适应（由暗到亮）
- 知觉特性：选择性、整体性、理解性、恒常性

**注意**：

| 类型 | 有无目的 | 是否需要意志 | 例 |
|------|---------|------------|------|
| 无意注意 | 无 | 否 | 突然巨响 |
| 有意注意 | 有 | 是 | 认真听课 |
| 有意后注意 | 有 | 否 | 熟练骑车 |

**记忆**：
- 记忆过程：识记→保持→再认/回忆
- 艾宾浩斯遗忘曲线：先快后慢
- 遗忘原因：衰退说、干扰说、压抑说、提取失败说
- 知识保持方法：及时复习、分散复习、超额学习（150%）

**思维**：
- 思维特征：间接性、概括性
- 思维种类：直觉动作思维、具体形象思维、抽象逻辑思维
- 创造性思维特征：流畅性、变通性、独创性

### 4.2 学习理论 🔴🔴🔴

**行为主义**：

| 理论 | 人物 | 核心实验 |
|------|------|---------|
| 经典条件作用 | 巴甫洛夫 | 狗的唾液分泌实验 |
| 试误说 | 桑代克 | 饿猫迷箱实验 |
| 操作性条件作用 | 斯金纳 | 白鼠按压杠杆 |
| 社会学习理论 | 班杜拉 | 波波玩偶实验 |

**斯金纳强化程式** 🔴：
- 正强化：给予愉快刺激 → 行为增加
- 负强化：撤销厌恶刺激 → 行为增加
- 惩罚：给予厌恶/撤销愉快 → 行为减少
- 消退：不给予任何强化 → 行为减少

**认知主义**：

| 理论 | 人物 | 核心观点 |
|------|------|---------|
| 认知-发现说 | 布鲁纳 | 发现学习、认知结构 |
| 认知-接受说 | 奥苏贝尔 | 有意义接受学习 |
| 信息加工理论 | 加涅 | 学习是信息加工过程 |

**建构主义** 🔴：
- 知识观：知识不是绝对真理，是解释和假设
- 学生观：学生不是空容器，有已有经验
- 学习观：主动建构、社会互动、情境性
- 教学观：创设情境、合作学习

**人本主义**：
- 代表人物：马斯洛、罗杰斯
- 核心主张：以学生为中心，促进自我实现

### 4.3 学习动机 🔴🔴

**学习动机分类**：
- 内部动机 vs 外部动机
- 高尚动机 vs 低级动机
- 近景直接动机 vs 远景间接动机

**学习动机理论**：

| 理论 | 人物 | 核心观点 |
|------|------|---------|
| 成就动机理论 | 阿特金森 | 追求成功 vs 避免失败 |
| 成败归因理论 | 韦纳 | 六因素三维度 |
| 自我效能感理论 | 班杜拉 | 对自己能否成功的信念 |
| 需要层次理论 | 马斯洛 | 七层需要由低到高 |

**韦纳归因理论** 🔴🔴：

| 因素 | 维度1：控制点 | 维度2：稳定性 | 维度3：可控性 |
|------|-------------|-------------|-------------|
| 能力 | 内部 | 稳定 | 不可控 |
| 努力 | 内部 | 不稳定 | **可控** |
| 任务难度 | 外部 | 稳定 | 不可控 |
| 运气 | 外部 | 不稳定 | 不可控 |

> 教师应引导学生归因于"努力"（唯一可控因素）

### 4.4 学习迁移 🔴

- **分类**：正迁移/负迁移、顺向迁移/逆向迁移、水平迁移/垂直迁移
- **理论**：形式训练说、相同要素说、概括化理论、关系转换理论

### 4.5 学习策略

| 类型 | 具体策略 |
|------|---------|
| 认知策略 | 复述策略、精加工策略、组织策略 |
| 元认知策略 | 计划策略、监控策略、调节策略 |
| 资源管理策略 | 时间管理、环境管理、努力管理、学业求助 |

## 第五章 中学发展心理 🟡

### 5.1 中学生认知发展

- 皮亚杰认知发展阶段：感知运动→前运算→具体运算→形式运算
- 中学生处于**形式运算阶段**（11岁+）：抽象思维、假设演绎推理

### 5.2 中学生情绪与情感

- 情绪分类：心境、激情、应激
- 情感分类：道德感、理智感、美感
- 中学生情绪特点：强烈性、不稳定性、内隐性

### 5.3 中学生人格发展

- 埃里克森人格发展阶段：同一性对角色混乱（12-18岁）
- 气质类型：胆汁质、多血质、黏液质、抑郁质
- 性格结构：态度特征、意志特征、情绪特征、理智特征

## 第六章 中学心理辅导 🟡

### 6.1 心理健康标准

- 智力正常
- 情绪稳定
- 意志健全
- 人格完整
- 人际关系良好
- 适应社会

### 6.2 常见心理问题

| 问题 | 表现 | 辅导方法 |
|------|------|---------|
| 焦虑症 | 过度担忧、紧张 | 放松训练、系统脱敏 |
| 抑郁症 | 情绪低落、兴趣丧失 | 认知疗法 |
| 强迫症 | 反复出现强迫观念和行为 | 暴露疗法 |
| 恐惧症 | 对特定对象强烈恐惧 | 系统脱敏 |

### 6.3 心理辅导方法

- **系统脱敏法**：逐步暴露+放松训练
- **合理情绪疗法**（ABC 理论）：A 事件→B 信念→C 结果，改变 B 来改变 C
- **来访者中心疗法**：无条件积极关注、真诚、共情

## 第七章 中学德育 🔴🔴🔴

> 🔴🔴🔴 高频大题

### 7.1 品德结构

- 道德认识（核心）+ 道德情感 + 道德意志 + 道德行为（标志）

### 7.2 品德发展理论

| 理论 | 人物 | 阶段 |
|------|------|------|
| 道德发展阶段论 | 皮亚杰 | 他律→自律 |
| 道德发展阶段论 | 科尔伯格 | 三水平六阶段 |

**科尔伯格三水平六阶段** 🔴：

| 水平 | 阶段 | 特征 |
|------|------|------|
| 前习俗水平 | 惩罚服从 | 怕惩罚 |
| | 相对功利 | 满足自己需要 |
| 习俗水平 | 寻求认可 | 做好人 |
| | 遵守法规 | 遵守法律 |
| 后习俗水平 | 社会契约 | 法律可修改 |
| | 普遍伦理 | 内心道德准则 |

### 7.3 德育过程 🔴🔴

**基本规律**（口诀：知、情、意、行，多开端）：

1. 知、情、意、行诸因素统一发展（多端性）
2. 在活动和交往中形成
3. 思想内部矛盾转化
4. 学生思想长期、反复、逐步提高

### 7.4 德育原则 🔴🔴🔴

| 原则 | 含义 | 关键词 |
|------|------|--------|
| 导向性原则 | 理想方向 | 大方向 |
| 疏导原则 🔴 | 循循善诱 | 不堵塞 |
| 尊重与严格要求相结合 | 严慈相济 | 马卡连柯 |
| 教育的一致性与连贯性 | 统一要求 | 家庭+学校 |
| 因材施教 | 根据实际 | 不同对待 |
| 长善救失 | 发扬优点克服缺点 | 一分为二 |
| 集体教育与个别教育相结合 | 通过集体教育个人 | 平行教育 |
| 正面教育与纪律约束相结合 | 说服+规章 | 两条腿走路 |

### 7.5 德育途径与方法

**主要途径**：思想政治课（基本途径）、各科教学、班主任工作、共青团活动、劳动与社会实践

**德育方法** 🔴：
- 说服教育法（基本方法）
- 榜样示范法
- 陶冶教育法
- 实际锻炼法
- 品德评价法
- 道德修养法

## 第八章 中学班级管理与教师心理 🟡

### 8.1 班级管理

- **班集体特征**：明确目标、核心队伍、规章制度、正确舆论
- **班主任工作内容**：了解学生→组建班集体→个别教育→班级日常管理

### 8.2 课堂管理

- **群体动力**：群体凝聚力、群体规范、课堂气氛、人际关系
- **课堂纪律**：教师促成的纪律、集体促成的纪律、任务促成的纪律、自我促成的纪律

### 8.3 教师心理

- **教师成长阶段**：关注生存→关注情境→关注学生
- **教师专业发展**：新手→高级新手→胜任者→精通者→专家
- **教师心理健康**：职业倦怠（情绪衰竭、去个性化、低个人成就感）
