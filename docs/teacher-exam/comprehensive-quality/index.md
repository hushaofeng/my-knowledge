# 综合素质（中学）

> 教师资格考试科目一，满分 150 分，考试时间 120 分钟。
> 题型：单选题 29 题&#215;2 分 + 材料分析题 3 题&#215;14 分 + 写作题 1 题&#215;50 分

---

| 模块 | 优先级 | 投入时间 | 学习方法 |

| 职业理念 | &#9733;&#9733;&#9733;&#9733;&#9733; | 3 天 | 背口诀+练材料分析题 |
