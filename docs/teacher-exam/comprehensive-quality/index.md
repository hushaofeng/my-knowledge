## 第一章 职业理念 🔴🔴🔴

> 🔴🔴🔴 必考材料分析题，必须背熟三观

### 1.1 教育观

**素质教育内涵**（口诀：提速个性创两全）：

1. **以提高国民素质为根本宗旨**
2. **面向全体学生** — 不放弃任何一个学生
3. **促进学生全面发展** — 德智体美劳，不能只看分数
4. **促进学生个性发展** — 尊重差异，因材施教
5. **以培养学生创新精神和实践能力为重点** — 核心特征

**新课改的教学观**（口诀：学习过人）：

1. 教学从"教育者为中心"转向"学习者为中心"
2. 教学从"教会学生知识"转向"教会学生学习"
3. 教学从"重结论轻过程"转向"重结论更重过程"
4. 教学从"关注学科"转向"关注人"

### 1.2 学生观 🔴🔴🔴

**以人为本的学生观**（口诀：两独一发）：

1. **学生是发展的人** — 具有巨大发展潜能，处于发展过程之中
2. **学生是独特的人** — 完整的人，有自身独特性，与成人有差异
3. **学生是具有独立意义的人** — 不以教师意志为转移，学习的主体，责权主体

### 1.3 教师观 🔴🔴🔴

**新课改的教师观**（口诀：四者一开放）：

| 角色 | 从…转向… |
|------|----------|
| 知识传授者 → 学生学习的促进者和引导者 |
| 管理者 → 课程的建设者和开发者 |
| 教书匠 → 教育教学的研究者 |
| 学校教师 → 社区型开放教师 |

**教师教学行为转变**：

1. 对待师生关系：尊重、赞赏
2. 对待教学：帮助、引导
3. 对待自我：反思
4. 对待其他教育者：合作

## 第二章 教育法律法规 🔴

> 🔴 单选题为主，不需要背条文，理解核心含义即可

### 2.1 《教育法》

- 地位：教育基本法、教育母法
- 教育性质：公益性
- 方针：教育为社会主义现代化建设服务、为人民服务
- 受教育者权利：参加教学活动、获得奖学金、获得学业证书、提出申诉

### 2.2 《义务教育法》

- 义务教育年限：九年
- 特征：强制性、免费性、普及性
- 免费范围：免收学费、杂费
- 适龄儿童免试就近入学

### 2.3 《教师法》

- 教师权利：教育教学权、学术研究权、指导评价权、获取报酬权、民主管理权、进修培训权
- 教师义务：遵守宪法法律、完成教学任务、教书育人、尊重学生人格、保护学生合法权益

### 2.4 《未成年人保护法》

- 未成年年龄界定：未满 18 周岁
- 保护原则：最有利于未成年人
- 四大保护：家庭保护、学校保护、社会保护、网络保护、政府保护、司法保护

### 2.5 其他重要法律

| 法律 | 核心要点 |
|------|---------|
| 《预防未成年人犯罪法》 | 不良行为→严重不良行为→犯罪 |
| 《学生伤害事故处理办法》 | 归责原则：过错责任原则 |
| 《国家中长期教育改革发展规划纲要》 | 战略主题：坚持以人为本、全面实施素质教育 |

## 第三章 教师职业道德规范 🔴🔴🔴

> 🔴🔴🔴 必考材料分析题

### 3.1 《中小学教师职业道德规范》（2008 年修订）

**六条规范**（口诀：三爱两人一终身）：

| 条目 | 内容 | 关键词 |
|------|------|--------|
| 爱国守法 | 热爱祖国，遵纪守法 | 底线要求 |
| **爱岗敬业** 🔴 | 忠诚于人民教育事业，勤恳敬业 | 本质要求 |
| **关爱学生** 🔴 | 关心爱护全体学生，尊重人格 | 核心/灵魂 |
| 教书育人 | 遵循教育规律，实施素质教育 | 天职/使命 |
| 为人师表 | 以身作则，廉洁从教 | 内在要求 |
| **终身学习** 🔴 | 拓宽知识视野，更新知识结构 | 不竭动力 |

### 3.2 《中小学班主任工作条例》

- 班主任是班级工作的组织者、领导者
- 班主任应成为学生的人生导师
- 关心学生全面发展

### 3.3 教师职业行为准则

- 坚定政治方向
- 自觉爱国守法
- 传播优秀文化
- 潜心教书育人
- 关心爱护学生
- 加强安全防范
- 坚持言行雅正
- 秉持公平诚信
- 坚守廉洁自律
- 规范从教行为

## 第五章 基本能力 🔴🔴

> 🔴🔴 写作 50 分占大头，必须练

### 5.1 信息处理能力

- Word/Excel/PPT 基本操作
- 信息检索与筛选

### 5.2 逻辑推理能力

- 概念间关系（全同、包含、交叉、矛盾、反对）
- 命题推理（假言命题、联言命题、选言命题）
- 朴素推理

### 5.3 阅读理解能力

- 理解重要词语和句子的含义
- 筛选并整合文中信息
- 分析文章结构，把握文章思路

### 5.4 写作能力 🔴🔴🔴

**作文评分标准**：

| 等级 | 分值 | 要求 |
|------|------|------|
| 一等 | 38-50 | 立意明确、结构严谨、语言流畅、论据充分 |
| 二等 | 25-37 | 立意较明确、结构较完整、语言通顺 |
| 三等 | 12-24 | 立意不明确、结构不完整、语言不通顺 |
| 四等 | 0-11 | 偏离题意、结构混乱、语病较多 |

**写作高分策略**：

1. **立意**：必须结合教育理念（三观一德），不可脱离教育
2. **结构**：五段式（引论+3个分论点+结论）
3. **标题**：亮明观点，不用问句
4. **开头**：引用名言+过渡+亮明观点
5. **论证**：每段一个分论点+理论阐述+教育实例
