## 3.1 《中小学教师职业道德规范》（2008 年修订）

### 一、爱国守法

| 维度 | 内容 |
|------|------|
| 完整表述 | 热爱祖国，热爱人民，拥护中国共产党领导，拥护社会主义。全面贯彻国家教育方针，自觉遵守教育法律法规，依法履行教师职责权利。不得有违背党和国家方针政策的言行。 |
| 地位 | 底线要求 |
| 具体要求 | ① 拥护党的领导 ② 贯彻教育方针 ③ 遵守法律法规 ④ 不得有违背方针政策的言行 |
| 违反情形 | 课堂上发表不当政治言论；体罚学生（违法）；乱收费（违法） |
| 判断关键词 | 遵纪守法、依法执教、贯彻方针 |

**选择题易错点**：爱国守法是"底线"而非"最高要求"，教师职业道德的最高境界是"为人师表"

### 二、爱岗敬业 🔴🔴

| 维度 | 内容 |
|------|------|
| 完整表述 | 忠诚于人民教育事业，志存高远，勤恳敬业，甘为人梯，乐于奉献。对工作高度负责，认真备课上课，认真批改作业，认真辅导学生。不得敷衍塞责。 |
| 地位 | 本质要求 |
| 具体要求 | ① 忠诚教育事业 ② 勤恳敬业 ③ 认真备课上课 ④ 认真批改作业 ⑤ 认真辅导学生 |
| 高度敬业 vs 敷衍了事 | 认真备课 vs 不备课就上课；认真批改 vs 不批改；辅导学生 vs 放任不管 |
| 判断关键词 | 认真负责、勤恳、敬业、甘为人梯、乐于奉献、不敷衍 |

**材料分析判断**：
- ✅ 教师课后耐心辅导学困生 → 爱岗敬业
- ✅ 教师精心备课到深夜 → 爱岗敬业
- ❌ 教师上课照本宣科 → 违背爱岗敬业

**选择题易错点**：爱岗敬业 = 本质要求（非核心、灵魂）

### 三、关爱学生 🔴🔴🔴

| 维度 | 内容 |
|------|------|
| 完整表述 | 关心爱护全体学生，尊重学生人格，平等公正对待学生。对学生严慈相济，做学生良师益友。保护学生安全，关心学生健康，维护学生权益。不讽刺、挖苦、歧视学生，不体罚或变相体罚学生。 |
| 地位 | 核心、灵魂 🔴🔴 |
| 具体要求 | ① 关心爱护全体学生 ② 尊重学生人格 ③ 平等公正对待 ④ 严慈相济 ⑤ 保护安全健康 ⑥ 不讽刺挖苦歧视 ⑦ 不体罚变相体罚 |

**关爱 vs 溺爱辨析**：

| 维度 | 关爱学生 | 溺爱学生 |
|------|---------|---------|
| 对待错误 | 严格要求+耐心引导 | 放任不管 |
| 对待全体 | 平等公正 | 偏爱优生 |
| 对待方式 | 严慈相济 | 只慈不严 |
| 结果 | 促进学生发展 | 不利于学生成长 |

**判断关键词**：关心爱护、尊重人格、平等公正、严慈相济、不体罚

**材料分析判断**：
- ✅ 教师平等对待每个学生 → 关爱学生
- ✅ 教师对学生严格要求但尊重人格 → 关爱学生（严慈相济）
- ❌ 教师体罚学生 → 违背关爱学生
- ❌ 教师只关注优生忽视差生 → 违背关爱学生

**选择题易错点**：关爱学生 = 核心/灵魂（考频最高的定位词）

### 四、教书育人

| 维度 | 内容 |
|------|------|
| 完整表述 | 遵循教育规律，实施素质教育。循循善诱，诲人不倦，因材施教。培养学生良好品行，激发学生创新精神，促进学生全面发展。不以分数作为评价学生的唯一标准。 |
| 地位 | 天职、使命 |
| 具体要求 | ① 遵循教育规律 ② 实施素质教育 ③ 循循善诱 ④ 因材施教 ⑤ 培养良好品行 ⑥ 不以分数为唯一标准 |

**素质教育 vs 应试教育的"教书育人"区别**：

| 维度 | 应试教育的教书育人 | 素质教育的教书育人 |
|------|-------------------|-------------------|
| 目标 | 追求分数 | 全面发展 |
| 方法 | 题海战术 | 循循善诱、因材施教 |
| 评价 | 分数唯一 | 多元评价 |
| 内容 | 考什么教什么 | 德智体美劳 |

**判断关键词**：循循善诱、因材施教、不以分数为唯一标准

**选择题易错点**："不以分数为唯一标准"≠ 不看分数，是综合评价

### 五、为人师表

| 维度 | 内容 |
|------|------|
| 完整表述 | 坚守高尚情操，知荣明耻，严于律己，以身作则。衣着得体，语言规范，举止文明。关心集体，团结协作，尊重同事，尊重家长。作风正派，廉洁奉公。自觉抵制有偿家教，不利用职务之便谋取私利。 |
| 地位 | 内在要求 |
| 具体要求 | ① 以身作则 ② 衣着得体、语言规范、举止文明 ③ 尊重同事和家长 ④ 廉洁奉公 ⑤ 抵制有偿家教 🔴 ⑥ 不谋取私利 |

**廉洁从教的具体规定**：
- ✅ 不得进行有偿家教 🔴🔴
- ✅ 不得利用职务之便谋取私利
- ✅ 不得收受家长礼品

**判断关键词**：以身作则、廉洁从教、抵制有偿家教、尊重同事家长

**选择题易错点**：有偿家教 → 违背"为人师表"（非"爱岗敬业"）

### 六、终身学习 🔴🔴

| 维度 | 内容 |
|------|------|
| 完整表述 | 崇尚科学精神，树立终身学习理念，拓宽知识视野，更新知识结构。潜心钻研业务，勇于探索创新，不断提高专业素养和教育教学水平。 |
| 地位 | 不竭动力 |
| 具体要求 | ① 树立终身学习理念 ② 拓宽知识视野 ③ 更新知识结构 ④ 潜心钻研 ⑤ 勇于创新 |
| 终身学习的途径 | ① 参加培训进修 ② 阅读专业书籍 ③ 同伴互助交流 ④ 网络研修 ⑤ 教学反思 |

**判断关键词**：不断学习、更新知识、进修培训、钻研创新

**选择题易错点**：终身学习 = 不竭动力（非"基本要求"）

### 六条规范定位速查 🔴🔴

| 条目 | 地位/关键词 |
|------|------------|
| 爱国守法 | 底线要求 |

## 3.2 《中小学班主任工作条例》

### 核心内容

1. **班主任的地位**：班级工作的组织者、领导者和教育者
2. **班主任的角色**：学生的人生导师 🔴
3. **班主任的配备**：每个班级应配备一名班主任
4. **班主任的选聘**：从班级任课教师中选聘
5. **班主任津贴**：纳入绩效工资管理

### 班主任的职责与任务

| 职责 | 具体内容 |
|------|---------|
| 全面了解学生 | 思想品德、学习情况、身心健康、家庭情况 |
| 班级日常管理 | 班规制定、班干部培养、班级文化建设 |
| 组织班级活动 | 班会、文体活动、社会实践活动 |
| 协调各方教育力量 | 与任课教师沟通、家校联系 |
| 做好学生评价 | 操行评定、综合素质评价 |

### 班主任的权利

1. 有权参与学校管理
2. 有权对学生的教育管理做出决定
3. 有权获得适当津贴
4. 有权拒绝不合理的工作安排

### 选择题高频考点

1. 班主任 = 班级工作的组织者、领导者、教育者
2. 班主任 = 学生的人生导师
3. 班主任从任课教师中选聘（非专门岗位）
4. 操行评定由班主任负责

## 3.4 材料分析题专题

### 答题模板

```
该老师的行为符合/违背了教师职业道德规范的要求，具体分析如下：

第一，该老师的行为符合/违背了"……"的要求。
   教师职业道德规范规定，……（阐述规范内容）。
   材料中，该老师……（结合材料），
   这体现了/违背了……（回扣规范）。

第二，……

第三，……

综上所述，该老师的行为值得学习/需要改进。
```

### 真题示例 1：关爱学生

**材料**：陈老师在课堂上发现学生小王在玩手机。陈老师没有当众没收手机并批评小王，而是走到小王身边轻轻拍了拍他的肩膀，示意他收起手机。课后，陈老师找小王谈话，了解到小王因为父母离婚心情不好，用手机和远在外地的妈妈聊天。陈老师表示理解，同时引导小王在课余时间和妈妈联系。之后，小王上课不再玩手机，学习也更加认真了。

**问题**：请从教师职业道德的角度，评析陈老师的行为。

**参考答案**：

陈老师的行为是恰当的，符合教师职业道德规范的要求，具体分析如下：

第一，陈老师的行为符合"关爱学生"的要求。关爱学生要求教师关心爱护全体学生，尊重学生人格，严慈相济。材料中，陈老师没有当众批评小王，而是课后私下谈话，了解了小王玩手机的真实原因并表示理解，体现了尊重学生人格、关心学生心理。

第二，陈老师的行为符合"教书育人"的要求。教书育人要求教师循循善诱，因材施教，不以分数为唯一标准。材料中，陈老师不是简单禁止手机使用，而是引导小王在课余时间联系妈妈，循循善诱地解决了问题。

第三，陈老师的行为符合"为人师表"的要求。为人师表要求教师严于律己，以身作则。材料中，陈老师处理问题的方式温和而有分寸，体现了良好的师德风范。

综上所述，陈老师的行为值得学习。

### 真题示例 2：爱岗敬业 + 终身学习

**材料**：赵老师是一名乡村中学的数学教师，虽然条件艰苦，但她始终坚守岗位，认真对待每一节课。她发现传统的教学方法效果不好，就利用业余时间学习新的教学理念，参加网上研修课程。她将翻转课堂引入教学，取得了很好的效果。暑假期间，她还自费参加了省级骨干教师培训。有人劝她调到城里条件好的学校，她说："这里的孩子更需要好老师。"

**问题**：请从教师职业道德的角度，评析赵老师的行为。

**参考答案**：

赵老师的行为是恰当的，符合教师职业道德规范的要求，具体分析如下：

第一，赵老师的行为符合"爱岗敬业"的要求。爱岗敬业要求教师忠诚于人民教育事业，勤恳敬业，甘为人梯。材料中，赵老师在条件艰苦的乡村坚守岗位，面对调离机会选择留下，说"这里的孩子更需要好老师"，体现了忠诚教育事业、勤恳敬业、甘为人梯的精神。

第二，赵老师的行为符合"终身学习"的要求。终身学习要求教师拓宽知识视野，更新知识结构，不断提高专业素养。材料中，赵老师利用业余时间学习新理念、参加网上研修、自费参加培训，体现了终身学习的精神。

第三，赵老师的行为符合"关爱学生"的要求。关爱学生要求教师关心爱护全体学生。材料中，赵老师发现教学方法不好就主动改进，为了学生的发展不断提升自己，体现了对学生成长的关心。

综上所述，赵老师的行为值得学习。

### 真题示例 3：为人师表 + 教书育人

**材料**：李老师是某中学的班主任。期中考试后，有家长提出要给李老师送礼，希望李老师能安排自己孩子坐前排。李老师婉言谢绝了礼物，并解释说座位是按照身高和视力合理安排的，会公平对待每一个学生。有同事建议李老师办课外辅导班增加收入，李老师也拒绝了，说："我要把精力放在课堂教学上，让每个学生都在课堂上学会。"

**问题**：请从教师职业道德的角度，评析李老师的行为。

**参考答案**：

李老师的行为是恰当的，符合教师职业道德规范的要求，具体分析如下：

第一，李老师的行为符合"为人师表"的要求。为人师表要求教师廉洁奉公，自觉抵制有偿家教，不利用职务之便谋取私利。材料中，李老师拒绝了家长的礼物和同事办辅导班的建议，体现了廉洁从教、不以权谋私。

第二，李老师的行为符合"关爱学生"的要求。关爱学生要求教师平等公正对待学生。材料中，李老师按身高和视力安排座位，公平对待每个学生，体现了平等公正。

第三，李老师的行为符合"教书育人"的要求。教书育人要求教师潜心教学，不以分数为唯一标准。材料中，李老师把精力放在课堂教学上，让每个学生都在课堂上学会，体现了潜心教书育人。

综上所述，李老师的行为值得学习。

### 常见失分点

1. ❌ 把"有偿家教"归到"爱岗敬业" → ✅ 归到"为人师表"
2. ❌ 只写条目名称不展开内容 → 扣理论分
3. ❌ 材料和理论脱节，没有逻辑衔接 → 扣分析分
4. ❌ 混淆"核心/灵魂"（关爱学生）与"本质要求"（爱岗敬业）

## 3.5 知识速记卡

### 口诀

**三爱两人一终身**：爱国守法、爱岗敬业、关爱学生、教书育人、为人师表、终身学习

### 六条规范关键词速查

| 条目 | 一句话记忆 | 地位 |
|------|-----------|------|
| 爱国守法 | 不违法 | 底线 |
| 爱岗敬业 | 不敷衍 | 本质 |
| 关爱学生 | 不体罚 | 核心/灵魂 |
| 教书育人 | 不唯分数 | 天职 |
| 为人师表 | 不有偿家教 | 内在 |
| 终身学习 | 不停止学习 | 不竭动力 |

### 易混淆点辨析

| 混淆点 | 区别 |
|--------|------|
| 关爱学生 vs 为人师表 | 关爱学生 = 对学生好；为人师表 = 自己做得好 |
| 爱岗敬业 vs 终身学习 | 爱岗敬业 = 认真工作；终身学习 = 持续提升 |
| 有偿家教归哪条？ | 为人师表（廉洁从教），不是爱岗敬业 |
| 收受礼品归哪条？ | 为人师表（不谋私利），不是关爱学生 |
| 体罚学生归哪条？ | 关爱学生（不体罚变相体罚），不是爱国守法 |
| 不以分数为唯一标准？ | 教书育人，不是关爱学生 |
