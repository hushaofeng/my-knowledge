## 八年级上册

### 第十一章 三角形 🔴

> 🔴 **中考高频**：三角形是平面几何的基石，中考必考。

#### 11.1 与三角形有关的线段 🟡

**知识点**：三角形的分类；三角形三边关系；三角形的中线、高、角平分线。

**案例**：
- 三边关系：下列能组成三角形的是？
  - A. 2, 3, 6 → 2+3<6 ✗
  - B. 3, 4, 5 → 3+4>5 ✓
  - C. 1, 2, 3 → 1+2=3 ✗（不能等于）
- 三角形两边长为 3 和 7，第三边 x 的范围：**4 < x < 10**

#### 11.2 与三角形有关的角 🔴

**知识点**：三角形内角和定理；外角性质（外角 = 不相邻两内角之和）。

**案例**：
- △ABC 中，∠A = 50°，∠B = 70°，则∠C = **60°**，外角 ∠ACD = **120°**
- 外角定理推论：三角形的一个外角大于任何一个与它不相邻的内角

#### 11.3 多边形及其内角和 🟡

**知识点**：n 边形内角和 = (n-2)×180°；外角和 = 360°。

**案例**：
- 八边形内角和 = (8-2)×180° = **1080°**
- 一个多边形的内角和为 720°，则边数 n = **6**

### 第十二章 全等三角形 🔴🔴

> 🔴🔴 **中考核心**：全等三角形是几何证明的核心工具，广州中考几何大题几乎必涉及全等。

#### 12.1 全等三角形的性质

**知识点**：全等三角形对应边相等、对应角相等。

#### 12.2 三角形全等的判定 🔴

| 判定方法 | 条件 | 适用说明 |
|---------|------|---------|
| SSS | 三边对应相等 | 任何三角形 |
| SAS | 两边及夹角对应相等 | 注意"夹角" |
| ASA | 两角及夹边对应相等 | 任何三角形 |
| AAS | 两角及对边对应相等 | 任何三角形 |
| HL | 斜边和一直角边对应相等 | 仅直角三角形 |

> ⚠️ **易错**：SSA 不能判定全等（两边及其中一边的对角）。

**案例**：
- 如图，AB = CD，∠ABE = ∠DCE，BE = CE，求证△ABE ≌ △DCE
  - 证明：在△ABE 和△DCE 中
    - AB = CD（已知）
    - ∠ABE = ∠DCE（已知）
    - BE = CE（已知）
    - ∴ △ABE ≌ △DCE（SAS）

#### 12.3 角平分线的性质 🟡

**知识点**：角平分线上的点到角两边距离相等；逆定理。

**案例**：
- ∠AOB 的平分线上一点 P 到 OA 距离为 3，则 P 到 OB 距离 = **3**

> 🔴 **中考高频**：等腰三角形的性质与判定是重要考点。

**知识点**：等边对等角；等角对等边；三线合一；等边三角形。

**案例**：
- 等腰三角形两边长为 3 和 7，则周长 = **17**（不能是 3+3+7=13，因为 3+3<7 不满足三边关系）

### 第十四章 整式的乘法与因式分解 🟡

#### 14.1 整式的乘法 🟡

**知识点**：幂的运算（同底数幂乘法、幂的乘方、积的乘方）。

**案例**：
- a³ · a⁵ = **a⁸**（同底数幂相乘，底数不变指数相加）
- (a²)³ = **a⁶**（幂的乘方，底数不变指数相乘）
- (2ab)³ = **8a³b³**

#### 14.2 乘法公式 🔴

**平方差公式**：(a+b)(a-b) = a² - b²

**完全平方公式**：(a±b)² = a² ± 2ab + b²

**案例**：
- (x+3)(x-3) = **x² - 9**
- (2x-1)² = **4x² - 4x + 1**
- 101² = (100+1)² = 10000 + 200 + 1 = **10201**
- 99×101 = (100-1)(100+1) = 10000-1 = **9999**

> ⚠️ **中考重点**：乘法公式在因式分解、化简求值中反复使用。

#### 14.3 因式分解 🔴

> 🔴 **中考必考**：因式分解是代数运算核心，每年必出。

**方法**：
1. 提公因式法
2. 公式法（平方差、完全平方）
3. 十字相乘法（拓展）

**案例**：
- 12x²y - 8xy = **4xy(3x - 2)**
- x² - 9 = **(x+3)(x-3)**
- x² - 6x + 9 = **(x-3)²**
- 2x² - 8 = **2(x+2)(x-2)**

> ⚠️ **注意**：因式分解必须分解到不能再分（在实数范围内）。

### 第十五章 分式 🔴

> 🔴 **中考高频**：分式运算与分式方程是广州中考代数部分的重点。

#### 15.1 分式 🟡

**知识点**：分式有意义的条件（分母 ≠ 0）；分式值为 0 的条件（分子 = 0 且分母 ≠ 0）。

**案例**：
- 分式 (x-2)/(x+3) 有意义 → x ≠ **-3**
- 分式 (x²-4)/(x-2) 值为 0 → x = **-2**（x=2 时分母为 0，舍去）

#### 15.2 分式的运算 🔴

**知识点**：分式乘除、加减；通分；混合运算。

**案例**：
- 化简：(x²-4)/(x²+2x) ÷ (x-2)/(x+2)
  - = [(x+2)(x-2)] / [x(x+2)] × (x+2)/(x-2)
  - = **(x+2)/x**

> ⚠️ **中考重点**：分式化简求值是常考题型，常与条件求值结合。

#### 15.3 分式方程 🟡

**案例**：
- 解方程：2/(x-1) = 4/(x²-1)
  - 去分母：2(x+1) = 4
  - x = **1**
  - ⚠️ 检验：x=1 代入分母 x²-1=0，为增根，**原方程无解**

> ⚠️ **必须验根**：分式方程一定要检验增根！

## 八年级下册

### 第十六章 二次根式 🟡

#### 16.1 二次根式

**知识点**：√a ≥ 0（a ≥ 0）；(√a)² = a；√a² = |a|。

**案例**：
- √(x-2) 有意义 → x ≥ **2**
- √(-3)² = |-3| = **3**

#### 16.2 二次根式的乘除 🟡

**案例**：
- √12 = √(4×3) = **2√3**
- √6 ÷ √2 = √(6/2) = **√3**

#### 16.3 二次根式的加减 🟡

**案例**：
- √8 + √18 - √50 = 2√2 + 3√2 - 5√2 = **0**
- (√3 + √2)(√3 - √2) = 3 - 2 = **1**（分母有理化常用技巧）

### 第十七章 勾股定理 🔴

> 🔴 **中考高频**：勾股定理是必考知识点，常与实际问题结合。

#### 17.1 勾股定理 🔴

**知识点**：直角三角形中，a² + b² = c²。

**常见勾股数**：

| 组合 | 倍数关系 |
|------|---------|
| 3, 4, 5 | 6-8-10, 9-12-15... |
| 5, 12, 13 | 10-24-26... |
| 8, 15, 17 | — |
| 7, 24, 25 | — |

**案例**：
- 直角三角形两边为 6 和 8，第三边 = ？
  - 若为直角边：c = √(36+64) = **10**
  - 若 8 为斜边：a = √(64-36) = **2√7**

#### 17.2 勾股定理的逆定理 🟡

**知识点**：若 a² + b² = c²，则三角形为直角三角形。

**案例**：
- 三边为 5, 12, 13 → 25+144=169 ✓ → **直角三角形**
- 三边为 4, 5, 6 → 16+25=41 ≠ 36 → **不是直角三角形**

#### 17.3 勾股定理的应用 🔴

**案例**：
- 一架 10 米长的梯子靠在墙上，梯子底端距墙 6 米，梯子顶端高度？
  - h = √(100-36) = √64 = **8 米**
- 梯子顶端下滑 2 米后，底端外移多少？
  - 新高度 = 6 米，底端距离 = √(100-36) = 8 米，外移 = 8-6 = **2 米**

### 第十八章 平行四边形 🔴

> 🔴 **中考高频**：特殊平行四边形是中考几何大题的常客。

#### 18.1 平行四边形 🟡

**知识点**：性质——对边平行且相等、对角相等、对角线互相平分。

**判定**：两组对边分别平行/相等、对角线互相平分、一组对边平行且相等。

#### 18.2 特殊的平行四边形 🔴

| 图形 | 性质 | 判定 |
|------|------|------|
| 矩形 | 四个角都是直角；对角线相等 | 有一个角是直角的平行四边形；对角线相等的平行四边形 |
| 菱形 | 四条边相等；对角线互相垂直 | 四条边相等的四边形；对角线互相垂直的平行四边形 |
| 正方形 | 具有矩形和菱形的所有性质 | 既是矩形又是菱形 |

**案例**：
- 菱形对角线长为 6 和 8，则面积 = (6×8)/2 = **24**，边长 = √(9+16) = **5**
- 矩形对角线长 10，一边长 6，则面积 = 6 × √(100-36) = 6×8 = **48**

> ⚠️ **中考趋势**：广州中考近年在特殊平行四边形上常出综合题，结合全等三角形与旋转变换。

### 第十九章 一次函数 🔴🔴

> 🔴🔴 **中考核心**：一次函数是初中函数的起点，也是中考重中之重。广州中考压轴题常涉及一次函数与几何综合。

#### 19.1 函数 🟡

**知识点**：函数的概念；自变量、因变量；函数值。

**案例**：
- y = 2x + 1 中，x = 3 时 y = **7**；y = 5 时 x = **2**

#### 19.2 一次函数 🔴

**知识点**：y = kx + b（k≠0）；k>0 时 y 随 x 增大而增大，k<0 时减小。

**案例**：
- y = -2x + 6 与 x 轴交点：令 y=0 → x=3，与 y 轴交点：(0, 6)
- 两点确定一次函数：过 (1,3) 和 (2,5)
  - k = (5-3)/(2-1) = 2，b = 3-2 = 1 → y = **2x + 1**

#### 19.3 课题学习：选择方案

**案例**：
- A 方案：月租 20 元，通话 0.1 元/分钟
- B 方案：无月租，通话 0.3 元/分钟
  - A：y = 0.1x + 20；B：y = 0.3x
  - 令 0.1x+20 = 0.3x → x = 100
  - 通话 > 100 分钟选 A，< 100 分钟选 B

#### 19.4 一次函数与方程、不等式 🔴

> 🔴 **重要联系**：

| 数学对象 | 一次函数视角 |
|---------|------------|
| 一元一次方程 kx+b=0 | 函数 y=kx+b 与 x 轴交点 |
| 一元一次不等式 kx+b>0 | 函数图像在 x 轴上方部分 |
| 二元一次方程组 | 两个一次函数图像的交点 |

### 第二十章 数据的分析 🟡

#### 20.1 数据的集中趋势 🟡

**知识点**：平均数、中位数、众数。

**案例**：
- 数据 2, 3, 5, 7, 8：平均数 = 5，中位数 = 5，众数 = 无
- 数据 2, 3, 3, 5, 7：众数 = **3**

#### 20.2 数据的波动程度 🟢

**知识点**：方差——衡量数据波动大小，方差越小越稳定。

**案例**：
- 甲组：2, 4, 6, 8, 10 → 平均 6，方差 = [(4+0+4+16+36)/5] = **8**
- 乙组：5, 5, 6, 6, 8 → 平均 6，方差 = **1.6**
- 乙组数据更稳定

> 💡 **中考提示**：广州中考统计题一般出一道选择题（3分），难度不大，注意概念区分。

### 初二学习建议

1. **全等三角形是重中之重** — 熟练掌握五种判定方法，学会书写规范证明
2. **一次函数开启函数大门** — 理解数形结合思想，学会从图像中读信息
3. **因式分解要多练** — 这项技能贯穿整个初中代数
4. **勾股定理重在应用** — 广州中考喜欢考实际情境中的勾股问题
5. **特殊四边形要理解关系** — 矩形→菱形→正方形的包含关系和判定链
