> 用交互式可视化工具，把抽象数学变成"看得见、摸得着"的体验。
> 不背口诀，自己动手，5分钟总结出规律。

## 武器1：动态二次函数探索器 🎯

**痛点**：学生记不住 a、b、c 对图像的影响，"左加右减"背了就忘。

**解决方案**：3个滑块实时控制 a、b、c 的值，拖动即刻看到抛物线变化，5分钟自己总结出规律。

**教学话术**：*"我不让学生背'左加右减'，让他们自己拖滑块，5分钟自己总结出规律。"*

**核心规律**（拖滑块自己验证）：

| 参数 | 对图像的影响 | 记忆方法 |
|------|-------------|---------|
| a > 0 | 开口向上，a 越大开口越窄 | a 正上，a 负下 |
| a < 0 | 开口向下，|a| 越大开口越窄 | 想象"笑脸"和"哭脸" |
| b | 与 a 共同决定对称轴位置 | "左同右异"——ab 同号对称轴在左 |
| c | 决定与 y 轴交点 (0, c) | c 就是 y 轴截距 |

## 武器2：勾股定理"拼图证明" 🧩

**痛点**：几何证明抽象，缺乏直观感，a²+b²=c² 只是符号运算。

**解决方案**：动画演示4个直角三角形拼成大正方形的两种摆法，面积关系一目了然。

**教学话术**：*"我用动画把'数'和'形'焊在一起，学生再也忘不了 a²+b²=c²。"*

**证明思路**（点击"下一步"跟动画走）：

```
摆法一：4个直角三角形围成边长 (a+b) 的大正方形
        中间空出边长 c 的正方形
        → 大正方形面积 = 4×½ab + c² = (a+b)²

摆法二：同样的4个三角形，中间空出 a² 和 b² 两个正方形
        → 大正方形面积 = 4×½ab + a² + b² = (a+b)²

两种摆法面积相等 → c² = a² + b²  ✅
```

## 武器3：投硬币模拟器（大数定律） 🪙

**痛点**：概率概念虚无缥缈，"频率趋近概率"只是课本上的一句话。

**解决方案**：从10次到10000次投掷，实时看正面频率如何逼近0.5，折线图直观展示大数定律。

**教学话术**：*"一节课让学生点100次'投掷'，比讲100道概率题都管用。"*

**大数定律核心**：

| 投掷次数 | 正面频率典型范围 | 偏离 0.5 |
|----------|----------------|---------|
| 10 次 | 0.2 ~ 0.8 | 可能很大 |
| 100 次 | 0.4 ~ 0.6 | 较小 |
| 1000 次 | 0.47 ~ 0.53 | 很小 |
| 10000 次 | 0.49 ~ 0.51 | 极小 |
| 100000 次 | 0.498 ~ 0.502 | 几乎为0 |
