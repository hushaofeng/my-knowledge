## 案例 1：特殊角三角函数值计算 ⭐

![案例1示意图](./images/trig-case01.png)

<!-- SVG: 表格形式的特殊角三角函数值对照表，展示30°、45°、60°的sin、cos、tan值 -->

**【题目】** 求下列各式的值：

（1）sin30&deg; &times; cos60&deg; + tan45&deg;

（2）2sin60&deg; &minus; cos30&deg; + tan60&deg;

**【解析】**

熟记特殊角三角函数值：

| 角度 | sin | cos | tan |
|------|-----|-----|-----|
| 30&deg; | 1/2 | &radic;3/2 | &radic;3/3 |
| 45&deg; | &radic;2/2 | &radic;2/2 | 1 |
| 60&deg; | &radic;3/2 | 1/2 | &radic;3 |

**（1）** sin30&deg; &times; cos60&deg; + tan45&deg;

= 1/2 &times; 1/2 + 1

= 1/4 + 1 = 5/4

**（2）** 2sin60&deg; &minus; cos30&deg; + tan60&deg;

= 2 &times; &radic;3/2 &minus; &radic;3/2 + &radic;3

= &radic;3 &minus; &radic;3/2 + &radic;3

= 2&radic;3 &minus; &radic;3/2 = 3&radic;3/2

**答：（1）5/4；（2）3&radic;3/2**

**【方法提炼】** 特殊角三角函数值必须熟记。记忆技巧：sin 值从 30&deg; 到 60&deg; 递增（1/2, &radic;2/2, &radic;3/2），cos 值递减（&radic;3/2, &radic;2/2, 1/2），tan30&deg; &times; tan60&deg; = 1。考试中直接代入计算，注意 &radic;3/3 与 &radic;3 的区分。

## 案例 2：直角三角形中求三角函数值 ⭐

![案例2示意图](./images/trig-case02.png)

<!-- SVG: 直角三角形ABC，∠C=90°，AC=5，BC=12，标注AB和各角 -->

**【题目】** 在 Rt&triangle;ABC 中，&angle;C = 90&deg;，AC = 5，BC = 12，求 sinA、cosA、tanA 的值。

**【解析】**

先求斜边 AB：

AB = &radic;(AC&sup2; + BC&sup2;) = &radic;(25 + 144) = &radic;169 = 13

由三角函数定义：

sinA = 对边/斜边 = BC/AB = 12/13

cosA = 邻边/斜边 = AC/AB = 5/13

tanA = 对边/邻边 = BC/AC = 12/5

验证：sin&sup2;A + cos&sup2;A = (12/13)&sup2; + (5/13)&sup2; = 144/169 + 25/169 = 169/169 = 1 ✓

**答：sinA = 12/13，cosA = 5/13，tanA = 12/5**

**【方法提炼】** 已知直角三角形两边求三角函数值：先由勾股定理求第三边，再按定义代入。口诀：sin 对斜、cos 邻斜、tan 对邻。务必验证 sin&sup2;A + cos&sup2;A = 1。

## 案例 3：解直角三角形（仰角俯角）⭐⭐

![案例3示意图](./images/trig-case03.png)

<!-- SVG: 建筑物AB底部为B，顶部为A，观测点C距B为60m，仰角为30°，标注各边和角 -->

**【题目】** 如图，在地面上的 C 处测得建筑物顶部 A 的仰角为 30&deg;，C 到建筑物底部 B 的距离为 60 m，求建筑物 AB 的高（结果保留根号）。

**【解析】**

在 Rt&triangle;ABC 中，&angle;B = 90&deg;，&angle;C = 30&deg;，BC = 60 m。

tan&angle;C = AB/BC

AB = BC &times; tan30&deg; = 60 &times; &radic;3/3 = 20&radic;3

**答：建筑物 AB 高 20&radic;3 m**

**【方法提炼】** 仰角问题基本模型：已知水平距离和仰角求高度，tan&angle; = 对边/邻边。关键：正确识别仰角所在的直角三角形，仰角是水平线与视线的夹角（向上）。

## 案例 4：坡度坡角问题 ⭐⭐

![案例4示意图](./images/trig-case04.png)

<!-- SVG: 坡面示意图，水平距离l=10m，坡角α=30°，标注坡面长度s和铅直高度h -->

**【题目】** 一段坡面的坡角为 30&deg;，水平距离为 10 m。求：

（1）坡面的铅直高度 h；

（2）坡面长度 s；

（3）坡度 i。

**【解析】**

坡角 &alpha; = 30&deg;，水平距离 l = 10 m。

**（1）** tan&alpha; = h/l

h = l &times; tan30&deg; = 10 &times; &radic;3/3 = 10&radic;3/3 m

**（2）** cos&alpha; = l/s

s = l/cos30&deg; = 10/(&radic;3/2) = 20&radic;3/3 m

（或 s = &radic;(l&sup2; + h&sup2;) = &radic;(100 + 100/3) = &radic;(400/3) = 20&radic;3/3）

**（3）** 坡度 i = h/l = tan&alpha; = tan30&deg; = &radic;3/3 = 1/&radic;3

写成 i = 1:&radic;3

**答：（1）h = 10&radic;3/3 m；（2）s = 20&radic;3/3 m；（3）i = 1:&radic;3**

**【方法提炼】** 坡度 i = 铅直高度/水平距离 = tan&alpha;（坡角）。坡角 &alpha; 是坡面与水平面的夹角。注意：坡度是比值（正切值），不是角度。i = 1:n 表示水平距离为铅直高度的 n 倍。

## 案例 5：方向角问题 ⭐⭐

![案例5示意图](./images/trig-case05.png)

<!-- SVG: 平面图，A点在左，B点在右，A处北偏东30°方向，B处南偏西45°方向，AB=100海里 -->

**【题目】** 如图，海面上 A、B 两岛相距 100 海里。从 A 岛看 B 岛在北偏东 30&deg; 方向，从 B 岛看 A 岛在南偏西 45&deg; 方向。一艘渔船从 A 岛出发沿正北方向航行，求渔船离 B 岛最近时距 B 岛的距离。

**【解析】**

设渔船沿正北方向航行到 C 处时距 B 最近，则 BC &perp; AC，即 &angle;ACB = 90&deg;。

在 A 处，B 在北偏东 30&deg; 方向，即 &angle;BAC = 30&deg;（正北方向与 AB 的夹角）。

在 Rt&triangle;ABC 中，&angle;ACB = 90&deg;，&angle;BAC = 30&deg;，AB = 100。

BC = AB &times; sin&angle;BAC = 100 &times; sin30&deg; = 100 &times; 1/2 = 50

**答：渔船离 B 岛最近距离为 50 海里**

**【方法提炼】** 方向角问题：北偏东 &alpha; 表示从正北方向顺时针转 &alpha; 度。求最短距离即作垂线段。关键是把方向角转化为三角形内角。注意：方位角是从正北顺时针量到目标方向线的角。

## 案例 6：仰角俯角综合（建筑物高度）⭐⭐⭐

![案例6示意图](./images/trig-case06.png)

<!-- SVG: 建筑物AB，地面点C和D在AB同侧，CD=40m，C处仰角30°，D处仰角45°，标注各边和角 -->

**【题目】** 如图，建筑物 AB 旁有一斜坡，在坡底 C 处测得建筑物顶部 A 的仰角为 30&deg;，在坡顶 D 处测得 A 的仰角为 45&deg;。已知 C、D 在同一水平面上，CD = 40 m，求建筑物 AB 的高度（结果保留根号）。

**【解析】**

设 AB = h m。

在 Rt&triangle;ABD 中，&angle;ADB = 45&deg;，

BD = AB/tan45&deg; = h/1 = h

在 Rt&triangle;ABC 中，&angle;ACB = 30&deg;，

BC = AB/tan30&deg; = h/(&radic;3/3) = h &times; 3/&radic;3 = &radic;3h

由 BC &minus; BD = CD：

&radic;3h &minus; h = 40

h(&radic;3 &minus; 1) = 40

h = 40/(&radic;3 &minus; 1) = 40(&radic;3 + 1)/((&radic;3 &minus; 1)(&radic;3 + 1)) = 40(&radic;3 + 1)/(3 &minus; 1) = 40(&radic;3 + 1)/2 = 20(&radic;3 + 1)

**答：建筑物 AB 高 20(&radic;3 + 1) m**

**【方法提炼】** 双仰角求高度模型：设高为 h，分别用 tan 表示两个水平距离，再利用两观测点的水平距离差列方程。分母有理化是常见化简步骤。推广：若两观测点在异侧（建筑物两侧），则 BC + BD = CD。

## 案例 7：航海问题 ⭐⭐⭐

![案例7示意图](./images/trig-case07.png)

<!-- SVG: 海面上灯塔A，船从B出发沿北偏东60°方向航行2小时到C，B处看A在北偏东30°，C处看A在西北方向，求BC距离 -->

**【题目】** 如图，一艘轮船在 B 处观测灯塔 A 在北偏东 30&deg; 方向。轮船沿北偏东 75&deg; 方向航行 10 海里到达 C 处，此时观测灯塔 A 在正西方向。求灯塔 A 到航线 BC 的距离。

**【解析】**

建立坐标系分析角度：

B 处看 A 在北偏东 30&deg;，即 &angle;NBA = 30&deg;（N 为正北方向）。

轮船沿北偏东 75&deg; 航行，即航线 BC 与正北方向夹角 75&deg;。

&angle;ABC = 75&deg; &minus; 30&deg; = 45&deg;（航线方向与 BA 方向的夹角）。

C 处看 A 在正西方向，即 CA 方向为正西。

航线 BC 方向为北偏东 75&deg;，正西方向为西偏南 90&deg; 与北偏东 75&deg; 的夹角：

&angle;BCA = 180&deg; &minus; 75&deg; &minus; 45&deg; = 60&deg;

（验证：三角形内角和 = 45&deg; + 60&deg; + 75&deg; = 180&deg; ✓）

在 &triangle;ABC 中，&angle;ABC = 45&deg;，&angle;BCA = 60&deg;，BC = 10。

作 AD &perp; BC 于 D，设 AD = h。

在 Rt&triangle;ABD 中，&angle;ABD = 45&deg;，BD = h/tan45&deg; = h。

在 Rt&triangle;ACD 中，&angle;ACD = 60&deg;，CD = h/tan60&deg; = h/&radic;3 = &radic;3h/3。

BD + CD = BC：h + &radic;3h/3 = 10

h(1 + &radic;3/3) = 10

h(3 + &radic;3)/3 = 10

h = 30/(3 + &radic;3) = 30(3 &minus; &radic;3)/((3 + &radic;3)(3 &minus; &radic;3)) = 30(3 &minus; &radic;3)/(9 &minus; 3) = 30(3 &minus; &radic;3)/6 = 5(3 &minus; &radic;3)

**答：灯塔 A 到航线 BC 的距离为 5(3 &minus; &radic;3) 海里**

**【方法提炼】** 航海问题核心：将方向角转化为三角形内角。步骤：(1) 标注所有方向角；(2) 计算航线与观测线的夹角；(3) 用正弦定理或作高求解。注意方向角与三角形内角的转化关系。

## 案例 8：多直角三角形组合 ⭐⭐⭐

![案例8示意图](./images/trig-case08.png)

<!-- SVG: 两栋建筑物AB和CD并排，AB高30m，从A看D仰角30°，从B看C俯角45°，标注各边和角 -->

**【题目】** 如图，两栋建筑物 AB 和 CD，AB 高 30 m。从 A 处观测 D 的仰角为 30&deg;，从 B 处观测 C 的俯角为 45&deg;。A、B、C、D 在同一竖直平面内，求建筑物 CD 的高（结果保留根号）。

**【解析】**

设两建筑物之间水平距离为 d m，CD 的高为 h m。

从 A 看 D 仰角 30&deg;：D 比 A 高出 d &times; tan30&deg; = &radic;3d/3。

所以 h = AB + &radic;3d/3 = 30 + &radic;3d/3 &hellip;&hellip; &circ;1

从 B 看 C 俯角 45&deg;：C 比 B 低 d &times; tan45&deg; = d（俯角向下，C 在 B 下方 d 处）。

所以 C 的高度 = AB &minus; d = 30 &minus; d。

而 C 的高度即 CD 的高 h = 30 &minus; d &hellip;&hellip; &circ;2

由 &circ;1 和 &circ;2：30 + &radic;3d/3 = 30 &minus; d

&radic;3d/3 = &minus;d

这说明 C 不能在 B 下方。重新分析：

从 B 处观测 C 的俯角为 45&deg;，C 是 CD 的底部。如果 C 在地面上，则 C 的高度为 0，俯角应更大。题目说俯角 45&deg;，说明 C 的高度不为 0。

重新理解：从 B 看 C（CD 的底部），俯角 45&deg;。

C 的高度 = AB &minus; d &times; tan45&deg; = 30 &minus; d。

CD 的高 h，从 A 看 D（CD 顶部）仰角 30&deg;：

D 的高度 = AB + d &times; tan30&deg; = 30 + &radic;3d/3。

h = D 的高度 &minus; C 的高度 = (30 + &radic;3d/3) &minus; (30 &minus; d) = &radic;3d/3 + d = (&radic;3 + 3)d/3

但 C 的高度 30 &minus; d &ge; 0，d &le; 30。

还需一个条件来定 d。题目未给出更多条件，但 h 的表达式中含 d。

重新审题：从 B 看 C 俯角 45&deg;，C 为建筑物 CD 的底部。如果两建筑物底部在同一水平面上，则 C 的高度为 0。

从 B（高 30 m）看地面上的 C，俯角 45&deg;：

d = 30/tan45&deg; = 30 m。

从 A（高 30 m）看 D 仰角 30&deg;：

D 比 A 高 &radic;3d/3 = &radic;3 &times; 30/3 = 10&radic;3 m。

h = 30 + 10&radic;3 m。

**答：建筑物 CD 高 (30 + 10&radic;3) m**

**【方法提炼】** 多建筑物高度问题：分别以每个观测点为基准，用仰角/俯角建立高度关系。关键：俯角向下时目标低于观测点，仰角向上时目标高于观测点。同一水平面上的底部高度为 0 是隐含条件。

## 案例 9：三角函数与圆综合 ⭐⭐⭐⭐

![案例9示意图](./images/trig-case09.png)

<!-- SVG: 圆O半径为5，AB为直径，C在圆上，∠BAC=30°，过C作CD⊥AB于D，标注各线段 -->

**【题目】** 如图，AB 是 &odot;O 的直径，AB = 10，C 是 &odot;O 上一点，&angle;BAC = 30&deg;。过 C 作 CD &perp; AB 于 D。

（1）求 AC 和 BC 的长；

（2）求 CD 和 AD 的长；

（3）求阴影部分（弓形 AC）的面积。

**【解析】**

AB 是直径，&angle;ACB = 90&deg;（直径所对的圆周角为直角）。

**（1）** 在 Rt&triangle;ABC 中，&angle;ACB = 90&deg;，&angle;BAC = 30&deg;，AB = 10。

BC = AB &times; sin30&deg; = 10 &times; 1/2 = 5

AC = AB &times; cos30&deg; = 10 &times; &radic;3/2 = 5&radic;3

**（2）** 在 Rt&triangle;ACD 中，&angle;ADC = 90&deg;，&angle;CAD = 30&deg;。

CD = AC &times; sin30&deg; = 5&radic;3 &times; 1/2 = 5&radic;3/2

AD = AC &times; cos30&deg; = 5&radic;3 &times; &radic;3/2 = 15/2

**（3）** 弓形 AC 的面积 = 扇形 OAC 面积 &minus; &triangle;OAC 面积

&angle;AOC = 2&times;&angle;ABC = 2 &times; 60&deg; = 120&deg;（圆心角 = 2 &times; 圆周角）

（因 &angle;ABC = 180&deg; &minus; 90&deg; &minus; 30&deg; = 60&deg;）

扇形 OAC 面积 = 120/360 &times; &pi; &times; 5&sup2; = 25&pi;/3

&triangle;OAC：OA = OC = 5，&angle;AOC = 120&deg;

S&triangle;OAC = 1/2 &times; OA &times; OC &times; sin120&deg; = 1/2 &times; 5 &times; 5 &times; &radic;3/2 = 25&radic;3/4

弓形面积 = 25&pi;/3 &minus; 25&radic;3/4 = 25(4&pi; &minus; 3&radic;3)/12

**答：（1）AC = 5&radic;3，BC = 5；（2）CD = 5&radic;3/2，AD = 15/2；（3）弓形面积 = 25(4&pi; &minus; 3&radic;3)/12**

**【方法提炼】** 圆与三角函数综合的关键桥梁：(1) 直径所对圆周角为 90&deg;，自然构造直角三角形；(2) 圆心角 = 2 &times; 圆周角；(3) 弓形面积 = 扇形面积 &minus; 三角形面积。垂径定理和圆周角定理是构造直角三角形的常用工具。

## 案例 10：三角函数与二次函数综合压轴 ⭐⭐⭐⭐

![案例10示意图](./images/trig-case10.png)

<!-- SVG: 坐标系中抛物线y=-x²+2x+3，与x轴交于A(-1,0)和B(3,0)，与y轴交于C(0,3)，P在抛物线上，标注各点 -->

**【题目】** （广州中考风格压轴）已知抛物线 y = &minus;x&sup2; + 2x + 3 与 x 轴交于 A、B 两点（A 在 B 左侧），与 y 轴交于点 C。

（1）求 A、B、C 的坐标及 tan&angle;ABC 的值；

（2）在抛物线上是否存在点 P（第一象限内），使 &angle;BCP = 45&deg;？若存在，求点 P 坐标；

（3）在（1）的条件下，求 &triangle;BCP 面积的最大值。

**【解析】**

**（1）** 令 y = 0：&minus;x&sup2; + 2x + 3 = 0，x&sup2; &minus; 2x &minus; 3 = 0，(x &minus; 3)(x + 1) = 0。

A(&minus;1, 0)，B(3, 0)。

令 x = 0：y = 3。C(0, 3)。

tan&angle;ABC：在 Rt&triangle;OBC 中（O 为原点），&angle;OBC 为 &angle;ABC，

OB = 3，OC = 3。

tan&angle;ABC = OC/OB = 3/3 = 1，&angle;ABC = 45&deg;。

**答：A(&minus;1, 0)，B(3, 0)，C(0, 3)，tan&angle;ABC = 1**

**（2）** &angle;ABC = 45&deg;，若 &angle;BCP = 45&deg;，则 &angle;ABP = &angle;ABC + &angle;BCP = 45&deg; + 45&deg; = 90&deg; 或 &angle;BCP = 45&deg; 在 BC 的另一侧。

**情况一：P 在 BC 上方**

设 BP 与 x 轴正方向夹角为 &theta;，则 &theta; = &angle;ABC + &angle;BCP = 45&deg; + 45&deg; = 90&deg;。

即 BP &perp; x 轴，P 的横坐标为 3，P 在抛物线上：

y = &minus;9 + 6 + 3 = 0，即 P(3, 0) = B，不合题意。

**情况二：&angle;BCP = 45&deg;，P 在 BC 上方另一侧**

由 &angle;ABC = 45&deg;，&angle;BCP = 45&deg;，在 &triangle;BCP 中：

设 BP 方向角为 &alpha;（从 x 轴正方向量起），则 &alpha; = 180&deg; &minus; &angle;ABP。

BC 方向：从 B(3,0) 到 C(0,3)，方向角 = 135&deg;。

&angle;BCP = 45&deg;，CP 方向 = BC 方向 + 45&deg; = 135&deg; + 45&deg; = 180&deg;（指向左）或 135&deg; &minus; 45&deg; = 90&deg;（指向上）。

CP 方向 90&deg;（竖直向上）：CP 所在直线为 x = 0（即 y 轴），P 在 y 轴上且在抛物线上，x = 0 时 y = 3，即 C(0,3)，不合题意。

CP 方向 180&deg;（水平向左）：CP 所在直线为 y = 3（过 C(0,3) 水平线），与抛物线交点：

&minus;x&sup2; + 2x + 3 = 3，&minus;x&sup2; + 2x = 0，x(&minus;x + 2) = 0

x = 0（即 C 点）或 x = 2。

P(2, 3)。

验证：P(2, 3) 在第一象限 ✓

BC = &radic;(9 + 9) = 3&radic;2

BP = &radic;((3&minus;2)&sup2; + (0&minus;3)&sup2;) = &radic;(1+9) = &radic;10

CP = &radic;((0&minus;2)&sup2; + (3&minus;3)&sup2;) = 2

在 &triangle;BCP 中，BC = 3&radic;2，CP = 2，BP = &radic;10。

cos&angle;BCP = (BC&sup2; + CP&sup2; &minus; BP&sup2;)/(2&times;BC&times;CP) = (18 + 4 &minus; 10)/(2 &times; 3&radic;2 &times; 2) = 12/(12&radic;2) = &radic;2/2

**（3）** S&triangle;BCP = 1/2 &times; BC &times; CP &times; sin&angle;BCP

**【方法提炼】** 三角函数+二次函数压轴题解题策略：(1) 先求关键点坐标和关键角（本题 &angle;ABC = 45&deg; 是突破口）；(2) 角度条件转化方向角，再转化为直线方程，与抛物线联立求交点；(3) 面积公式 S = 1/2&times;a&times;b&times;sinC 是联系三角函数与面积的桥梁。核心：把角度条件转化为方向，方向转化为坐标关系。

### 锐角三角函数核心公式

- **定义**：sinA = 对边/斜边，cosA = 邻边/斜边，tanA = 对边/邻边
- **恒等式**：sin&sup2;A + cos&sup2;A = 1，tanA = sinA/cosA
- **互余关系**：sinA = cos(90&deg; &minus; A)，cosA = sin(90&deg; &minus; A)
- **特殊角**：

| &alpha; | sin&alpha; | cos&alpha; | tan&alpha; |
|---------|-----------|-----------|-----------|
| 30&deg; | 1/2 | &radic;3/2 | &radic;3/3 |
| 45&deg; | &radic;2/2 | &radic;2/2 | 1 |
| 60&deg; | &radic;3/2 | 1/2 | &radic;3 |

### 解题通法

1. **画图标注**：将所有已知量（边、角、仰角、俯角、方向角）标注在图上
2. **构造直角三角形**：作高/垂线，将一般三角形拆分为直角三角形
3. **选对公式**：已知一角一边求另一边 &mdash;&mdash; 选含该角的三角函数
4. **列方程求解**：多个直角三角形共享的边是列方程的桥梁
5. **检验**：验证 sin&sup2;A + cos&sup2;A = 1，检验结果的合理性
