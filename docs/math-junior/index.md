### 高频易错点

| 易错点 | 正确理解 |
|--------|---------|
| (-3)² 与 -3² | (-3)²=9，-3²=-9 |
| 增根 | 分式方程必须检验 |
| SSA 判定全等 | SSA 不能判定，只有 HL 适用于直角三角形 |
| 不等式变号 | 两边同乘负数，不等号方向改变 |
| 相似比与面积比 | 面积比 = 相似比的平方 |
| 垂径定理条件 | 弦不能是直径 |

### 备考时间线

```
9月-1月   新课学习（九年级上下册）
2月-4月   一轮复习（按章节系统过）
4月-5月   二轮复习（专题突破）
5月-6月   冲刺模拟（真题+模拟卷）
```
