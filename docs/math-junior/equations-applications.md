# 一元一次方程应用题 — 广州中考典型试题与解析

> **一元一次方程应用题是广州中考必考题型，常见于选择题或应用大题，分值 6-8 分。**
> 难度标注：⭐ 基础 | ⭐⭐ 中等 | ⭐⭐⭐ 较难 | ⭐⭐⭐⭐ 压轴

### 案例 1：行程问题（追及） ⭐

![案例1示意图](./images/equation-case01.png)

<!-- SVG: 甲乙两人从同地出发，甲先出发2小时，乙追赶，标注速度和追及点 -->

**【题目】**

甲、乙两人从A地出发沿同一条路去B地，甲步行速度为4 km/h，乙骑自行车速度为12 km/h。甲先出发2小时后乙才出发，问乙出发后几小时能追上甲？

**【解析】**

设乙出发后 x 小时追上甲。

此时：
- 甲行走的时间 = (x + 2) 小时
- 乙行走的时间 = x 小时

追及条件：两人行走路程相等。

$$4(x + 2) = 12x$$

$$4x + 8 = 12x$$

$$8 = 8x$$

$$x = 1$$

**答：** 乙出发后 1 小时能追上甲。

**【方法提炼】**

行程问题核心关系：路程 = 速度 &times; 时间。追及问题的关键是找到"路程相等"的等量关系，注意区分各自行走的时间。设追及时间 x，则先出发者的行走时间要加上提前量。

### 案例 2：工程问题 ⭐

![案例2示意图](./images/equation-case02.png)

<!-- SVG: 一项工程被分成两部分，甲独做和乙独做的效率对比示意 -->

**【题目】**

一项工程，甲单独做需要15天完成，乙单独做需要10天完成。若甲先单独做3天后，两人合做完成剩余部分，问还需合做多少天才能完成这项工程？

**【解析】**

设还需合做 x 天完成。

将整个工程看作"1"：
- 甲的工作效率 = 1/15
- 乙的工作效率 = 1/10

甲先做3天完成的工作量：3 &times; 1/15 = 1/5

两人合做一天完成的工作量：1/15 + 1/10 = 2/30 + 3/30 = 5/30 = 1/6

等量关系：甲先做的工作量 + 合做的工作量 = 1

$$\frac{1}{5} + \frac{1}{6}x = 1$$

$$\frac{1}{6}x = \frac{4}{5}$$

$$x = \frac{4}{5} \times 6 = \frac{24}{5} = 4.8$$

**答：** 还需合做 4.8 天（即4天又4小时）才能完成这项工程。

**【方法提炼】**

工程问题核心：设总工程量为"1"，工作效率 = 1 &divide; 独做时间。等量关系为"各部分工作量之和 = 1"。注意分阶段计算工作量，先做部分 + 合做部分 = 总量。

### 案例 3：利润与打折问题 ⭐⭐

![案例3示意图](./images/equation-case03.png)

<!-- SVG: 商品进价、标价、折扣、利润之间的关系示意 -->

**【题目】**

某商店以每件80元的价格购进一批商品，加价后按标价的八折出售，仍可获利20%。求该商品的标价是多少元？

**【解析】**

设标价为 x 元。

分析各量关系：
- 进价 = 80 元
- 售价 = 标价 &times; 0.8 = 0.8x 元
- 利润率 = 20%，即售价 = 进价 &times; (1 + 20%)

$$0.8x = 80 \times (1 + 20\%)$$

$$0.8x = 80 \times 1.2$$

$$0.8x = 96$$

$$x = 120$$

**答：** 该商品的标价为 120 元。

**【方法提炼】**

利润问题核心公式：售价 = 进价 &times; (1 + 利润率)，售价 = 标价 &times; 折扣。打折问题的常见错误是混淆"标价"与"售价"，要分清：标价是打折前的价格，售价是打折后顾客实际支付的金额。利润率是相对于进价而言的。

### 案例 4：配套问题 ⭐⭐

![案例4示意图](./images/equation-case04.png)

<!-- SVG: 工人分组生产螺钉和螺母，按比例配套示意 -->

**【题目】**

某车间有22名工人，每人每天可以生产螺钉1200个或螺母2000个。1个螺钉需要配2个螺母，为使每天生产的螺钉和螺母刚好配套，应安排多少名工人生产螺钉？

**【解析】**

设安排 x 名工人生产螺钉，则 (22 - x) 名工人生产螺母。

每天生产的数量：
- 螺钉：1200x 个
- 螺母：2000(22 - x) 个

配套条件：螺母数量 = 2 &times; 螺钉数量

$$2000(22 - x) = 2 \times 1200x$$

$$44000 - 2000x = 2400x$$

$$44000 = 4400x$$

$$x = 10$$

生产螺母的工人数：22 - 10 = 12（名）

验证：螺钉 1200 &times; 10 = 12000 个，螺母 2000 &times; 12 = 24000 个，24000 = 2 &times; 12000，配套正确。

**答：** 应安排 10 名工人生产螺钉，12 名工人生产螺母。

**【方法提炼】**

配套问题的关键是理解"配套比例"并据此建立等量关系。如1个螺钉配2个螺母，则螺母数 = 2 &times; 螺钉数。常见错误是将比例关系写反，建议写出"1螺钉 : 2螺母"后再列等式。总人数约束：两组人数之和 = 总人数。

### 案例 5：年龄问题 ⭐⭐

![案例5示意图](./images/equation-case05.png)

<!-- SVG: 父子年龄时间线，标注现在与过去的年龄关系 -->

**【题目】**

父亲今年45岁，儿子今年15岁。几年前父亲的年龄是儿子年龄的5倍？

**【解析】**

设 x 年前父亲的年龄是儿子年龄的5倍。

此时：
- 父亲的年龄 = 45 - x
- 儿子的年龄 = 15 - x

等量关系：父亲年龄 = 5 &times; 儿子年龄

$$45 - x = 5(15 - x)$$

$$45 - x = 75 - 5x$$

$$4x = 30$$

$$x = 7.5$$

验证：7.5年前父亲 37.5 岁，儿子 7.5 岁，37.5 = 5 &times; 7.5，正确。

**答：** 7.5 年前父亲的年龄是儿子年龄的5倍。

**【方法提炼】**

年龄问题核心：经过相同的时间，每个人增加（或减少）的年龄相同——这是隐含的等量关系基础。设"几年前"为 x，则各人当时年龄 = 现在年龄 - x；设"几年后"为 x，则各人当时年龄 = 现在年龄 + x。注意结果是否合理（年龄不能为负）。

### 案例 6：数字问题 ⭐⭐

![案例6示意图](./images/equation-case06.png)

<!-- SVG: 三位数 ABC，标注百位、十位、个位的位值关系 -->

**【题目】**

一个两位数，十位上的数字是个位上数字的2倍少1。如果把十位上的数字与个位上的数字对调，得到的新两位数比原两位数小36。求原两位数。

**【解析】**

设个位数字为 x，则十位数字为 (2x - 1)。

原两位数 = 10 &times; (2x - 1) + x = 20x - 10 + x = 21x - 10

对调后的新两位数 = 10x + (2x - 1) = 12x - 1

等量关系：原两位数 - 新两位数 = 36

$$(21x - 10) - (12x - 1) = 36$$

$$9x - 9 = 36$$

$$9x = 45$$

$$x = 5$$

十位数字 = 2 &times; 5 - 1 = 9

原两位数 = 90 + 5 = 95

验证：对调后为 59，95 - 59 = 36，正确。

**答：** 原两位数为 95。

**【方法提炼】**

数字问题核心：用数位表示多位数。两位数 = 10 &times; 十位数字 + 个位数字，三位数 = 100 &times; 百位数字 + 10 &times; 十位数字 + 个位数字。注意数字取值范围：各位数字为 0~9 的整数，首位数字不能为0。对调后要重新用位值展开，不能简单交换。

### 案例 7：方案设计题（租车问题） ⭐⭐⭐

![案例7示意图](./images/equation-case07.png)

<!-- SVG: 大客车和小客车组合运送学生，标注座位数和费用 -->

**【题目】**

某学校组织240名师生乘车去参加社会实践活动，可租用的车辆有两种：甲种客车每辆载客45人，租金400元；乙种客车每辆载客30人，租金280元。要求租用车辆总数不超过6辆且每辆车都坐满。

（1）共有几种租车方案？

（2）哪种方案最省钱？最低费用是多少？

**【解析】**

设租用甲种客车 x 辆，则租用乙种客车 y 辆。

条件1：载客量 = 240

$$45x + 30y = 240$$

化简：$3x + 2y = 16$

条件2：车辆总数 &le; 6

$$x + y \le; 6$$

由 $3x + 2y = 16$ 得 $y = \frac{16 - 3x}{2}$

因为 x、y 为非负整数，逐一检验：

| x | y = (16-3x)/2 | x + y | 是否 &le; 6 |
|---|---|---|---|
| 0 | 8 | 8 | &times; |
| 2 | 5 | 7 | &times; |
| 4 | 2 | 6 | &check; |
| 6 | -1 | — | &times; |

所以 x = 4，y = 2 是唯一满足条件的方案。

方案：租甲种客车4辆，乙种客车2辆。

费用 = 4 &times; 400 + 2 &times; 280 = 1600 + 560 = 2160（元）

**答：** （1）共有1种租车方案：租甲种客车4辆、乙种客车2辆。（2）最低费用为2160元。

**【方法提炼】**

方案设计题的一般步骤：①设未知数，列方程（或不等式）；②根据整数约束筛选可行方案；③比较各方案的优劣。当方程的整数解唯一时，方案唯一。注意审题中的"不超过""至少""恰好"等关键词，它们决定等式或不等式的方向。

### 案例 8：分段计费问题 ⭐⭐⭐

![案例8示意图](./images/equation-case08.png)

<!-- SVG: 阶梯水价/电价示意，不同用量区间对应不同单价 -->

**【题目】**

广州市居民用水实行阶梯水价：每月用水量不超过20立方米的部分按2.5元/立方米收费；超过20立方米但不超过30立方米的部分按3.5元/立方米收费；超过30立方米的部分按5元/立方米收费。某户居民5月份交水费82元，求该户5月份用水量是多少立方米？

**【解析】**

先判断用水量所在区间：

- 若用水量 &le; 20 m&sup3;，水费 &le; 20 &times; 2.5 = 50 元
- 若用水量在 20~30 m&sup3;，水费 = 50 + 超出部分 &times; 3.5，最高 = 50 + 10 &times; 3.5 = 85 元
- 若用水量 > 30 m&sup3;，水费 > 85 元

由于 82 元在 50~85 之间，用水量在 20~30 m&sup3; 之间。

设5月份用水量为 x m&sup3;（20 < x &le; 30）。

$$50 + 3.5(x - 20) = 82$$

$$3.5(x - 20) = 32$$

$$x - 20 = \frac{32}{3.5} = \frac{64}{7} \approx 9.14$$

$$x = 20 + \frac{64}{7} = \frac{204}{7} \approx 29.14$$

验证：50 + 3.5 &times; (204/7 - 20) = 50 + 3.5 &times; 64/7 = 50 + 224/7 &divide; 7 &times; 3.5 → 50 + 32 = 82 ✓

**答：** 该户5月份用水量为 204/7（约29.1）立方米。

**【方法提炼】**

分段计费题步骤：①计算各段分界点对应的费用，判断未知量所在区间；②仅在对应区间列方程；③用分段式表示总费用 = 第一段费用 + 第二段费用 + ...。常见错误：未判断区间直接列方程，或将分段标准弄混。务必先算分界点费用来定位。

### 案例 9：比赛积分问题 ⭐⭐⭐

![案例9示意图](./images/equation-case09.png)

<!-- SVG: 球队积分榜示意，胜场、平场、负场与积分关系 -->

**【题目】**

某篮球联赛中，每场比赛都要分出胜负，每队胜一场得2分，负一场得1分。某队参加了14场比赛，共得23分。问该队胜、负各多少场？

**【解析】**

设该队胜 x 场，则负 (14 - x) 场。

等量关系：胜场积分 + 负场积分 = 总积分

$$2x + 1 \times (14 - x) = 23$$

$$2x + 14 - x = 23$$

$$x = 9$$

负场数 = 14 - 9 = 5

验证：2 &times; 9 + 1 &times; 5 = 18 + 5 = 23 ✓

**答：** 该队胜9场，负5场。

**【方法提炼】**

比赛积分问题等量关系：胜场得分 + 平场得分 + 负场得分 = 总积分，且胜场 + 平场 + 负场 = 总场数。当只有胜负（无平局）时，负场 = 总场数 - 胜场。广州中考常将此题型拓展为"至少赢几场才能晋级"，此时需结合不等式分析。

### 案例 10：方程+不等式综合方案优化 ⭐⭐⭐⭐

![案例10示意图](./images/equation-case10.png)

<!-- SVG: 采购A、B两种物资，预算约束与数量约束的关系图 -->

**【题目】**

某学校为改善办公条件，计划购买A、B两种型号的电脑。已知A型电脑每台5000元，B型电脑每台3000元。学校计划购买两种电脑共20台，且B型电脑的数量不少于A型电脑数量的2倍。学校可用于购买电脑的经费不超过72000元。

（1）求购买A型电脑数量的取值范围；

（2）有哪几种购买方案？哪种方案最省钱？

**【解析】**

设购买A型电脑 x 台，则购买B型电脑 (20 - x) 台。

**条件1：** B型电脑数量 &ge; A型电脑数量的2倍

$$20 - x \ge; 2x$$

$$20 \ge; 3x$$

$$x \le; \frac{20}{3} \approx 6.67$$

因为 x 为非负整数，所以 x &le; 6。

**条件2：** 总费用不超过72000元

$$5000x + 3000(20 - x) \le; 72000$$

$$5000x + 60000 - 3000x \le; 72000$$

$$2000x \le; 12000$$

$$x \le; 6$$

又 x &ge; 0，综合条件：0 &le; x &le; 6，x 为整数。

购买方案：

| 方案 | A型 x 台 | B型 (20-x) 台 | 总费用（元） |
|------|---------|--------------|-------------|
| 一 | 0 | 20 | 3000&times;20 = 60000 |
| 二 | 1 | 19 | 5000+3000&times;19 = 62000 |
| 三 | 2 | 18 | 10000+54000 = 64000 |
| 四 | 3 | 17 | 15000+51000 = 66000 |
| 五 | 4 | 16 | 20000+48000 = 68000 |
| 六 | 5 | 15 | 25000+45000 = 70000 |
| 七 | 6 | 14 | 30000+42000 = 72000 |

B型越多样费用越低，所以方案一最省钱。

**答：** （1）A型电脑购买数量为 0~6 台（整数）；（2）共7种方案，方案一（不买A型，全买B型20台）最省钱，费用60000元。

**【方法提炼】**

方程+不等式综合题是广州中考压轴的常见形式。解题策略：①用方程表示等量约束，用不等式表示范围约束（"不超过"→&le;，"至少"→&ge;）；②求出未知数的整数取值范围；③枚举所有可行方案并比较。注意隐含条件（数量为非负整数）和题目要求（"省钱"即总费用最小）。

### 案例 11：方程+函数综合压轴 ⭐⭐⭐⭐

![案例11示意图](./images/equation-case11.png)

<!-- SVG: 两种采购方案的费用随数量变化的函数图像，交点为临界点 -->

**【题目】**

某学校需采购一批同型号的课桌椅，现有甲、乙两家家具厂可供选择。甲厂的收费标准为：每套课桌椅120元，另收运费500元；乙厂的收费标准为：每套课桌椅140元，免运费。

（1）分别写出两家工厂收费 y（元）与采购数量 x（套）之间的函数关系式；

（2）采购多少套时，两家工厂收费相同？

（3）学校决定采购不少于30套且不多于50套，选择哪家工厂更合算？

**【解析】**

**（1）** 函数关系式：

甲厂：$y_甲 = 120x + 500$

乙厂：$y_乙 = 140x$

**（2）** 收费相同条件：$y_甲 = y_乙$

$$120x + 500 = 140x$$

$$500 = 20x$$

$$x = 25$$

**答：** 采购25套时，两家工厂收费相同。

**（3）** 当 x = 25 时，两家收费相同。

当 x > 25 时：$140x > 120x + 500$，即 $y_乙 > y_甲$，甲厂便宜。

当 x < 25 时：$y_乙 < y_甲$，乙厂便宜。

因为学校采购 30~50 套，均大于25，所以选甲厂更合算。

验证：x = 30 时，y_甲 = 3600 + 500 = 4100 元，y_乙 = 4200 元，甲厂便宜100元。

x = 50 时，y_甲 = 6000 + 500 = 6500 元，y_乙 = 7000 元，甲厂便宜500元。

**答：** 学校采购30~50套时，选择甲厂更合算。

**【方法提炼】**

方程与函数综合题的核心：①用方程建立函数关系；②用方程求临界点（等值点）；③根据临界点分段讨论，选择最优方案。这类题的实质是"两条直线的交点问题"——交点处方案等价，两侧各有优劣。广州中考常考此题型，务必掌握"求交点&rarr;分区间&rarr;比较大小"的三步法。

## 方法总结

### 一、列方程解应用题的一般步骤

1. **审题**：弄清题意，找出已知量和未知量
2. **设元**：用字母表示未知数（直接设或间接设）
3. **列方程**：找出等量关系，列出方程
4. **解方程**：求出未知数的值
5. **检验**：检验解是否符合实际意义
6. **作答**：写出完整答案

### 二、常见题型与核心等量关系

| 题型 | 核心等量关系 |
|------|------------|
| 行程问题 | 路程 = 速度 &times; 时间；相遇：路程之和 = 总路程；追及：路程相等 |
| 工程问题 | 工作效率 = 1 &divide; 独做时间；各部分工作量之和 = 1 |
| 利润问题 | 售价 = 进价 &times; (1 + 利润率)；售价 = 标价 &times; 折扣 |
| 配套问题 | 按配套比例建立等量关系 |
| 年龄问题 | 经过相同时间，年龄变化量相同 |
| 数字问题 | 多位数 = 位值 &times; 对应数字之和 |
| 分段计费 | 总费用 = 各段费用之和 |
| 积分问题 | 总积分 = 胜场积分 + 平场积分 + 负场积分 |

### 三、易错点提醒

1. **单位统一**：列方程前必须统一单位（如 km/h 与 m/s 不能混用）
2. **设与求**：问什么设什么（直接设法），有时间接设更简便
3. **检验实际意义**：人数、件数必须为非负整数；时间、距离不能为负
4. **打折与利润率**：利润率是相对进价而非售价；注意"几折"即原价的十分之几
5. **分段计费先定位**：先算分界点费用，判断未知量所在区间再列方程
6. **不等式与方程结合**：注意"不超过"用 &le;，"至少"用 &ge;，最后取整数解

### 四、广州中考趋势

广州中考对一元一次方程应用题的考查逐年综合化，从单一方程向"方程+不等式""方程+函数"延伸。命题侧重考查建模能力和实际应用能力，常见命题方向包括：方案优化、分段计费、利润方案设计等。建议在掌握基础题型后，重点训练综合题的分析策略。
