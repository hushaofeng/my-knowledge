# 一次函数 — 广州中考典型试题与解析

> 一次函数是初中函数的起点，广州中考压轴题常涉及一次函数与几何综合，分值 10&ndash;12 分。
>
> 难度标注：⭐ 基础 | ⭐⭐ 中等 | ⭐⭐⭐ 较难 | ⭐⭐⭐⭐ 压轴

## 案例 1：求一次函数解析式 ⭐

![案例1示意图](./images/linear-case01.png)

<!-- SVG: 坐标系中一条直线过点(1,3)和点(3,7)，标注两点坐标 -->

**【题目】** 已知一次函数 y = kx + b 的图象经过点 A(1, 3) 和点 B(3, 7)，求这个一次函数的解析式。

**【解析】**

将 A(1, 3) 代入 y = kx + b：

k + b = 3 &hellip;&hellip; &theta;

将 B(3, 7) 代入 y = kx + b：

3k + b = 7 &hellip;&hellip; &Theta;

&Theta; &minus; &theta;：2k = 4，k = 2。

代入 &theta;：2 + b = 3，b = 1。

&there4; 一次函数解析式为 **y = 2x + 1**。

验证：A(1, 3)：2 &times; 1 + 1 = 3 &check;；B(3, 7)：2 &times; 3 + 1 = 7 &check;。

**答：y = 2x + 1**

**【方法提炼】** 待定系数法求一次函数解析式是中考必考基础题。步骤：(1) 设 y = kx + b；(2) 将已知点坐标代入，得二元一次方程组；(3) 解方程组求 k、b。两个独立条件确定一个一次函数。

## 案例 2：一次函数图象与性质 ⭐

![案例2示意图](./images/linear-case02.png)

<!-- SVG: 坐标系中直线y=-2x+4，标注与x轴交点(2,0)、与y轴交点(0,4)，阴影表示y>0的区域 -->

**【题目】** 已知一次函数 y = &minus;2x + 4，回答下列问题：

（1）函数图象与 x 轴、y 轴的交点坐标分别是什么？

（2）当 x 取何值时，y &gt; 0？

（3）该函数图象经过哪些象限？

**【解析】**

**（1）** 令 y = 0：&minus;2x + 4 = 0，x = 2。与 x 轴交点为 (2, 0)。

令 x = 0：y = &minus;2 &times; 0 + 4 = 4。与 y 轴交点为 (0, 4)。

**（2）** y &gt; 0 即 &minus;2x + 4 &gt; 0，&minus;2x &gt; &minus;4，x &lt; 2。

&there4; 当 x &lt; 2 时，y &gt; 0。

**（3）** k = &minus;2 &lt; 0，b = 4 &gt; 0。

图象从左到右下降（k &lt; 0），过 y 轴正半轴（b &gt; 0），与 x 轴交于正半轴。

&there4; 图象经过第一、二、四象限。

**答：（1）与 x 轴交点 (2, 0)，与 y 轴交点 (0, 4)；（2）x &lt; 2 时 y &gt; 0；（3）经过一、二、四象限**

**【方法提炼】** 一次函数图象性质的核心是 k 和 b 的符号判断：k &gt; 0 图象上升，k &lt; 0 图象下降；b &gt; 0 过 y 轴正半轴，b &lt; 0 过 y 轴负半轴。求交点令 y = 0 或 x = 0 即可。象限判断口诀：&ldquo;k 正 b 正一二三，k 正 b 负一三四，k 负 b 正一二四，k 负 b 负二三四&rdquo;。

## 案例 3：一次函数与不等式 ⭐⭐

![案例3示意图](./images/linear-case03.png)

<!-- SVG: 坐标系中两条直线y=2x-3和y=-x+6，标注交点(3,3)，阴影区域表示2x-3>-x+6即x>3 -->

**【题目】** 已知一次函数 y&sub1; = 2x &minus; 3 和 y&sub2; = &minus;x + 6。

（1）求两函数图象的交点坐标；

（2）当 x 取何值时，y&sub1; &gt; y&sub2;？

**【解析】**

**（1）** 令 y&sub1; = y&sub2;：

2x &minus; 3 = &minus;x + 6

3x = 9

x = 3

y = 2 &times; 3 &minus; 3 = 3

&there4; 交点坐标为 (3, 3)。

**（2）** y&sub1; &gt; y&sub2; 即 2x &minus; 3 &gt; &minus;x + 6：

3x &gt; 9，x &gt; 3。

从图象上看：在交点右侧，y&sub1; 的图象在 y&sub2; 的图象上方。

验证：x = 4 时，y&sub1; = 5，y&sub2; = 2，5 &gt; 2 &check;；x = 0 时，y&sub1; = &minus;3，y&sub2; = 6，&minus;3 &lt; 6 &check;。

**答：（1）交点坐标 (3, 3)；（2）x &gt; 3 时 y&sub1; &gt; y&sub2;**

**【方法提炼】** 利用一次函数图象解不等式是数形结合的典型应用。交点把 x 轴分成两个区间，在每个区间内两函数的大小关系一致。口诀：&ldquo;交点分界，上大下小&rdquo;。也可以直接解不等式，但图象法更直观。

## 案例 4：一次函数实际应用（行程问题）⭐⭐

![案例4示意图](./images/linear-case04.png)

<!-- SVG: 两条线段表示甲乙两人的路程-时间图，甲从原点出发斜率较大，乙从(0,30)出发斜率为负，标注交点 -->

**【题目】** 甲、乙两地相距 30 km，小明从甲地出发骑自行车去乙地，速度为 10 km/h；同时小红从乙地出发步行去甲地，速度为 5 km/h。

（1）分别写出两人与甲地的距离 y（km）与时间 x（h）的函数关系式；

（2）出发后几小时两人相遇？相遇点距甲地多远？

**【解析】**

**（1）** 小明从甲地出发向乙地：

y&sub1; = 10x（0 &le; x &le; 3）

小红从乙地出发向甲地，与甲地距离逐渐减小：

y&sub2; = 30 &minus; 5x（0 &le; x &le; 6）

**（2）** 相遇时两人与甲地距离相等：

10x = 30 &minus; 5x

15x = 30

x = 2

y = 10 &times; 2 = 20

&there4; 出发后 2 小时两人相遇，相遇点距甲地 20 km。

验证：小明走了 2h &times; 10 = 20 km &check;，小红走了 2h &times; 5 = 10 km，30 &minus; 10 = 20 km &check;。合计 20 + 10 = 30 km &check;。

**答：（1）y&sub1; = 10x，y&sub2; = 30 &minus; 5x；（2）2 小时后相遇，距甲地 20 km**

**【方法提炼】** 行程问题的函数建模关键：(1) 弄清起始位置和运动方向；(2) 相向而行，两人距离之和=总路程；(3) 同向而行，考虑追及问题。画出路程&ndash;时间图是解题利器。注意自变量取值范围。

## 案例 5：一次函数实际应用（利润问题）⭐⭐

![案例5示意图](./images/linear-case05.png)

<!-- SVG: 坐标系中两条直线，一条表示收入y=10x，一条表示成本y=500+6x，标注交点(125,1250)为盈亏平衡点 -->

**【题目】** 某商店购进一批商品，固定成本为 500 元，每件商品的进价为 6 元，售价为 10 元。设销售 x 件商品。

（1）分别写出总成本 C（元）和总收入 R（元）关于 x 的函数关系式；

（2）至少销售多少件才能不亏本？

（3）若要获利不低于 1000 元，至少销售多少件？

**【解析】**

**（1）** 总成本 = 固定成本 + 变动成本：

C = 500 + 6x（x &ge; 0）

总收入 = 售价 &times; 销量：

R = 10x（x &ge; 0）

**（2）** 不亏本即 R &ge; C：

10x &ge; 500 + 6x

4x &ge; 500

x &ge; 125

&there4; 至少销售 125 件才能不亏本。

**（3）** 获利不低于 1000 元：

R &minus; C &ge; 1000

10x &minus; (500 + 6x) &ge; 1000

4x &minus; 500 &ge; 1000

4x &ge; 1500

x &ge; 375

&there4; 至少销售 375 件才能获利不低于 1000 元。

**答：（1）C = 500 + 6x，R = 10x；（2）至少 125 件；（3）至少 375 件**

**【方法提炼】** 利润问题的函数建模：利润 = 收入 &minus; 成本。固定成本是截距 b，单位变动成本影响斜率。盈亏平衡点是收入线与成本线的交点。不等式与一次函数结合是中考常考应用题类型。

## 案例 6：一次函数与三角形面积 ⭐⭐⭐

![案例6示意图](./images/linear-case06.png)

<!-- SVG: 坐标系中直线y=-3/4x+3，与x轴交点A(4,0)、与y轴交点B(0,3)，原点O，标注三角形OAB -->

**【题目】** 已知一次函数 y = &minus;(3/4)x + 3 的图象与 x 轴交于点 A，与 y 轴交于点 B。

（1）求点 A、B 的坐标；

（2）求 &triangle;OAB 的面积；

（3）在直线 AB 上是否存在点 P，使 &triangle;OAP 的面积为 3？若存在，求出点 P 的坐标。

**【解析】**

**（1）** 令 y = 0：&minus;(3/4)x + 3 = 0，x = 4。A(4, 0)。

令 x = 0：y = 3。B(0, 3)。

**（2）** S&triangle;OAB = 1/2 &times; OA &times; OB = 1/2 &times; 4 &times; 3 = 6。

**（3）** 设 P 在直线 AB 上，P(a, &minus;3a/4 + 3)。

&triangle;OAP 以 OA 为底，高为 |P 的纵坐标|：

S&triangle;OAP = 1/2 &times; OA &times; |y&subP;| = 1/2 &times; 4 &times; |&minus;3a/4 + 3| = 2|&minus;3a/4 + 3| = 3

|&minus;3a/4 + 3| = 3/2

情况一：&minus;3a/4 + 3 = 3/2，3a/4 = 3/2，a = 2。

P(2, &minus;3/4 &times; 2 + 3) = (2, 3/2)。

情况二：&minus;3a/4 + 3 = &minus;3/2，3a/4 = 9/2，a = 6。

P(6, &minus;3/4 &times; 6 + 3) = (6, &minus;3/2)。

验证：P(2, 3/2)，S = 1/2 &times; 4 &times; 3/2 = 3 &check;；P(6, &minus;3/2)，S = 1/2 &times; 4 &times; 3/2 = 3 &check;。

**答：（1）A(4, 0)，B(0, 3)；（2）S&triangle;OAB = 6；（3）P(2, 3/2) 或 P(6, &minus;3/2)**

**【方法提炼】** 坐标系中求三角形面积，选择坐标轴上的边为底是关键技巧，此时高就是另一个顶点纵（横）坐标的绝对值。遇到绝对值要分类讨论，不要遗漏解。这是广州中考填空/解答题高频考点。

## 案例 7：分段函数应用 ⭐⭐⭐

![案例7示意图](./images/linear-case07.png)

<!-- SVG: 分段函数图象，0-3km为起步价10元（水平线），3km以上每公里2元（上升直线），标注分段点(3,10) -->

**【题目】** 某城市出租车收费标准如下：起步价 10 元（3 km 以内，含 3 km），超过 3 km 的部分每公里加收 2 元（不足 1 km 按 1 km 计）。

（1）写出车费 y（元）与行驶路程 x（km）的函数关系式（x 取整数）；

（2）小明乘车行驶了 8 km，应付车费多少元？

（3）若小明付了 24 元，他最多行驶了多少公里？

**【解析】**

**（1）** 当 0 &lt; x &le; 3 时，y = 10（起步价）；

当 x &gt; 3 时，y = 10 + 2(x &minus; 3) = 2x + 4（x 取整数）。

即：

y = { 10,        0 &lt; x &le; 3
    { 2x + 4,    x &gt; 3（x 为正整数）

**（2）** x = 8 &gt; 3，y = 2 &times; 8 + 4 = 20 元。

**答：应付车费 20 元。**

**（3）** y = 24，因为 24 &gt; 10，使用第二段：

2x + 4 = 24，2x = 20，x = 10。

验证：y = 2 &times; 10 + 4 = 24 &check;。

**答：最多行驶了 10 km。**

**【方法提炼】** 分段函数是广州中考应用题常见题型。关键：(1) 准确确定分段点（此处为 x = 3）；(2) 每段分别建立函数关系式；(3) 求解时先判断属于哪一段再代入。注意自变量取值范围的标注不能遗漏。

## 案例 8：方案设计问题 ⭐⭐⭐

![案例8示意图](./images/linear-case08.png)

<!-- SVG: 两条直线y1=20x+500和y2=30x+200，标注交点(30,1100)，x<30时方案二便宜，x>30时方案一便宜 -->

**【题目】** 某学校需购买一批教具，有甲、乙两种方案：

方案甲：每套教具 20 元，另付运费 500 元；

方案乙：每套教具 30 元，免运费。

设购买 x 套教具（x 为正整数）。

（1）分别写出两种方案的费用 y（元）关于 x 的函数关系式；

（2）购买多少套时，两种方案费用相同？

（3）请你根据购买数量帮助学校选择最省钱的方案。

**【解析】**

**（1）** 方案甲：y&sub1; = 20x + 500

方案乙：y&sub2; = 30x

**（2）** 令 y&sub1; = y&sub2;：

20x + 500 = 30x

10x = 500

x = 50

**答：购买 50 套时费用相同。**

**（3）** 当 x = 50 时，y&sub1; = y&sub2; = 1500 元，两种方案一样；

当 x &lt; 50 时，y&sub1; &minus; y&sub2; = 500 &minus; 10x &gt; 0，y&sub1; &gt; y&sub2;，选方案乙；

当 x &gt; 50 时，y&sub1; &minus; y&sub2; = 500 &minus; 10x &lt; 0，y&sub1; &lt; y&sub2;，选方案甲。

也可以从斜率角度理解：方案甲斜率小（20 &lt; 30），随着 x 增大，方案甲增长更慢，但截距大（500 &gt; 0），所以存在&ldquo;交叉点&rdquo;。

**答：购买不足 50 套选乙，恰好 50 套均可，超过 50 套选甲**

**【方法提炼】** 方案设计题的核心是比较两个一次函数的大小关系。步骤：(1) 分别建立函数；(2) 求交点（费用相同点）；(3) 交点两侧分别比较。交点就是&ldquo;临界点&rdquo;，是决策的分界线。此类题是广州中考大题的常客。

## 案例 9：一次函数与动点面积最值 ⭐⭐⭐⭐

![案例9示意图](./images/linear-case09.png)

<!-- SVG: 坐标系中直线y=-x+4与x轴交A(4,0)、与y轴交B(0,4)，点P在x轴上从O向A运动，标注三角形OBP的面积随P位置变化 -->

**【题目】** 如图，直线 y = &minus;x + 4 与 x 轴交于点 A，与 y 轴交于点 B。点 P 在 x 轴的正半轴上运动，设 P(t, 0)（0 &lt; t &lt; 4）。

（1）求点 A、B 的坐标；

（2）当 P 在线段 OA 上运动时，设 &triangle;OBP 的面积为 S，求 S 关于 t 的函数关系式，并求 S 的最大值；

（3）过 P 作 PQ &perp; x 轴交直线 AB 于 Q，当 t 为何值时，&triangle;OPQ 的面积最大？最大面积是多少？

**【解析】**

**（1）** 令 y = 0：&minus;x + 4 = 0，x = 4。A(4, 0)。

令 x = 0：y = 4。B(0, 4)。

**（2）** &triangle;OBP 以 OP = t 为底，OB = 4 为高：

S = 1/2 &times; t &times; 4 = 2t（0 &lt; t &lt; 4）

S 随 t 增大而增大，当 t &rarr; 4 时 S &rarr; 8。

但 t &lt; 4（P 不与 A 重合），所以 S 没有最大值（开区间）。

若题目允许 t = 4（P 与 A 重合），则 S&submax; = 2 &times; 4 = 8。

**答：S = 2t，S&submax; = 8（t = 4 时）**

**（3）** P(t, 0)，Q 在直线 AB 上，Q(t, &minus;t + 4)（0 &lt; t &lt; 4）。

&triangle;OPQ 以 OP = t 为底，PQ = |&minus;t + 4| = 4 &minus; t 为高：

S&subOPQ; = 1/2 &times; t &times; (4 &minus; t) = &minus;t&sup2;/2 + 2t = &minus;1/2(t &minus; 2)&sup2; + 2

当 t = 2 时，S&subOPQ; 取最大值 2。

此时 P(2, 0)，Q(2, 2)，验证：S = 1/2 &times; 2 &times; 2 = 2 &check;。

**答：t = 2 时 &triangle;OPQ 面积最大，最大面积为 2**

**【方法提炼】** 动点面积最值问题是广州中考压轴题的典型模型。核心方法：(1) 设动点参数 t，用 t 表示面积 S；(2) S 的表达式通常是二次函数 S = &minus;at&sup2; + bt（a &gt; 0）；(3) 配方求最值 S = &minus;a(t &minus; b/2a)&sup2; + b&sup2;/4a；(4) 注意检验最值点是否在取值范围内。底选坐标轴上的边，高用函数纵坐标表示，是简化计算的关键。

## 案例 10：一次函数与几何综合压轴 ⭐⭐⭐⭐

![案例10示意图](./images/linear-case10.png)

<!-- SVG: 坐标系中直线y=-3/4x+6与x轴交A(8,0)、与y轴交B(0,6)，点C(0,2)，点P在直线AB上运动，标注三角形APC -->

**【题目】** （广州中考风格压轴）如图，直线 y = &minus;(3/4)x + 6 与 x 轴交于点 A，与 y 轴交于点 B，点 C(0, 2)。

（1）求点 A 的坐标及直线 AB 与坐标轴围成的三角形面积；

（2）点 P 在直线 AB 上，且 &triangle;APC 是以 AC 为底的等腰三角形，求点 P 的坐标；

（3）点 P 在直线 AB 上运动，连接 PC，当 &triangle;APC 面积最大时，求点 P 的坐标及最大面积。

**【解析】**

**（1）** 令 y = 0：&minus;(3/4)x + 6 = 0，x = 8。A(8, 0)。

令 x = 0：y = 6。B(0, 6)。

S&triangle;OAB; = 1/2 &times; OA &times; OB = 1/2 &times; 8 &times; 6 = 24。

**答：A(8, 0)，S&triangle;OAB = 24**

**（2）** AC = &radic;((8&minus;0)&sup2; + (0&minus;2)&sup2;) = &radic;(64+4) = &radic;68 = 2&radic;17。

&triangle;APC 以 AC 为底等腰，即 AP = CP。

设 P(m, &minus;3m/4 + 6)（0 &le; m &le; 8）。

AP&sup2; = (m&minus;8)&sup2; + (&minus;3m/4+6)&sup2;

CP&sup2; = m&sup2; + (&minus;3m/4+6&minus;2)&sup2; = m&sup2; + (&minus;3m/4+4)&sup2;

令 AP&sup2; = CP&sup2;：

(m&minus;8)&sup2; + (&minus;3m/4+6)&sup2; = m&sup2; + (&minus;3m/4+4)&sup2;

展开左边：(m&sup2; &minus; 16m + 64) + (9m&sup2;/16 &minus; 9m + 36)

= m&sup2; &minus; 16m + 64 + 9m&sup2;/16 &minus; 9m + 36

= 25m&sup2;/16 &minus; 25m + 100

展开右边：m&sup2; + (9m&sup2;/16 &minus; 6m + 16)

= m&sup2; + 9m&sup2;/16 &minus; 6m + 16

= 25m&sup2;/16 &minus; 6m + 16

令左右相等：

25m&sup2;/16 &minus; 25m + 100 = 25m&sup2;/16 &minus; 6m + 16

&minus;25m + 100 = &minus;6m + 16

&minus;19m = &minus;84

m = 84/19

y = &minus;3/4 &times; 84/19 + 6 = &minus;63/19 + 114/19 = 51/19

验证：0 &le; 84/19 &asymp; 4.4 &le; 8 &check;

**答：P(84/19, 51/19)**

**（3）** &triangle;APC 以 AC 为底，AC = 2&radic;17。

P 到 AC 的距离最大时，面积最大。

AC 所在直线：过 A(8, 0) 和 C(0, 2)，斜率 = (2&minus;0)/(0&minus;8) = &minus;1/4。

直线 AC：y &minus; 0 = (&minus;1/4)(x &minus; 8)，即 y = &minus;x/4 + 2，即 x/4 + y &minus; 2 = 0。

设 P(m, &minus;3m/4 + 6)，P 到直线 AC 的距离：

d = |m/4 + (&minus;3m/4+6) &minus; 2| / &radic;(1/16 + 1) = |m/4 &minus; 3m/4 + 4| / &radic;(17/16) = |&minus;m/2 + 4| &times; 4/&radic;17

= |4 &minus; m/2| &times; 4/&radic;17

当 0 &le; m &le; 8 时，4 &minus; m/2 &ge; 0，所以：

d = (4 &minus; m/2) &times; 4/&radic;17 = (16 &minus; 2m)/&radic;17

d 在 m = 0 时取最大值 16/&radic;17。

但需验证 P(0, 6) = B 点是否在直线 AB 上 &check;。

S&triangle;APC; = 1/2 &times; AC &times; d = 1/2 &times; 2&radic;17 &times; (16 &minus; 2m)/&radic;17 = 16 &minus; 2m

当 m = 0 时，S&submax; = 16。

即 P 与 B 重合时面积最大。

验证：S&triangle;ABC; = 1/2 &times; AC &times; B到AC距离。AC = 2&radic;17，B(0,6) 到 AC：d = |0/4 + 6 &minus; 2|/&radic;(17/16) = 4 &times; 4/&radic;17 = 16/&radic;17。S = 1/2 &times; 2&radic;17 &times; 16/&radic;17 = 16 &check;。

**答：P(0, 6)（即 B 点）时面积最大，最大面积为 16**

**【方法提炼】** 一次函数与几何综合压轴题是广州中考的&ldquo;终极BOSS&rdquo;。核心方法体系：(1) 坐标法是基础工具，将几何问题代数化；(2) 等腰三角形问题用距离公式列方程；(3) 面积最值转化为点到直线距离的最值；(4) 点到直线距离公式 d = |Ax&sub0; + By&sub0; + C|/&radic;(A&sup2; + B&sup2;) 是解题利器；(5) 注意检验最值点是否满足条件（在指定线段上）。广州中考压轴题往往2&ndash;3问，第(1)问送分，第(2)问中等，第(3)问拉开差距。

## 方法总结

| 题型 | 核心方法 | 易错点 | 难度 |
|------|---------|--------|------|
| 求解析式 | 待定系数法，代入两点列方程组 | 计算错误，遗漏验证 | ⭐ |
| 图象与性质 | k 决定增减性，b 决定截距，k&middot;b 判定象限 | 混淆 k、b 符号与象限的关系 | ⭐ |
| 函数与不等式 | 交点分界，图象上大下小 | 不等号方向弄反 | ⭐⭐ |
| 行程应用 | 路程 = 速度 &times; 时间，注意方向 | 自变量取值范围遗漏 | ⭐⭐ |
| 利润应用 | 利润 = 收入 &minus; 成本，不等式求解 | 盈亏平衡点理解错误 | ⭐⭐ |
| 三角形面积 | 选坐标轴上的边为底，高 = 纵/横坐标绝对值 | 绝对值分类讨论遗漏 | ⭐⭐⭐ |
| 分段函数 | 准确分段，每段独立建模 | 分段点归属不清，范围标注遗漏 | ⭐⭐⭐ |
| 方案设计 | 比较两个一次函数大小，求交点临界值 | 交点两侧方案选择弄反 | ⭐⭐⭐ |
| 动点面积最值 | 面积 S 用参数 t 表示，配方求二次函数最值 | 未检验最值点是否在取值范围内 | ⭐⭐⭐⭐ |
| 几何综合压轴 | 坐标法 + 距离公式 + 点到直线距离 + 分类讨论 | 计算量大需耐心，等腰三角形漏解 | ⭐⭐⭐⭐ |

### 解题通法口诀

> **一次函数解题五步法：**
>
> 1. **设** &mdash;&mdash; 设解析式 y = kx + b（或设动点参数 t）
> 2. **代** &mdash;&mdash; 代入已知条件，列方程（组）
> 3. **解** &mdash;&mdash; 解方程，求 k、b 或参数值
> 4. **验** &mdash;&mdash; 验证解的合理性（范围、意义）
> 5. **答** &mdash;&mdash; 规范作答，注意单位

> **面积问题三原则：**
>
> - 底选坐标轴上的边，高用坐标表示
> - 遇绝对值必分类讨论
> - 最值问题配方转化

> **广州中考一次函数考情：**
>
> - 基础题（选择/填空）：求解析式、判断象限，2&ndash;3 分
> - 中档题（解答题前段）：实际应用、方案设计，6&ndash;8 分
> - 压轴题（解答题最后）：一次函数 + 动点 + 几何综合，10&ndash;12 分
> - 高频组合：一次函数 &times; 等腰三角形 &times; 面积最值 &times; 分类讨论
