## 九年级上册

### 第二十一章 一元二次方程 🔴

> 🔴 **中考高频**：一元二次方程是代数核心，必考知识点。

#### 21.1 一元二次方程

**知识点**：标准形式 ax² + bx + c = 0（a≠0）。

#### 21.2 解一元二次方程 🔴

| 方法 | 适用场景 | 案例 |
|------|---------|------|
| 直接开平方法 | (x+a)² = b 形式 | (x-1)² = 9 → x = 4 或 -2 |
| 因式分解法 | 方程易分解 | x²-5x+6=0 → (x-2)(x-3)=0 → x=2 或 3 |
| 配方法 | 通用方法 | x²+4x-1=0 → (x+2)²=5 → x=-2±√5 |
| 公式法 | 通用方法 | x = (-b±√(b²-4ac))/(2a) |

**案例**：
- 2x² - 5x - 3 = 0
  - Δ = 25 + 24 = 49 > 0
  - x = (5±7)/4 → x₁ = **3**，x₂ = **-1/2**

#### 21.3 根的判别式 🔴

**知识点**：Δ = b²-4ac

| Δ 值 | 根的情况 |
|------|---------|
| Δ > 0 | 两个不相等的实数根 |
| Δ = 0 | 两个相等的实数根 |
| Δ < 0 | 无实数根 |

**案例**：
- 方程 x² - 2x + k = 0 有两个不相等的实数根，求 k 的范围
  - Δ = 4 - 4k > 0 → k < **1**

> ⚠️ **中考重点**：根的判别式常与二次函数结合出题。

#### 21.4 实际问题与一元二次方程 🔴

**案例**：
- 某商品原价 100 元，连续两次降价的百分率相同，降后价 81 元，求每次降价的百分率
  - 100(1-x)² = 81 → (1-x)² = 0.81 → 1-x = 0.9 → x = **10%**

### 第二十二章 二次函数 🔴🔴🔴

> 🔴🔴🔴 **中考最核心**：二次函数是广州中考压轴题的首选载体，每年必考，分值 10-14 分。

#### 22.1 二次函数的图像和性质 🔴🔴

**知识点**：y = ax² + bx + c（a≠0）

| 参数 | 作用 |
|------|------|
| a | a>0 开口向上，a<0 开口向下；|a| 越大开口越小 |
| 对称轴 | x = -b/(2a) |
| 顶点 | (-b/(2a), (4ac-b²)/(4a)) |
| c | 与 y 轴交点纵坐标 |

**顶点式**：y = a(x-h)² + k，顶点 (h, k)

**案例**：
- y = x² - 4x + 3
  - 配方：y = (x-2)² - 1
  - 开口**向上**，对称轴 x = **2**，顶点 **(2, -1)**
  - 与 x 轴交点：令 y=0 → (x-2)² = 1 → x = 1 或 3 → **(1,0) 和 (3,0)**
  - 与 y 轴交点：**c = 3 → (0, 3)**

#### 22.2 用函数观点看一元二次方程 🔴

**知识点**：二次函数图像与 x 轴的交点 ↔ 一元二次方程的实数根。

| 交点情况 | Δ 值 |
|---------|-------|
| 两个交点 | Δ > 0 |
| 一个交点（相切） | Δ = 0 |
| 无交点 | Δ < 0 |

#### 22.3 实际问题与二次函数 🔴🔴

> 🔴🔴 **压轴题标配**：利润最大化、面积最大化、抛物线形实际问题。

**案例 1（利润最大化）**：
- 某商品进价 30 元，售价 50 元时每天可卖 100 件，售价每提高 1 元少卖 2 件。求最大利润。
  - 设提价 x 元，利润 y = (50+x-30)(100-2x) = (20+x)(100-2x)
  - = -2x² + 60x + 2000
  - 对称轴 x = 60/4 = 15
  - 最大利润 = (20+15)×(100-30) = 35×70 = **2450 元**，售价 **65 元**

**案例 2（面积最大化）**：
- 用 40 米围栏靠墙围一个矩形场地，求最大面积。
  - 设宽 x 米，长 = 40-2x，面积 S = x(40-2x) = -2x² + 40x
  - 对称轴 x = 10，最大面积 = 10×20 = **200 m²**

**案例 3（抛物线形问题）** 🔴🔴：
- 某涵洞截面为抛物线形，水面宽 4 米时水面距顶 2 米，求水面宽 6 米时水面距顶多少？
  - 建系：顶点(0,0)，水面过(-2,-2)和(2,-2)
  - y = ax²，代入 -2 = 4a → a = -1/2 → y = -x²/2
  - 水面宽 6 米 → x=3，y = -9/2 = -4.5 → 距顶 **4.5 米**

### 第二十四章 圆 🔴🔴

> 🔴🔴 **中考核心**：圆是广州中考几何部分的重点，每年出 1-2 道大题。

#### 24.1 圆的有关性质 🔴

**知识点**：垂径定理；弧、弦、圆心角的关系；圆周角定理。

**垂径定理**：垂直于弦的直径平分弦及所对的两条弧。

**圆周角定理**：同弧上的圆周角是圆心角的一半。

**案例**：
- ⊙O 中弦 AB = 8，圆心到弦的距离 d = 3，则半径 r = √(16+9) = **5**
- 圆心角 ∠AOB = 80°，则圆周角 ∠ACB = **40°**（同弧）
- 半圆（直径）所对的圆周角 = **90°**

#### 24.2 点和圆的位置关系 🟡

**知识点**：d < r → 在圆内；d = r → 在圆上；d > r → 在圆外。

**三角形的外心**：三边垂直平分线交点，到三个顶点距离相等。

#### 24.3 直线和圆的位置关系 🔴

| 位置关系 | d 与 r 的关系 | 交点数 |
|---------|-------------|--------|
| 相离 | d > r | 0 |
| 相切 | d = r | 1 |
| 相交 | d < r | 2 |

**切线定理** 🔴：切线垂直于过切点的半径。

**切线长定理** 🟡：从圆外一点引圆的两条切线，切线长相等。

**案例**：
- ⊙O 半径为 3，圆心 O 到直线 l 的距离为 5，则直线与圆 **相离**
- PA、PB 切⊙O 于 A、B，PA = 5，则 PB = **5**

#### 24.4 弧长和扇形面积 🟡

**公式**：
- 弧长 l = nπr/180
- 扇形面积 S = nπr²/360 = lr/2

**案例**：
- 半径 6，圆心角 120° 的弧长 = 120×π×6/180 = **4π**
- 对应扇形面积 = 120×π×36/360 = **12π**

#### 24.5 圆锥 🟢

**知识点**：侧面积 = πrl（r 底面半径，l 母线长）。

## 九年级下册

### 第二十五章 概率初步 🟡

#### 25.1 随机事件与概率

**知识点**：必然事件（P=1）、不可能事件（P=0）、随机事件（0<P<1）。

#### 25.2 用列举法求概率 🟡

**知识点**：列表法、树形图法。

**案例**：
- 掷两枚硬币，一正一反的概率？
  - (正,正), (正,反), (反,正), (反,反) → P = **2/4 = 1/2**
- 口袋中有 3 红 2 白共 5 个球，摸出 2 个球恰好一红一白的概率？
  - C(3,1)×C(2,1) / C(5,2) = 6/10 = **3/5**

#### 25.3 用频率估计概率 🟢

> 💡 **中考提示**：概率题一般出一道选择/填空（3分），难度不大。

### 第二十六章 反比例函数 🔴

> 🔴 **中考高频**：反比例函数常与一次函数结合出综合题。

#### 26.1 反比例函数 🟡

**知识点**：y = k/x（k≠0），图像为双曲线。

| k 值 | 图像位置 | 增减性 |
|------|---------|--------|
| k > 0 | 一、三象限 | 每个象限内 y 随 x 增大而减小 |
| k < 0 | 二、四象限 | 每个象限内 y 随 x 增大而增大 |

**案例**：
- y = 6/x 过点 (2, ? ) → y = 3，过点 (3, ? ) → y = 2
- |k| 的几何意义：双曲线上任一点与坐标轴围成矩形面积 = **|k|**

#### 26.2 实际问题与反比例函数 🟡

**案例**：
- 电压 U = 220V 不变，电流 I 与电阻 R 成反比：I = 220/R
- R = 44Ω 时 I = **5A**；I = 2A 时 R = **110Ω**

#### 27.2 相似三角形 🔴🔴

**判定方法**：

| 方法 | 条件 |
|------|------|
| AA | 两角对应相等 |
| SAS | 两边成比例且夹角相等 |
| SSS | 三边成比例 |

**性质**：
- 对应边成比例、对应角相等
- 面积比 = 相似比的平方
- 周长比 = 相似比

**案例**：
- △ABC ∼ △DEF，相似比 2:3，S△ABC = 12，则 S△DEF = 12 × (3/2)² = **27**

#### 27.3 位似 🟢

**知识点**：位似图形是特殊的相似图形，对应点连线交于位似中心。

### 第二十八章 锐角三角函数 🔴

> 🔴 **中考高频**：解直角三角形是必考内容，常出现在应用题中。

#### 28.1 锐角三角函数 🔴

| 函数 | 定义 | 特殊角 30° | 特殊角 45° | 特殊角 60° |
|------|------|-----------|-----------|-----------|
| sin A | 对边/斜边 | 1/2 | √2/2 | √3/2 |
| cos A | 邻边/斜边 | √3/2 | √2/2 | 1/2 |
| tan A | 对边/邻边 | √3/3 | 1 | √3 |

> ⚠️ **必须记忆**：30°、45°、60° 的三角函数值是中考必考。

**案例**：
- 在 Rt△ABC 中，∠C=90°，AB=10，sinA=3/5，求 BC
  - BC/AB = 3/5 → BC = **6**，AC = 8

#### 28.2 解直角三角形 🔴

**知识点**：已知两元素（至少一个边）可解直角三角形。

**案例**：
- 某人在山脚 A 处测得山顶 B 仰角 30°，沿 30° 坡面走 200 米到 C，再测山顶仰角 60°，求山高。
  - 设山高 h，作辅助线后：
  - 在△BCD 中，∠BCD = 30°，∠CBD = 30°
  - BD = BC = 200（等腰三角形判定）
  - h = BD × sin60° = 200 × √3/2 = **100√3 ≈ 173.2 米**

#### 28.3 实际问题与锐角三角函数 🔴

> 常见应用场景：仰角/俯角问题、坡度/坡角问题、方向角问题、航海问题。

## 广州中考数学总览

### 全三年考点分值分布（满分 120 分）

| 板块 | 分值 | 占比 |
|------|------|------|
| 数与代数（方程、函数、不等式） | 约 42-48 分 | 35-40% |
| 图形与几何（三角形、四边形、圆、相似） | 约 42-48 分 | 35-40% |
| 统计与概率 | 约 12-15 分 | 10-12% |
| 综合与实践（压轴题） | 约 15-20 分 | 12-17% |

### 广州中考压轴题规律（2022-2025）

| 年份 | 第 24 题 | 第 25 题 |
|------|---------|---------|
| 2022 | 二次函数+面积最值 | 几何旋转+相似综合 |
| 2023 | 二次函数+线段最值 | 圆+相似+最值 |
| 2024 | 二次函数+动点问题 | 几何变换+相似综合 |
| 2025 | 二次函数+存在性问题 | 圆+三角函数综合 |

**压轴题规律总结**：
1. **第 24 题几乎必考二次函数** — 常见类型：面积最值、线段最值、动点存在性
2. **第 25 题考几何综合** — 相似三角形 + 圆/旋转/最值
3. **动点问题年年有** — 分类讨论思想是核心
4. **数形结合贯穿始终** — 建系、解析法越来越受重视

### 备考策略

1. **二轮复习法**：一轮按章节过（2月-4月），二轮按专题突破（4月-5月）
2. **压轴题训练**：每天一道二次函数综合 + 一道几何综合
3. **计算零失误**：有理数运算、解方程、因式分解必须快且准
4. **规范书写**：几何证明格式、辅助线作法描述，广州阅卷非常看重规范性
5. **真题为王**：近 5 年广州中考真题至少做 3 遍，理解出题思路比刷题量更重要
