## 笔记模板

每篇读书笔记建议包含：

| 模块 | 说明 |
|------|------|
| 基本信息 | 书名、作者、出版时间、阅读时间 |
| 一句话总结 | 用一句话概括全书核心 |
| 核心观点 | 3-5 个关键 take-away |
| 精彩摘录 | 原文金句 + 页码 |
| 我的思考 | 联系自身经历的反思 |
| 行动清单 | 读完要做什么改变 |
