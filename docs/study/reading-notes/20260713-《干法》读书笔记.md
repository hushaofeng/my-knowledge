# 《干法》读书笔记

> 作者：[日] 稻盛和夫 | 译者：曹岫云 | 出版：华文出版社，2010年5月
> 阅读时间：2026-07-13

---

## 📌 一、核心观点

### 本书讲了什么

本书是"日本经营之圣"稻盛和夫关于"工作"的哲学思考，是《活法》的姊妹篇。核心主题：**工作的真正意义是什么？如何通过工作磨炼灵魂、提升心志，获得幸福的人生？**

稻盛和夫27岁创业，40年间创建京瓷和KDDI两家"世界500强"企业，78岁出任破产重建的日航CEO并成功扭亏为盈。他将自己一生的实践经验提炼为"劳动观"和"工作观"。

### 核心论点

**1. 工作的本质意义——磨炼灵魂、提升心志 🔴🔴🔴**

> "工作是人生最尊贵、最重要、最有价值的行为。"

大多数人认为工作只是为了吃饭获取报酬，是"必要之恶"。稻盛和夫认为，工作的真正目的是**提升心志、磨炼人格**。日复一日勤奋工作，可以起到锻炼心志、提升人性的了不起作用。

**2. 改变心态——让自己喜欢上工作 🔴🔴🔴**

> "要想度过一个充实的人生，只有两种选择：一种是'从事自己喜欢的工作'，另一种是'让自己喜欢上工作'。"

能够碰上自己喜欢的工作的几率不足千分之一。与其寻找自己喜欢的工作，不如先喜欢上已有的工作。从"厌恶工作"转变为"迷恋工作"，就能进入良性循环：喜欢→投入→成果→肯定→更喜欢。

**3. 付出"不亚于任何人的努力" 🔴🔴🔴**

> "平平常常的努力，不管是企业还是个人，都不可能获得理想的成果。"

这不是普通的努力，而是**无限度的、永无止境的努力**。用百米赛的速度跑马拉松——这是京瓷从后进者变成领先者的秘诀。自然界的一切生物都在拼命努力求生，人类追求安乐反而是违背自然机理的。

**4. 持续的力量——将"平凡"变为"非凡" 🔴🔴**

> "伟大的事业乃是朴实、枯燥工作的积累。"

金字塔是由无数无名氏一块一块砌上去的。专心致志于一行一业，不腻烦、不焦躁，埋头苦干，就能让"平凡的人"变成"非凡的人"。那些看起来不起眼、只是踏踏实实工作的人，往往能成为优秀的领导者。

**5. 追求"完美主义"——不是"最佳"，而是"完美" 🔴🔴**

> "要完成一个产品，99%的努力是不够的。一点差错、一点疏忽、一点马虎都不能允许。"

"最佳"是相对的，是同别人比较。"完美"是绝对的，是自身具备可靠的价值。成败取决于"最后1%的努力"，懈怠最后1%，前面的努力全都前功尽弃。

**6. 创造性地工作——走别人没走过的路 🔴🔴**

> "铺装平整的大道是大家都想走的路。若只知步别人的后尘，则绝不能开拓新的事业。"

每天都要钻研创新，即使一天的进步微不足道，但若经过十年的积累，就能孕育巨大的变化。外行的长处是可以"自由发想"，不被成见束缚。

**7. 人生方程式——思维方式×热情×能力 🔴🔴🔴**

> 人生·工作的结果 = 思维方式 × 热情 × 能力

这是稻盛和夫最核心的公式：
- **能力**：天赋资质，0-100分（先天）
- **热情**：努力程度，0-100分（后天可控）
- **思维方式**：态度和价值观，-100到100分（最关键）

乘法运算意味着：如果思维方式是负值，能力越强、热情越高，结果反而越糟糕。正确的思维方式包括：积极向上、诚实正直、谦虚谨慎、勤奋努力、不自私、有感恩心等。

### 关键框架

| 框架 | 核心内容 |
|------|---------|
| **工作意义** | 磨炼灵魂、提升心志，而非仅仅谋生 |
| **心态转变** | 从"厌恶"到"迷恋"，创造良性循环 |
| **努力程度** | 不亚于任何人，用百米速度跑马拉松 |
| **持续积累** | 每天进步一点，十年积累成巨大变化 |
| **完美主义** | 不是"最佳"（相对），而是"完美"（绝对） |
| **创新精神** | 走别人没走过的路，每天钻研创新 |
| **人生方程式** | 思维方式×热情×能力，思维方式最关键 |

## 💡 二、启发思考

### 对一般人的启发

1. **重新定义"工作"**——工作不是为了谋生，而是人生修行的道场
2. **"天职"是自己创造的**——不是碰上喜欢的才努力，而是努力了才会喜欢
3. **努力可以弥补天赋不足**——60分能力×90分努力 > 90分能力×30分努力
4. **持续比爆发更重要**——金字塔是一块石头一块石头堆起来的
5. **思维方式决定人生走向**——负面的思维方式会让一切努力走向反面

### 从心理学和实践角度的启发 🎯

#### 🌱 人生发展维度

1. **"必要之恶"观念的心理根源**：稻盛和夫指出，西方基督教文化将劳动视为对原罪的惩罚，这种观念深刻影响了现代人的工作观。而日本传统文化认为劳动能带来喜悦和自豪——这种文化差异提醒我们：**观念决定体验**，改变对工作的认知，就能改变工作的感受

2. **"自燃型"人格的心理特质**：稻盛和夫将人分为"可燃型"、"不燃型"和"自燃型"。心理学研究表明，"自燃型"对应的是**内在动机驱动**——不是为了外部奖励而工作，而是因为工作本身有意义。17年技术开发和管理，我的工作动力是否已经从"内在动机"转向了"外部压力"？

3. **苦难的意义重构**：稻盛和夫年轻时连续遭遇挫折，但他认为"乍看的不幸，实际上是幸事"。心理学中的**创伤后成长**（Post-traumatic Growth）理论证实：适度苦难可以促进人格成熟。我目前的"精力下降"和"转型困境"是否也可以被重新定义为成长的契机？

#### 🏠 家庭关系维度

1. **工作态度的家庭传承**：稻盛和夫说"工作造就人格"。作为两个孩子的父亲，我对待工作的态度会直接影响孩子对"努力"的认知。如果我在工作中表现出抱怨和逃避，孩子学到的是"工作很苦"；如果表现出投入和热爱，孩子学到的是"努力有意义"

2. **家庭决策的"思维方式"**：稻盛和夫强调思维方式是最关键的乘数。在家庭重大决策中（如转型方向、教育投资、财务规划），负面的思维方式（抱怨、焦虑、自我否定）会让决策质量大幅下降

3. **感谢之心对家庭氛围的影响**：稻盛和夫反复强调"感谢"的重要性。心理学研究表明，感恩练习可以显著提升家庭幸福感。每天表达感谢，可以改善夫妻关系、亲子关系

#### 💼 事业发展维度

1. **"付出不亚于任何人的努力"的现实意义**：稻盛和夫用百米赛的速度跑马拉松的比喻，让我反思：17年的技术管理经验积累，是否已经让我进入了"舒适区"？当精力下降时，我是选择降低努力程度，还是寻找新的努力方向？

2. **"能力要用将来进行时"的转型启示**：稻盛和夫主张设定超过现有能力的目标，然后想办法提高能力来达成。转型过程中，我常常说"我没有那个行业的能力"——但这恰恰是"现有能力"思维，而非"将来进行时"思维

3. **"持续就是力量"的深意**：京瓷从一个只有28人的小公司成长为世界500强，靠的是持续的努力而非一时的爆发。转型也需要这样的思维——不是寻求"快速成功"，而是接受"长期积累"的过程

#### 📚 小孩子教育维度

1. **努力观的传承**：稻盛和夫认为"付出不亚于任何人的努力"是做人最基本的条件。这种观念可以教给孩子：努力不是痛苦，而是成长的必经之路

2. **"完美主义"的教育意义**：稻盛和夫追求的不是"差不多就行"，而是"完美"。这可以转化为对孩子的教育：学习不是为了完成作业，而是为了真正掌握知识；做事不是为了敷衍，而是为了做到最好

3. **人生方程式的启蒙**：思维方式×热情×能力的公式可以简化给孩子：聪明不等于成功，努力很重要，但态度最关键。一个诚实、善良、积极的孩子，即使天赋普通，人生也会很美好

## 🎯 三、以我的背景给的建议

### 背景：17年开发+技术管理，两个孩子父亲，精力下降想转型

#### 🌱 人生发展建议

**建议一：重新定义"转型"的意义 🔴🔴🔴**

稻盛和夫说工作是为了"磨炼灵魂、提升心志"。转型不是逃避工作，而是寻找新的"修行道场"。

心理学上的"意义疗法"（维克多·弗兰克尔）表明：当人们找到苦难的意义时，就能承受任何苦难。将转型定义为：
- 不是"逃离疲惫的技术工作"
- 而是"寻找新的成长方式和人格修炼场"

**建议二：检查"思维方式"是否转向负面 🔴🔴🔴**

稻盛和夫的人生方程式中，思维方式是-100到100分。如果思维方式变成负值，能力再强也会失败。

常见的负面思维方式对照表：

| 负面思维方式 | 具体表现 | 转化为正面 |
|-------------|---------|-----------|
| 抱怨现有工作 | "技术工作太累" | "这是我磨炼心志的机会" |
| 焦虑未来 | "转型失败怎么办" | "先全力以赴，失败也是成长" |
| 自我否定 | "我没有其他能力" | "我可以学习新能力" |
| 嫉妒他人 | "别人转型成功了" | "我要向成功者学习" |

**建议三：从"不燃型"转向"自燃型" 🔴🔴**

稻盛和夫将人分为三种类型。作为管理者，我可能已经习惯了"等待命令"的模式。

自燃型的特征：
- 不等别人吩咐就主动行动
- 有明确的目标和问题意识
- 进入"旋涡中心"，带动周围的人

转型需要这种自燃型精神：不是等待机会，而是主动创造机会；不是被动适应，而是积极开拓。

#### 🏠 家庭关系建议

**建议四：让家人理解你的"工作哲学" 🔴🔴**

稻盛和夫说"工作造就人格"。和家人分享这本书的核心观点：
- 努力工作不是为了牺牲家庭，而是为了成为更好的自己
- 一个有追求的父亲，本身就是对孩子最好的教育

这种沟通可以减少家人的担忧，让转型成为全家共同的目标而非个人的冒险。

**建议五：建立家庭"感谢仪式" 🔴**

稻盛和夫反复强调感谢的重要性。建议：
- 每天晚餐时，每个人说一件今天值得感谢的事
- 周末时，向家人表达对本周努力的感谢

心理学研究表明，这种简单的练习可以显著提升家庭幸福感。

#### 💼 事业发展建议

**建议六：用"能力将来进行时"思维规划转型 🔴🔴🔴**

稻盛和夫主张：设定超过现有能力的目标，然后想办法提高能力来达成。

具体做法：
| 现有思维 | 将来进行时思维 |
|---------|---------------|
| "我没有运营经验" | "我可以在6个月内学习运营基础" |
| "我没有行业人脉" | "我可以通过参加行业活动建立人脉" |
| "我精力下降" | "我可以调整工作方式，用智慧而非体力" |

**建议七：接受"持续积累"的转型节奏 🔴🔴**

稻盛和夫用金字塔比喻：伟大事业是无数块石头积累起来的。转型不是"快速成功"，而是：
- 每天学习一点新领域知识
- 每周尝试一点新方向
- 每月完成一个小目标
- 一年积累成可观的成果

这符合心理学中的**渐进改变理论**——大目标分解为小步骤，更容易坚持。

**建议八：从"最佳"转向"完美" 🔴**

稻盛和夫区分"最佳"（相对）和"完美"（绝对）。在技术管理中，我们习惯了"比别人好就行"的思维。转型时，应该追求：
- 不是"比其他转型者成功"
- 而是"在自己的领域做到无可挑剔"

这种思维转变可以减少攀比焦虑，专注于自己的成长。

#### 📚 小孩子教育建议

**建议九：用"人生方程式"教育孩子 🔴🔴**

简化稻盛和夫的公式给孩子：

**成功 = 态度 × 努力 × 天赋**

告诉孩子：
- 天赋是上天给的，不必比较
- 努力是自己决定的，越努力越好
- **态度最重要**——诚实、善良、积极，比聪明更重要

**建议十：培养"自燃型"品格 🔴**

教孩子成为自燃型的人：
- 不等父母催促就主动学习
- 遇到问题先自己想办法
- 不抱怨作业多，而是想办法高效完成

这比教孩子具体知识更重要——因为品格决定一生的走向。

**建议十一：用工作态度做示范 🔴🔴**

稻盛和夫说"工作造就人格"。孩子在观察父亲：
- 如果父亲在工作中表现出热爱和投入，孩子学到的是"努力有意义"
- 如果父亲表现出抱怨和逃避，孩子学到的是"工作很苦"

转型过程中，保持积极的态度本身就是对孩子最好的教育。

## 📋 四、行动计划

### 短期（1-3个月）

- [ ] **思维方式体检**：对照负面思维方式清单，检查自己是否有抱怨、焦虑、自我否定等倾向
- [ ] **建立每日反省习惯**：稻盛和夫每晚反省自己的过失——开始写"每日三省"日记
- [ ] **重新定义转型目标**：将"逃离技术工作"转变为"寻找新的成长道场"
- [ ] **学习"将来进行时"思维**：对每个"我不会"的想法，后面加上"但我可以学习"
- [ ] **和家人分享工作哲学**：用稻盛和夫的观点，让家人理解努力工作的正面意义

### 中期（3-6个月）

- [ ] **每天付出"不亚于任何人的努力"**：不是降低标准，而是找到新的努力方式
- [ ] **建立家庭感谢仪式**：每天晚餐时表达感谢，改善家庭氛围
- [ ] **分解转型目标**：将大目标分解为每月的小目标，持续积累
- [ ] **成为"自燃型"管理者**：不等上级指示就主动发现问题、解决问题
- [ ] **阅读稻盛和夫其他著作**：《活法》《心》《稻盛和夫的实学》等，建立完整的思想体系

### 长期（6-24个月）

- [ ] **形成个人的"工作哲学"**：将稻盛和夫的思想与自己的实践结合，形成自己的工作观
- [ ] **用"完美主义"精神完成转型**：不是"差不多就行"，而是做到无可挑剔
- [ ] **成为转型领域的"自燃型"领导者**：带动周围的人一起成长
- [ ] **教导孩子人生方程式**：态度×努力×天赋，态度最重要
- [ ] **持续积累新领域的知识和经验**：像砌金字塔一样，一块石头一块石头往上堆
