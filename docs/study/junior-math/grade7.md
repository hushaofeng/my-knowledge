# 初一数学知识体系（七年级）

> 🔴 **中考高频**　🟡 **中考常考**　🟢 **基础掌握**
>
> 基于人教版教材，结合 2022—2025 年广州中考命题规律标注重点。

---

## 七年级上册

### 第一章 有理数 🟡

#### 1.1 正数和负数

**知识点**：正数、负数的概念；用正负数表示具有相反意义的量。

**案例**：
- 若零上 8°C 记作 +8°C，则零下 5°C 记作 **-5°C**
- 向东走 50 米记作 +50 米，则向西走 30 米记作 **-30 米**

#### 1.2 有理数

**知识点**：有理数的分类（整数 + 分数）；数轴、相反数、绝对值。

**案例**：
- |-3| = **3**，|0| = **0**，|+5| = **5**
- -3 的相反数是 **3**，0 的相反数是 **0**
- 在数轴上，右边的数总比左边的数大

#### 1.3 有理数的加减法 🟡

**知识点**：有理数加法法则、减法法则；加减混合运算。

**案例**：
- (-3) + (+5) = **2**
- (-8) - (-3) = (-8) + (+3) = **-5**
- 某天最低温 -5°C，最高温 6°C，温差 = 6 - (-5) = **11°C**

#### 1.4 有理数的乘除法 🟡

**知识点**：乘法法则（同号得正、异号得负）；除法转化乘法；乘方运算。

**案例**：
- (-2) &times; (-3) = **6**
- (-12) &divide; (+4) = **-3**
- (-2)³ = **-8**，(-3)² = **9**，-3² = **-9**（注意区分）

> ⚠️ **易错点**：`(-3)² = 9` 与 `-3² = -9` 是中考常考陷阱。

#### 1.5 有理数的混合运算 🟡

**知识点**：运算顺序（先乘方、再乘除、后加减）；科学记数法。

**案例**：
- -1⁴ - (-1)⁴ = -1 - 1 = **-2**
- 科学记数法：567000 = **5.67 &times; 10⁵**

### 第二章 整式的加减 🟢

#### 2.1 整式

**知识点**：单项式（系数、次数）、多项式（项、次数、常数项）。

**案例**：
- 3x²y 的系数是 **3**，次数是 **3**
- 2x³ - 5x + 3 是三次三项式，常数项为 **3**

#### 2.2 整式的加减

**知识点**：合并同类项；去括号法则。

**案例**：
- 3x² + 2x - x² + 5x = (3-1)x² + (2+5)x = **2x² + 7x**
- 2(x - 3) - 3(2x + 1) = 2x - 6 - 6x - 3 = **-4x - 9**

### 第三章 一元一次方程 🔴

> 🔴 **中考高频**：方程是初中数学核心，每年必考，常出现在选择题和应用题中。

#### 3.1 从算式到方程

**知识点**：方程的概念；等式的性质。

**案例**：
- 等式性质 1：若 a = b，则 a + c = b + c
- 等式性质 2：若 a = b，则 ac = bc（c &ne; 0）

#### 3.2 解一元一次方程（一）——合并同类项与移项 🟡

**案例**：
- 解方程：3x + 5 = 2x - 1
  - 移项：3x - 2x = -1 - 5
  - 合并：x = **-6**

#### 3.3 解一元一次方程（二）——去括号与去分母 🔴

**案例**：
- 解方程：(2x + 1)/3 - (x - 1)/2 = 1
  - 去分母（乘 6）：2(2x + 1) - 3(x - 1) = 6
  - 去括号：4x + 2 - 3x + 3 = 6
  - 合并：x + 5 = 6 → x = **1**

> ⚠️ **中考重点**：去分母时不要漏乘不含分母的项。

#### 3.4 实际问题与一元一次方程 🔴

> 🔴 **必考**：应用题是方程的考查核心，常见类型：

| 题型 | 关键等量关系 |
|------|------------|
| 行程问题 | 路程 = 速度 &times; 时间 |
| 工程问题 | 工作量 = 工作效率 &times; 工作时间 |
| 利润问题 | 利润率 = 利润/成本 &times; 100% |
| 积分问题 | 胜场积分 + 平场积分 + 负场积分 = 总积分 |

**案例**：
- 某商品进价 200 元，标价 360 元，打折后仍获利 20%，求打几折？
  - 设打 x 折：360 &times; x/10 = 200 &times; (1 + 20%)
  - 36x = 240 → x = **6.67 折**

### 第四章 几何图形初步 🟢

#### 4.1 几何图形

**知识点**：立体图形与平面图形；展开图。

#### 4.2 直线、射线、线段

**知识点**：两点确定一条直线；两点之间线段最短。

**案例**：
- 已知线段 AB = 10cm，M 是 AB 中点，N 是 MB 中点，则 MN = **2.5cm**

#### 4.3 角 🟡

**知识点**：角的度量；角平分线；余角和补角。

**案例**：
- ∠α = 35°，则余角 = **55°**，补角 = **145°**
- 一个角的补角比它的余角的 3 倍少 10°，求这个角
  - 设角为 x°，则 (180-x) = 3(90-x) - 10
  - 180 - x = 270 - 3x - 10 → 2x = 80 → x = **40°**

## 七年级下册

### 第五章 相交线与平行线 🟡

#### 5.1 相交线 🟡

**知识点**：对顶角相等；邻补角互补；垂线（最短距离）。

**案例**：
- 两条直线相交，∠1 = 70°，则对顶角 ∠3 = **70°**，邻补角 ∠2 = **110°**

#### 5.2 平行线及其判定 🔴

> 🔴 **中考高频**：平行线性质与判定是几何证明的基础，中考必考。

**知识点**：同位角相等 → 两直线平行；内错角相等 → 两直线平行；同旁内角互补 → 两直线平行。

**案例**：
- 已知 ∠1 = ∠2，判断 AB∥CD 的依据是 **同位角相等，两直线平行**

#### 5.3 平行线的性质 🔴

**知识点**：两直线平行 → 同位角相等；内错角相等；同旁内角互补。

**案例**：
- AB∥CD，∠1 = 65°，则内错角 ∠2 = **65°**，同旁内角 ∠3 = **115°**

> ⚠️ **中考规律**：广州中考常将平行线与三角形内角和结合出综合题。

#### 5.4 平移

**知识点**：平移的性质（对应点连线平行且相等）；平移作图。

### 第六章 实数 🟡

#### 6.1 平方根 🟡

**知识点**：算术平方根、平方根的概念与表示。

**案例**：
- &radic;9 = **3**（算术平方根），9 的平方根 = **&plusmn;3**
- &radic;(-3)² = &radic;9 = **3**

#### 6.2 立方根 🟢

**案例**：
- ³&radic;8 = **2**，³&radic;(-27) = **-3**

#### 6.3 实数 🟡

**知识点**：无理数的概念；实数的分类；实数与数轴的一一对应。

**案例**：
- 下列是无理数的是：&radic;2、π、0.1010010001...（每两个 1 之间 0 的个数递增）
- |&radic;2 - 1| + |&radic;2 - 2| = (&radic;2 - 1) + (2 - &radic;2) = **1**

#### 7.2 平面直角坐标系 🟡

**知识点**：各象限点的坐标特征；关于坐标轴和原点对称的点。

### 第八章 二元一次方程组 🟡

#### 8.1 二元一次方程组

#### 8.2 消元——解二元一次方程组 🔴

**知识点**：代入消元法、加减消元法。

**案例**：
- 解方程组：{ 2x + y = 5 ...①，3x - 2y = 4 ...② }

### 第九章 不等式与不等式组 🔴

> 🔴 **中考高频**：不等式是广州中考重点，常与方案设计、最值问题结合。

#### 9.1 不等式

**知识点**：不等式的基本性质（特别注意：两边同乘负数，不等号方向改变）。

**案例**：
- 若 a > b，则 -2a **<** -2b

#### 9.2 一元一次不等式 🟡

**案例**：
- 解不等式：2(x - 1) &le; 3x + 1
  - 2x - 2 &le; 3x + 1 → -x &le; 3 → x &ge; **-3**

#### 9.3 一元一次不等式组 🔴

**案例**：
- 解不等式组：{ 2x + 1 > 0 ...①，3x - 2 &le; 7 ...② }
  - 由①：x > -1/2；由②：x &le; 3
  - 解集：**-1/2 < x &le; 3**

> ⚠️ **口诀**：同大取大，同小取小，大小小大取中间，大大小小无解。

#### 9.4 实际问题与一元一次不等式 🔴

**案例**：
- 某商店购进单价 40 元的商品，标价 60 元，打折销售要保证利润率不低于 5%，最多打几折？
  - 60 &times; x/10 &ge; 40 &times; (1 + 5%) → 6x &ge; 42 → x &ge; 7
  - 至少打 **7 折**

## 初一重点总结

### 广州中考初一相关考点权重

| 章节 | 中考占比 | 重要性 | 说明 |
|------|---------|--------|------|
| 一元一次方程 | 约 8-10% | 🔴 高 | 应用题必考 |
| 不等式与不等式组 | 约 6-8% | 🔴 高 | 方案设计题常考 |
| 平行线 | 约 5-7% | 🟡 中高 | 几何证明基础 |
| 有理数运算 | 约 4-5% | 🟡 中 | 选择填空必出 |
| 实数 | 约 3-4% | 🟡 中 | 常与二次根式结合 |
| 二元一次方程组 | 约 3-4% | 🟡 中 | 应用题偶尔出现 |
| 坐标系 | 约 2-3% | 🟡 中 | 与函数图像结合 |
| 整式加减 | 约 2% | 🟢 低 | 基础运算 |
| 几何初步 | 约 2% | 🟢 低 | 角度计算偶出 |
| 数据统计 | 约 2% | 🟢 低 | 基础概念 |

### 初一学习建议

1. **有理数运算是根基** — 计算能力直接影响所有后续章节
2. **方程思想要贯穿始终** — 从算术思维过渡到方程思维是初一最大转折
3. **不等式易错点多** — 变号问题是考试常设陷阱
4. **平行线是几何入门** — 掌握"判定"和"性质"的区别，为初二三角形打基础
