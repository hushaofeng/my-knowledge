## 3.1 教学概述 🔴

### 教学的任务

1. 传授系统基础知识（首要任务）🔴
2. 形成基本技能
3. 发展智力和体力
4. 培养品德和审美
5. 促进个性发展

### 教学工作的基本环节 🔴🔴

| 环节 | 地位 | 要点 |
|------|------|------|
| 备课 🔴 | 上课的前提 | 备教材、备学生、备教法（三备） |
| 上课 🔴🔴 | 中心环节 | 好课标准：目标明确、内容正确、方法恰当、结构紧凑 |
| 作业布置与批改 | 课堂延续 | 量适当、有针对性 |
| 课外辅导 | 补充 | 因材施教 |
| 学业成绩评价 | 检验 | 形成性+终结性 |

**备课的三种计划** 🔴：
1. 学期教学计划（一学期）
2. 单元教学计划（一单元）
3. 课时教学计划（一节课，即教案）

## 3.2 教学过程 🔴🔴🔴

### 教学过程的本质

教学过程是一种**特殊的认识过程** 🔴🔴：
- 认识的间接性 — 以书本知识为主
- 认识的引导性 — 在教师引导下
- 认识的简捷性 — 走捷径，不必重复人类探索的全部过程

### 教学过程的基本规律 🔴🔴🔴

**口诀：舰长直教（间长知交）**

| 规律 | 含义 | 教育启示 | 判断关键词 |
|------|------|---------|-----------|
| 间接性规律 🔴🔴 | 直接经验与间接经验相统一，以间接经验为主 | 以书本知识为主，辅以直接经验 | 书本、间接、理论联系实际 |
| 发展性规律 🔴🔴 | 掌握知识与发展能力相统一 | 既要传授知识又要发展能力 | 知识+能力、授人以渔 |
| 教育性规律 🔴 | 传授知识与思想教育相统一 | 教学永远具有教育性（赫尔巴特） 🔴 | 教书育人、教学的教育性 |
| 双边性规律 🔴🔴 | 教师主导与学生主体相统一 | 教师主导+学生主动 | 师生互动、教师引导+学生参与 |

**辨析题高频** 🔴🔴🔴：

1. "教学就是传授知识" → ❌ 还包括发展能力、思想教育
2. "掌握知识就等于发展了能力" → ❌ 知识不等于能力，需要转化
3. "教学永远具有教育性" → ✅ 赫尔巴特名言
4. "教师主导与学生的主体地位矛盾" → ❌ 二者统一

### 教学过程阶段 🔴

**赫尔巴特五段论**：预备→提示→联想→总结→应用

**我国常用阶段**：引发动机→领会知识→巩固知识→运用知识→检查知识

## 3.3 教学原则 🔴🔴🔴

> 必考简答题或材料分析题

### 八大教学原则

| 原则 | 含义 | 关键词 | 名言/出处 🔴 | 判断材料关键词 |
|------|------|--------|-------------|--------------|
| 直观性原则 | 通过实物或直观教具使学生获得感性认识 | 看得见 | 夸美纽斯"一切事物尽量用感官去感知" | 演示、模型、图片、参观 |
| 启发性原则 🔴🔴🔴 | 引导学生独立思考，主动学习 | 不直接给答案 | 孔子"不愤不启，不悱不发" 🔴 | 提问、讨论、引导思考 |
| 巩固性原则 | 复习已学知识，牢固掌握 | 温故知新 | 孔子"温故而知新" 🔴 | 复习、练习、反复 |
| 循序渐进原则 🔴 | 按学科逻辑顺序，由易到难 | 逐步深入 | 《学记》"学不躐等""不陵节而施" 🔴 | 由简到难、由浅入深 |
| 因材施教原则 🔴 | 根据学生差异，区别对待 | 不同对待 | 朱熹"夫子教人各因其材" 🔴 | 分层、针对不同学生 |
| 理论联系实际原则 | 学以致用，知行合一 | 知行合一 | — | 实践活动、生活实例 |
| 量力性原则 | 适应学生发展水平，可接受 | 可接受 | 墨子"夫智者必量其力所能至" | 难度适中、跳一跳摘桃子 |
| 科学性与思想性统一 | 传授科学知识+思想教育 | 既教书又育人 | — | 既要科学准确又要思想引导 |

**口诀**：直启巩循因理量科（直起弓寻因理量科）

### 教学原则辨析 🔴🔴

| 易混点 | 区别 |
|--------|------|
| 循序渐进 vs 量力性 | 循序渐进 = 按知识逻辑顺序；量力性 = 适应学生接受能力 |
| 直观性 vs 启发性 | 直观性 = 让知识可见；启发性 = 让思维活跃 |
| 因材施教 vs 循序渐进 | 因材施教 = 针对不同学生；循序渐进 = 按顺序教 |

## 3.4 教学方法 🔴🔴

### 常用教学方法

| 类型 | 方法 | 含义 | 适用场景 |
|------|------|------|---------|
| 语言传递 | 讲授法 🔴 | 教师口头系统讲解 | 概念、原理 |
| | 谈话法 | 师生问答 | 启发思维 |
| | 讨论法 | 学生分组讨论 | 开放性问题 |
| | 读书指导法 | 指导学生阅读 | 自学能力培养 |
| 直观感知 | 演示法 | 展示实物或模型 | 形象认知 |
| | 参观法 | 组织实地参观 | 实践体验 |
| 实际训练 | 练习法 🔴 | 反复操作巩固 | 技能训练 |
| | 实验法 | 学生动手操作实验 | 科学探究 |
| | 实习作业法 | 校外实践 | 职业教育 |
| 引导探究 | 发现法 🔴 | 引导学生自主发现 | 规律探究 |

**教学方法选择依据** 🔴：
1. 教学目的和任务
2. 课程性质和教材特点
3. 学生特点
4. 教学时间、设备、条件
5. 教师业务水平

## 3.5 教学组织形式 🔴🔴

| 形式 | 特点 | 创立者/代表 |
|------|------|------------|
| 个别教学 | 最早的形式，教师个别指导 | 古代 |
| 班级授课制 🔴🔴🔴 | 最基本的形式，固定班级、固定课表 | 夸美纽斯《大教学论》🔴 |
| 分组教学 | 外部分组+内部分组 | — |
| 设计教学法 | 学生自主决定学习内容和活动 | 克伯屈 🔴 |
| 道尔顿制 | 自学+答疑，教师不再系统讲授 | 柏克赫斯特 🔴 |
| 特朗普制 | 大班上课+小班讨论+个人自学 | 特朗普 |

**班级授课制的优缺点** 🔴🔴：

| 优点 | 缺点 |
|------|------|
| 效率高，大面积培养人才 | 不易因材施教 |
| 发挥教师主导作用 | 不易发挥学生主体性 |
| 系统性强 | 实践性不足 |
| 集体教育效果好 | 灵活性不够 |

## 本章速记卡

### 教学原则口诀

**直起弓寻因理量科**：直观、启发、巩固、循序渐进、因材施教、理论联系实际、量力性、科学性与思想性统一

### 教学过程规律

**舰长直教**：间接性、发展性、教育性、双边性

### 简答题必背

**教学过程四大规律** 🔴🔴🔴：
1. 直接经验与间接经验相统一（间接性）
2. 掌握知识与发展能力相统一（发展性）
3. 传授知识与思想教育相统一（教育性）
4. 教师主导与学生主体相统一（双边性）

**启发性原则贯彻要求** 🔴🔴：
1. 调动学生主动性
2. 启发独立思考
3. 让学生动手实践
4. 发扬教学民主
