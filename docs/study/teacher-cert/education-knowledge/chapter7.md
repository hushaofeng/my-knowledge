# 第七章 中学德育

> 本章考情：高频大题，约 15-20 分

## 7.1 品德结构 🔴🔴

| 要素 | 含义 | 地位 |
|------|------|------|
| 道德认识 🔴 | 对道德规范的理解和掌握 | 核心基础 |
| 道德情感 | 伴随道德认识产生的情感体验 | 动力 |
| 道德意志 | 克服困难、坚持道德行为 | 调节 |
| 道德行为 🔴 | 在道德认识指导下的行为表现 | 标志/关键 |

**口诀**：知情意行（认识→情感→意志→行为）

**辨析** 🔴：
- "学生道德认识好，品德就好" → ❌ 道德认识是基础，但还要看行为
- "道德行为是品德的标志" → ✅ 行为是品德的外在表现

---

## 7.2 品德发展理论 🔴🔴🔴

### 皮亚杰道德发展阶段 🔴🔴

| 阶段 | 年龄 | 特点 |
|------|------|------|
| 前道德阶段 | 2-5岁 | 无规则意识 |
| 他律道德阶段 🔴 | 5-8岁 | 以成人规则为标准，看结果不看动机 |
| 自律道德阶段 🔴 | 8-10岁 | 认识到规则可变，看动机 |
| 公正道德阶段 | 10岁+ | 关注公平正义 |

**他律→自律的关键**：可逆性（认识到规则是同伴间共同约定的）

### 科尔伯格道德发展阶段 🔴🔴🔴

**三水平六阶段**：

| 水平 | 阶段 | 核心动机 | 判断特征 |
|------|------|---------|---------|
| 前习俗水平 🔴 | 1.惩罚服从 | 怕惩罚 | 不做→怕被抓 |
| | 2.相对功利 | 满足自己需要 | 做了→能得到好处 |
| 习俗水平 🔴🔴 | 3.寻求认可 | 做好人 | 做了→别人会夸我 |
| | 4.遵守法规 | 遵守法律 | 做了→法律规定的 |
| 后习俗水平 | 5.社会契约 | 社会契约 | 法律可修改以适应社会需要 |
| | 6.普遍伦理 | 内心道德准则 | 做了→因为这是对的 |

**经典实验**：海因茨偷药困境 🔴

**辨析** 🔴：
- "小学生都处于前习俗水平" → ❌ 小学生也可能处于习俗水平
- "道德发展越高越不遵守规则" → ❌ 后习俗水平遵守的是更高层次的道德准则

## 7.3 德育过程 🔴🔴🔴

### 德育过程的基本规律 🔴🔴🔴

**口诀：知情意行多开端，活动交往中形成，内部矛盾促转化，长期反复逐步高**

1. **知、情、意、行诸因素统一发展** 🔴🔴
   - 多端性：不一定从"知"开始，可从任一要素开始
   - 实际中"晓之以理、动之以情、持之以恒、导之以行"

2. **在活动和交往中形成** 🔴
   - 学生的思想品德是在活动与交往中形成的
   - 学校活动和交往是主要途径

3. **思想内部矛盾转化** 🔴
   - 教育者提出的道德要求→学生已有品德水平之间的矛盾
   - 这是德育过程的主要矛盾

4. **学生思想长期、反复、逐步提高** 🔴
   - 德育是一个反复的过程
   - 需要反复抓、抓反复

**辨析** 🔴🔴：
- "德育总是从知开始" → ❌ 德育具有多端性，可从知情意行任一要素开始
- "学生思想品德一次形成" → ❌ 长期反复逐步提高

## 7.4 德育原则 🔴🔴🔴

> 必考简答题或材料分析题

### 九大德育原则

| 原则 | 含义 | 关键词 | 名言/出处 🔴 |
|------|------|--------|-------------|
| 导向性原则 | 坚持正确方向 | 大方向 | — |
| 疏导原则 🔴🔴🔴 | 循循善诱，以理服人 | 不堵塞 | "循循善诱" |
| 尊重与严格要求相结合 🔴🔴 | 严慈相济 | 马卡连柯 | 马卡连柯"尽量多地要求一个人，也尽可能地尊重一个人" 🔴 |
| 教育的一致性与连贯性 | 统一要求 | 家校统一 | — |
| 因材施教原则 🔴 | 根据实际 | 不同对待 | — |
| 长善救失原则 🔴🔴 | 发扬优点克服缺点 | 一分为二 | — |
| 集体教育与个别教育相结合 🔴 | 通过集体教育个人 | 平行教育 | 马卡连柯"平行教育" 🔴 |
| 正面教育与纪律约束相结合 | 说服+规章 | 两条腿走路 | — |
| 知行统一原则 | 理论与实践统一 | 言行一致 | — |

**口诀**：导疏尊一连，因长集正知

### 德育原则辨析 🔴🔴

| 易混点 | 区别 |
|--------|------|
| 疏导 vs 尊重与严格要求 | 疏导 = 说服引导；严格要求 = 有标准有底线 |
| 长善救失 vs 因材施教 | 长善救失 = 扬长避短；因材施教 = 针对差异 |
| 集体教育 vs 一致性 | 集体教育 = 通过集体影响个人；一致性 = 统一各方要求 |

### 疏导原则贯彻要求 🔴🔴🔴（简答题必背）

1. 讲明道理，疏通思想
2. 因势利导，循循善诱
3. 以表扬激励为主，坚持正面教育

## 7.5 德育途径与方法 🔴🔴

### 主要途径 🔴

1. 思想政治课 — 基本途径 🔴🔴
2. 其他学科教学 — 最经常、最基本的途径
3. 班主任工作 — 重要途径
4. 共青团、少先队活动
5. 劳动与社会实践
6. 课外活动与校外活动

### 德育方法 🔴🔴🔴

| 方法 | 含义 | 判断关键词 |
|------|------|-----------|
| 说服教育法 🔴🔴🔴 | 通过说理使学生接受 | 讲道理、说服、讲解 |
| 榜样示范法 🔴🔴 | 以他人优良品行影响学生 | 榜样、示范、英雄人物 |
| 陶冶教育法 🔴 | 创设情境潜移默化 | 环境、氛围、熏陶 |
| 实际锻炼法 | 组织实践活动 | 实践、活动、训练 |
| 品德评价法 🔴 | 表扬或批评 | 评价、奖惩、操行评定 |
| 道德修养法 | 引导自我教育 | 自我反思、自我修养 |

**辨析** 🔴：
- "说服教育法就是口头说教" → ❌ 说服包括讲解、谈话、讨论、参观等多种形式
- "陶冶教育法就是放任不管" → ❌ 陶冶是有目的地创设情境

## 7.6 材料分析题专题 🔴🔴🔴

### 答题模板

```
该教师的教育行为符合/违背了德育原则的要求，具体分析如下：

第一，符合/违背了"……原则"的要求。
   该原则要求……（阐述原则内容）。
   材料中，该教师……（结合材料），体现了/违背了……。

第二，……

第三，……

综上所述，该教师的行为值得学习/需要改进。
```

### 真题示例

**材料**：班主任李老师发现班上小张同学经常迟到、不交作业。李老师没有直接批评他，而是找小张谈话，了解到小张父母常年在外打工，他独自照顾年迈的奶奶。李老师一方面耐心给小张讲解遵守纪律的重要性，另一方面组织同学成立互助小组帮助小张。同时，李老师还发现小张在体育方面很有天赋，就推荐他加入校运动队。一个学期后，小张不仅不再迟到，成绩也有了明显进步。

**问题**：请运用德育原则分析李老师的做法。

**参考答案**：

李老师的做法符合德育原则的要求，具体分析如下：

第一，符合疏导原则。疏导原则要求循循善诱，以理服人，以表扬激励为主。材料中，李老师没有直接批评小张，而是耐心讲解遵守纪律的重要性，循循善诱地引导他认识问题。

第二，符合长善救失原则。该原则要求发扬学生优点、克服缺点。材料中，李老师发现小张体育方面的特长，推荐他加入运动队，通过发扬优点增强自信，从而带动其他方面的改进。

第三，符合因材施教原则。该原则要求根据学生实际情况采取不同教育方法。材料中，李老师了解到小张的家庭特殊情况后，采取互助小组等针对性措施，体现了因材施教。

第四，符合尊重与严格要求相结合原则。该原则要求尊重学生人格的同时坚持合理要求。材料中，李老师尊重小张的实际困难，同时要求他遵守纪律，严慈相济。

综上所述，李老师的做法值得学习。
