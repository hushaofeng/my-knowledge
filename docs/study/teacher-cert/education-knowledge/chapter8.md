# 第八章 中学班级管理与教师心理

> 本章考情：选择题为主，约 6-8 分
> 重点：班集体建设、课堂管理、教师成长阶段

---

## 8.1 班级管理 🔴🔴

### 班集体 🔴🔴

**班集体的基本特征** 🔴🔴：

1. 明确的共同目标 — 奋斗方向
2. 一定的组织结构 — 核心队伍（班干部）
3. 一定的共同生活准则 — 规章制度
4. 集体成员之间平等、心理相容 — 舆论氛围

**班集体的形成阶段** 🔴：

| 阶段 | 特点 |
|------|------|
| 组建阶段 | 依赖教师，松散 |
| 核心形成阶段 | 骨干出现，凝聚力增强 🔴 |
| 发展阶段 | 自我管理，舆论形成 |
| 成熟阶段 | 自觉遵守，共同进步 |

### 班主任工作 🔴

**班主任工作内容**：

1. 了解和研究学生 — 前提 🔴
2. 组织和培养班集体 — 中心任务 🔴🔴
3. 协调校内外各种教育力量
4. 班级日常管理
5. 做好个别教育工作
6. 做好学生操行评定

**班主任了解学生的方法** 🔴：
- 观察法 — 最基本的方法 🔴
- 谈话法
- 书面材料分析法
- 问卷法

## 8.2 课堂管理 🔴

### 群体动力 🔴

| 要素 | 含义 | 教育启示 |
|------|------|---------|
| 群体凝聚力 | 群体对成员的吸引力 | 增强凝聚力提高学习效率 |
| 群体规范 | 群体成员共同遵守的行为准则 | 制定合理的班规 |
| 课堂气氛 | 课堂中的情感状态 | 营造积极和谐的课堂氛围 |
| 人际关系 | 师生、生生关系 | 建立良好师生关系 |

### 课堂纪律 🔴🔴

| 类型 | 含义 | 形成方式 |
|------|------|---------|
| 教师促成的纪律 | 教师要求下形成的 | 教师监督指导 |
| 集体促成的纪律 | 同辈群体压力下形成的 | 同伴影响 |
| 任务促成的纪律 | 为完成特定任务而遵守 | 任务驱动 |
| 自我促成的纪律 🔴 | 学生自觉遵守 | 内化自律（最高级） |

### 课堂问题行为 🔴

**分类**：
- 品行性问题行为 — 攻击、破坏
- 性格性问题行为 — 孤僻、退缩
- 情绪上、社会上的不适应行为

**处理原则**：
1. 预防为主
2. 非言语暗示优先
3. 最小干预原则 🔴
4. 就事论事，不翻旧账

## 8.3 课外校外教育 🔴

### 特点 🔴

1. 自愿性 — 学生自主选择 🔴
2. 自主性 — 学生主动参与
3. 灵活性 — 形式多样
4. 实践性 — 亲身体验

### 形式

1. 群众性活动 — 讲座、比赛
2. 小组活动 — 兴趣小组 🔴（最普遍的形式）
3. 个人活动 — 个别指导

## 8.4 教师心理 🔴🔴

### 教师成长阶段 🔴🔴🔴

| 阶段 | 关注焦点 | 特点 |
|------|---------|------|
| 关注生存阶段 🔴 | 别人怎么看我 | 想被认可、被喜欢 |
| 关注情境阶段 | 教学情境 | 关注课怎么上好 |
| 关注学生阶段 🔴🔴 | 学生发展 | 因材施教、关注差异 |

### 教师专业发展 🔴

| 阶段 | 特点 |
|------|------|
| 新手教师 | 严格遵守规则，缺乏灵活性 |
| 高级新手 | 能处理简单情境 |
| 胜任者 | 能有效教学，有明确目标 |
| 精通者 | 教学直觉强，灵活应变 |
| 专家教师 | 自动化，举重若轻 |

**新手与专家教师对比** 🔴：

| 维度 | 新手教师 | 专家教师 |
|------|---------|---------|
| 课前准备 | 关注细节，写详细教案 | 关注整体，灵活预案 |
| 课堂教学 | 按部就班 | 灵活应变 |
| 课后反思 | 关注自己表现 | 关注学生学习效果 |
| 课堂规则 | 事后制定 | 事先建立 |

### 教师心理健康 🔴

**职业倦怠** 🔴🔴：

| 维度 | 表现 |
|------|------|
| 情绪衰竭 🔴 | 疲惫、缺乏热情 |
| 去个性化 | 冷漠、疏远学生 |
| 低个人成就感 | 觉得工作没意义 |

**职业倦怠的应对** 🔴：
1. 社会支持
2. 改善工作环境
3. 自我调节与成长
4. 建立合理期望
