## Hermes Agent 核心能力全景

Hermes 不是"套壳聊天机器人"，它是一个**自主代理（autonomous agent）**，能调用工具、执行代码、操作浏览器、定时运行任务，并且会**从经验中学习**——解决问题的过程自动生成为 Skill（技能文件），跨会话记忆持续积累。

你当前运行的是 **v0.14.0**，后端走 DeepSeek V4 Flash。以下是它核心能做的事情：

---

### 一、编程（代码能力最强项）

| 能力 | 说明 |
|------|------|
| **读/写/搜索文件** | 直接读写任意文件，支持正则搜索、patch 精准替换 |
| **执行 Shell 命令** | 在本地 macOS 上跑任何命令（git、编译、部署、pip 安装等） |
| **Python 脚本执行** | `execute_code` 工具可持续写 Python 脚本调用 Hermes 的其他工具，把多步操作压缩成一步 |
| **子代理并行工作** | `delegate_task` 同时派 3 个子代理干不同的活（比如同时查文档、写代码、跑测试） |
| **项目管理** | 自动加载项目中的 `AGENTS.md`/`CLAUDE.md`/`.cursorrules` 等上下文文件 |
| **代码回滚** | 修改文件前自动快照，`/rollback` 一键还原 |
| **IDE 集成** | 通过 ACP 协议接入 VS Code、Zed、JetBrains 等编辑器 |
| **LSP 支持** | 内置语言服务器协议，写代码时做语法检查 |

**实际能做的例子：** 写前端/后端项目、调试 bug、代码重构、git 操作、自动化测试、CI/CD 流水线
